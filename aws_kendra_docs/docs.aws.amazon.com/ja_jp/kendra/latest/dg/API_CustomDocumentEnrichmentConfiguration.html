CustomDocumentEnrichmentConfiguration

ドキュメントの取り込みプロセス中にドキュメントのメタデータとコンテンツを変更するための構成情報を提供します。

詳細については、取り込みプロセス中のドキュメントのメタデータのカスタマイズを参照してください。

目次

InlineConfigurations

Amazon Kendra へのドキュメントの取り込み時に、ドキュメントの属性またはメタデータフィールドとコンテンツを変更するための構成情報。

型: InlineCustomDocumentEnrichmentConfiguration オブジェクトの配列

配列メンバー: 最小数は 0 項目です。最大数は 100 項目です。

必須: いいえ

PostExtractionHookConfiguration

メタデータと抽出されたテキストを含む構造化ドキュメントに関する AWS Lambda の Lambda 関数を呼び出すための構成情報。Lambda 関数を使用して、ドキュメントのメタデータとコンテンツを作成、変更、または削除するための高度なロジックを適用できます。詳細については、高度なデータ操作を参照してください。

型: HookConfiguration オブジェクト

必須: いいえ

PreExtractionHookConfiguration

メタデータとテキストを抽出する前に元のドキュメントまたは未処理のドキュメントに関する AWS Lambda の Lambda 関数を呼び出すための構成情報。Lambda 関数を使用して、ドキュメントのメタデータとコンテンツを作成、変更、または削除するための高度なロジックを適用できます。詳細については、高度なデータ操作を参照してください。

型: HookConfiguration オブジェクト

必須: いいえ

RoleArn

ドキュメントの取り込みプロセス中のドキュメントのメタデータとコンテンツを変更するための PreExtractionHookConfiguration および PostExtractionHookConfiguration を実行する許可を持つロールの Amazon リソースネーム (ARN)。詳細については、Amazon Kendra の IAM ロールを参照してください。

タイプ: 文字列

長さの制限: 最小長は 0 です。最大長は 1,284 です。

パターン: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

必須: いいえ
