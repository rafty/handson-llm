QueryResultItem

1 つのクエリ結果。

クエリ結果には、クエリで返されたドキュメントに関する情報が含まれています。これには、ドキュメントの元の場所、ドキュメントに割り当てられた属性のリスト、およびクエリを満たすドキュメントの関連テキストが含まれます。

目次

AdditionalAttributes

クエリ結果に関連付けられた 1 つ以上の追加属性。

型: AdditionalResultAttribute オブジェクトの配列

必須: いいえ

DocumentAttributes

検索結果内のドキュメントに割り当てられたドキュメント属性の配列。例えば、ドキュメントの作成者 (_author) またはドキュメントのソース URI (_source_uri) などです。

型: DocumentAttribute オブジェクトの配列

必須: いいえ

DocumentExcerpt

ドキュメント内のテキストの抽出。抜粋で関連する用語を強調表示する方法に関する情報が含まれています。

型: TextWithHighlights オブジェクト

必須: いいえ

DocumentId

ドキュメントの一意の識別子。

タイプ: 文字列

長さの制限: 最小長は 1 です。最大長は 2,048 です。

必須: いいえ

DocumentTitle

ドキュメントのタイトル。タイトルのテキストと、タイトル内の関連用語を強調表示するための情報が含まれます。

型: TextWithHighlights オブジェクト

必須: いいえ

DocumentURI

ドキュメントの元の場所の URI。

タイプ: 文字列

長さの制限: 最小長は 1 です。最大長は 2,048 です。

パターン: ^(https?|ftp|file):\/\/([^\s]*)

必須: いいえ

FeedbackToken

特定のクエリから特定の結果を識別するトークン。このトークンを使用して、結果のクリックスルーフィードバックを提供します。詳細については、フィードバックの送信を参照してください。

タイプ: 文字列

長さの制限: 最小長は 1 です。最大長は 2,048 です。

パターン: ^\P{C}*.\P{C}*$

必須: いいえ

Format

レスポンス内のType of ドキュメントがの場合ANSWER、TABLEそれは回答か回答のどちらかです。TEXT表形式の回答の場合は、表の抜粋が返されますTableExcerpt。テキスト回答の場合は、テキストの抜粋が返されますDocumentExcerpt。

タイプ: 文字列

有効な値: TABLE | TEXT

必須: いいえ

Id

クエリ結果の一意の識別子。

タイプ: 文字列

長さの制限: 最小長は 1 です。最大長は 73 です。

必須: いいえ

ScoreAttributes

Amazon Kendra が指定したクエリと結果が一致する結果を持つ信頼度を示します。各結果は、信頼度、VERY_HIGH、HIGH、MEDIUM および LOW を示すビンに配置されます。スコアを使用して、レスポンスがアプリケーションに必要な信頼を満たしているかどうかを判断できます。

このフィールドは、LOW フィールドが Type に設定され、Amazon Kendra は、結果がクエリと一致することに対して信頼度が無い場合は、DOCUMENT にしか設定されません。

型: ScoreAttributes オブジェクト

必須: いいえ

TableExcerpt

文書内の表からの抜粋。

型: TableExcerpt オブジェクト

必須: いいえ

Type

レスポンス内のドキュメントのタイプ。たとえば、応答にはクエリに関連する質問と回答を含めることができます。

タイプ: 文字列

有効な値: DOCUMENT | QUESTION_ANSWER | ANSWER

必須: いいえ
