TagResource

指定されたタグを、指定されたインデックス、よくある質問、またはデータソースリソースに追加します。タグがすでに存在する場合、既存の値は新しい値に置き換えられます。

リクエストの構文

{
   "ResourceARN": "string",
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

リクエストパラメータ

すべてのアクションに共通のパラメータの詳細については、「共通パラメータ」を参照してください。

リクエストは以下の JSON 形式のデータを受け入れます。

ResourceARN

タグ付けするインデックス、よくある質問、またはデータソースの Amazon リソースネーム (ARN)。

型: 文字列

長さの制限: 最小長は 1 です。最大長は 1011 です。

必須: はい

Tags

インデックス、よくある質問、またはデータソースに追加するタグキーのリスト。タグがすでに存在する場合、既存の値は新しい値に置き換えられます。

型: Tag オブジェクトの配列

配列メンバー: 最小数は 0 項目です。最大数は 200 項目です。

必須: はい

レスポンス要素

アクションが成功した場合、サービスは空の HTTP 本文を持つ HTTP 200 応答を返します。

エラー

すべてのアクションに共通のエラーについては、「一般的なエラー」を参照してください。

AccessDeniedException

このアクションを実行する十分なアクセス権限がありません。必要な権限ポリシーとユーザーアカウントがあることを確認して、もう一度試してください。

HTTP ステータスコード: 400

InternalServerException

Amazon Kendra サービスに使用されている内部サーバーで問題が発生しました。数分後にもう一度お試しSupport さい。

HTTP ステータスコード: 500

ResourceUnavailableException

使用するリソースは使用できません。正しいリソースを提供したことを確認して、もう一度試してください。

HTTP ステータスコード: 400

ThrottlingException

リクエストは、制限が必要なために実行が拒否されました。リクエストの数を減らして、もう一度試してください。

HTTP ステータスコード: 400

ValidationException

入力が Amazon Kendra サービスで設定された制約を満たしていません。正しい入力を、もう一度お試しください。

HTTP ステータスコード: 400
