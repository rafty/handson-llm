Status

インデックス作成のために送信されたドキュメントのステータスに関する情報を提供します。

目次

DocumentId

ドキュメントの一意の識別子。

タイプ: 文字列

長さの制限: 最小長は 1 です。最大長は 2,048 です。

必須: いいえ

DocumentStatus

ドメインの現在のステータス。

ドキュメントが削除のために送信された場合、ステータスはドキュメントが削除された後、NOT_FOUND になります。

タイプ: 文字列

有効な値: NOT_FOUND | PROCESSING | INDEXED | UPDATED | FAILED | UPDATE_FAILED

必須: いいえ

FailureCode

エラーの原因を示します。

タイプ: 文字列

長さの制限: 最小長は 1 です。最大長は 2,048 です。

必須: いいえ

FailureReason

ドキュメントにインデックス作成できなかった理由に関する詳細情報を提供します。インデックス作成のためにドキュメントを再送信する前に、この情報を使用してエラーを修正してください。

タイプ: 文字列

長さの制限: 最小長は 1 です。最大長は 2,048 です。

分位数は、確率分布を等しい確率の領域に分割したものです。
