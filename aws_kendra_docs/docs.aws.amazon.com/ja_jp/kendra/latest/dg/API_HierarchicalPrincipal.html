HierarchicalPrincipal

ユーザーがアクセスできるドキュメントの階層を定義する情報。

目次

PrincipalList

プリンシパルのリストは、ユーザーがアクセスすべきドキュメントの階層を定義するリストです。各階層リストでは、各ドキュメントに対してアクセスの許可または拒否権を持つユーザーまたはグループを指定します。

型: Principal オブジェクトの配列

必須: はい
