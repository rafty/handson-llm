ServiceNow

ServiceNow は、IT サービス、チケットシステム、サポートなど、組織レベルのワークフローを作成および管理するためのクラウドベースのサービス管理システムを提供します。Amazon Kendraを使用して、 ServiceNow カタログ、ナレッジ記事、インシデント、および添付ファイルのインデックスを作成できます。

Amazon Kendraコンソール、API、または TemplateConfigurationAPIAmazon Kendra ServiceNow のいずれかを使用してデータソースに接続できます。ServiceNowConfiguration

Amazon Kendraには 2 ServiceNow つのバージョンのコネクタがあります。各バージョンでサポートされる機能は次のとおりです。

ServiceNow コネクタ V1.0/ServiceNowConfigurationAPI

  • フィールドマッピング

  • ServiceNow インスタンスバージョン:ロンドン、その他

  • 包含/除外パターン:サービスカタログ、ナレッジ記事、添付ファイル

ServiceNow コネクタ V2.0/TemplateConfigurationAPI

  • フィールドマッピング

  • ユーザーコンテキストのフィルター

  • Virtual Private Cloud (VPC)

  • すべてのドキュメントを同期/新規、変更、削除されたドキュメントのみを同期

  • ServiceNow インスタンスバージョン:ローマ、サンディエゴ、東京、その他

  • 包含/除外パターン:サービスカタログ、ナレッジ記事、インシデント、添付ファイル

注記

ServiceNow コネクタ V1.0/ ServiceNowConfiguration API Support は、2023 年 6 月までに終了する予定です。 ServiceNow コネクタ V2.0/ TemplateConfiguration API の使用をお勧めします。
