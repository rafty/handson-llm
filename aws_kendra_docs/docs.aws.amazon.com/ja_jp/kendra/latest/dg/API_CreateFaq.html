CreateFaq

Amazon S3 バケットに保存された指定したよくある質問ファイルを使用して、よくある質問のセット (FAQ) を作成します。

よくある質問のインデックスへのよくある質問の追加は、非同期演算です。

Python と Java SDK を使用して FAQ をインデックスに追加する例については、「FAQ ファイルの使用」を参照してください。

リクエストの構文

{
   "ClientToken": "string",
   "Description": "string",
   "FileFormat": "string",
   "IndexId": "string",
   "LanguageCode": "string",
   "Name": "string",
   "RoleArn": "string",
   "S3Path": { 
      "Bucket": "string",
      "Key": "string"
   },
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

リクエストパラメータ

すべてのアクションに共通のパラメータの詳細については、「共通パラメータ」を参照してください。

リクエストは以下の JSON 形式のデータを受け入れます。

ClientToken

よくある質問の作成リクエストを識別するために指定するトークン。CreateFaqRequest同じクライアントトークンを使用した API を複数回呼び出すと、よくある質問は 1 つだけ作成されます。

型: 文字列

長さの制限: 最小長は 1 です。最大長は 100 です。

必須: いいえ

Description

FAQ の説明。

型: 文字列

長さの制限: 最小長は 0 です。最大長は 1,000 です。

Pattern: ^\P{C}*$

必須: いいえ

FileFormat

よくある質問入力ファイルの形式。基本的な CSV 形式、ヘッダーにカスタム属性を含む CSV 形式、カスタム属性を含む JSON 形式から選択できます。

形式は、S3Path パラメータで指定された S3 バケットに保存されているファイルの形式と一致する必要があります。

詳細については、質問と回答を追加するを参照してください。

型: 文字列

有効な値: CSV | CSV_WITH_HEADER | JSON

必須: いいえ

IndexId

よくある質問のインデックスの識別子。

型: 文字列

長さの制限: 最大長は 36 です。

パターン: [a-zA-Z0-9][a-zA-Z0-9-]*

必須: はい

LanguageCode

言語のコード。これにより、よくある質問ドキュメントの言語をサポートできます。英語はデフォルトでサポートされています。コードを含む、サポートされている言語の詳細については、英語以外の言語でドキュメントを追加するを参照してください。

型: 文字列

長さの制限: 最小長は 2 です。最大長は 10 です。

パターン: [a-zA-Z-]*

必須: いいえ

Name

よくある質問の名前。

型: 文字列

長さの制限: 最小長は 1 です。最大長は 100 です。

パターン: [a-zA-Z0-9][a-zA-Z0-9_-]*

必須: はい

RoleArn

よくある質問が含まれている S3 バケットへのアクセス許可を持つIAM ロールの Amazon リソースネーム (ARN)。詳細については、Amazon Kendra の IAM アクセスロールを参照してください。

型: 文字列

長さの制限: 最小長は 0 です。最大長は 1,284 です。

パターン: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

必須: はい

S3Path

S3 でのよくある質問ファイルのパス。

型: S3Path オブジェクト

必須: はい

Tags

よくある質問を識別するキーと値のペアのリスト。タグを使用して、リソースを識別および整理し、リソースへのアクセスを制御できます。

型: Tag オブジェクトの配列

配列メンバー: 最小数は 0 項目です。最大数は 200 項目です。

必須: いいえ

レスポンスの構文

{
   "Id": "string"
}

レスポンス要素

アクションが成功すると、サービスは HTTP 200 レスポンスを返します。

サービスから以下のデータが JSON 形式で返されます。

Id

よくある質問の識別子。

型: 文字列

長さの制限: 最小長は 1 です。最大長は 100 です。

パターン: [a-zA-Z0-9][a-zA-Z0-9_-]*

エラー

すべてのアクションに共通のエラーについては、「一般的なエラー」を参照してください。

AccessDeniedException

このアクションを実行する十分なアクセス権限がありません。必要な権限ポリシーとユーザーアカウントがあることを確認して、もう一度試してください。

HTTP ステータスコード: 400

ConflictException

リクエストに矛盾が発生しました。リソースとの不一致を修正して、もう一度試してください。

HTTP ステータスコード: 400

InternalServerException

Amazon Kendra サービスに使用されている内部サーバーで問題が発生しました。数分待ってからもう一度おSupport。

HTTP ステータスコード: 500

ResourceNotFoundException

使用したいリソースは存在しません。正しいリソースが提供されていることを確認して、もう一度お試しください。

HTTP ステータスコード: 400

ServiceQuotaExceededException

Amazon Kendra サービスに設定された制限を超えました。詳細についてはクォータを参照するか、制限の引き上げについてはSupport にお問い合わせください。

HTTP ステータスコード: 400

ThrottlingException

リクエストは、制限が必要なために実行が拒否されました。リクエストの数を減らして、もう一度お試しください。

HTTP ステータスコード: 400

ValidationException

入力が Amazon Kendra サービスで設定された制約を満たしていません。正しい入力して、もう一度お試しください。

HTTP ステータスコード: 400
