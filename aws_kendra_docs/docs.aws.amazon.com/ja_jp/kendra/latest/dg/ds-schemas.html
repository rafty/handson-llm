Amazon S3テンプレートスキーマ

テンプレート設定の一部として、データソーススキーマを含む JSON を含めます。S3 バケットの名前は、接続設定またはリポジトリエンドポイントの詳細の一部として指定します。また、データソースのタイプをとして指定しS3、その他の必要な構成を指定します。次に、TEMPLATETypeを呼び出すときにとしてを指定しますCreateDataSource。

この開発者ガイドで提供されているテンプレートを使用できます。S3 JSON スキーマ を参照してください。

同期モード すべてのドキュメントを同期してインデックスを更新するか、新規、変更、Amazon Kendra削除されたドキュメントのみを同期するかを指定します。以下を選択することができます。

  • FORCED_FULL_CRAWLデータソースがインデックスと同期するたびに、すべてのコンテンツを新たに再クロールし、既存のコンテンツを置き換えます

  • FULL_CRAWLデータソースがインデックスと同期するたびに、新規、変更、および削除されたコンテンツのみを段階的にクロールする

  • CHANGE_LOGデータソースがインデックスと同期するたびに、新しいコンテンツや変更されたコンテンツだけを段階的にクロールする
