Confluence

Confluence は、プロジェクト計画、ソフトウェア開発、製品管理を共有、保存、作業するために設計された共同作業管理ツールです。Amazon Kendraを使用して、Confluence スペース、ページ (ネストされたページを含む)、ブログ、およびインデックスされたページやブログへのコメントや添付ファイルのインデックスを作成できます。

Amazon KendraConfluence サーバーと Confluence クラウドの両方をサポートしています。

注記

デフォルトでは、ConfluenceAmazon Kendra アーカイブとパーソナルスペースのインデックスを作成しません。データソースの作成時に、インデックス作成を行なうことができます。Amazon Kendraスペースにインデックスを作成したくない場合は、Confluence でプライベートにマークします。

Amazon KendraコンソールAmazon Kendra、API、または TemplateConfigurationAPI のいずれかを使用して Confluence データソースに接続できます。ConfluenceConfiguration

Amazon Kendraには 2 つのバージョンの Confluence コネクタがあります。各バージョンでサポートされる機能は次のとおりです。

コンフルエンスコネクター ConfluenceConfigurationV1.0/API

  • フィールドマッピング

  • ユーザーコンテキストフィルタリング

  • 包含/除外フィルター

  • (Confluence サーバーのみ) Virtual Cloud (VPC)

Confluence コネクタ TemplateConfigurationV2.0/API

  • フィールドマッピング

  • ユーザーコンテキストフィルタリング

  • Virtual Private Cloud (VPC)

  • すべてのドキュメントを同期/新規、変更、または削除されたドキュメントのみを同期

  • 包含/除外パターン

注記

Confluence コネクタ V1.0/ ConfluenceConfiguration API Support は、2023 年 6 月までに終了する予定です。Confluence コネクタ V2.0/ TemplateConfiguration API の使用をお勧めします。
