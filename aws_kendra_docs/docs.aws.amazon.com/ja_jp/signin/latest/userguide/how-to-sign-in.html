へのサインイン方法 AWS

サインインする方法は、AWSAWSユーザーのタイプによって異なります。AWSユーザーにはさまざまなタイプがあります。アカウントのルートユーザー、IAM ユーザー、IAM Identity Center でのユーザー、フェデレーション ID のいずれかです。Builder ID AWS を使用することもできます。詳細については、「ユーザータイプ」を参照してください。

AWS次のいずれかの方法でサインインしてアクセスできます。

  • にサインイン AWS Management Consoleルートユーザーまたは IAM ユーザーとして

  • AWSアクセスポータルにサインインするIAM Identity Center のユーザーとして

  • フェデレーテッド ID としてサインイン

  • からサインイン AWS Command Line Interfaceおよび、API や SDK (ソフトウェア開発キット) のような他のプログラムによる方法

  • AWSビルダー ID でサインイン

新規作成する場合はAWS アカウント、『AWSセットアップガイド』の「パート 1: AWS アカウント 新規セットアップ」を参照してください。
