So melden Sie sich an AWS

Die Art und Weise, wie Sie sich anmelden, AWS hängt davon ab, was für ein AWS -Benutzer Sie sind. Es gibt verschiedene Arten von AWS Benutzern. Sie können ein Konto-Root-Benutzer, ein IAM-Benutzer, ein -Benutzer in IAM Identity Center, eine Verbundidentität sein oder die -Entwickler-ID verwendenAWS. Weitere Informationen finden Sie unter Benutzertypen.

Sie können darauf zugreifen, AWS indem Sie sich mit einer der folgenden Methoden anmelden:

  • Loggen Sie sich ein bei AWS Management Consoleals Root-Benutzer oder IAM-Benutzer

  • Melden Sie sich im AWS Zugangsportal anals Benutzer in IAM Identity Center

  • Melden Sie sich als föderierte Identität an

  • Loggen Sie sich ein über AWS Command Line Interfaceund andere programmgesteuerte Methoden wie API und SDK (Software Development Kit)

  • Melden Sie sich mit der AWS Builder-ID an

Wenn Sie ein neues erstellen möchtenAWS-Konto, lesen Sie Teil 1: Neues AWS einrichten AWS-Konto im Einrichtungsleitfaden.
