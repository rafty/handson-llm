Melden Sie sich im AWS Zugangsportal an

Ein Benutzer im IAM Identity Center ist Mitglied von. AWS Organizations Ein Benutzer in IAM Identity Center kann auf mehrere AWS-Konten und geschäftliche Anwendungen zugreifen, indem er sich mit einer bestimmten Anmelde-URL beim AWS Zugriffsportal anmeldet. Weitere Informationen zur spezifischen Anmelde-URL finden Sie unterAWS-Zugriffsportal.

Bevor Sie sich bei einem AWS-Konto als Benutzer in IAM Identity Center anmelden, sammeln Sie die folgenden erforderlichen Informationen.

  • Unternehmens-Benutzername

  • Unternehmenspasswort

  • Spezifische Anmelde-URL

Anmerkung

Nachdem Sie sich angemeldet haben, ist Ihre AWS Access Portal-Sitzung 8 Stunden lang gültig. Sie müssen sich nach 8 Stunden erneut anmelden.

Um sich beim AWS Zugangsportal anzumelden

  1. Fügen Sie in Ihrem Browserfenster die Anmelde-URL ein, die Sie per E-Mail erhalten haben, z. B. https://your_subdomain.awsapps.com/start Drücken Sie anschließend die Eingabetaste.

  2. Melden Sie sich mit Ihren Unternehmensanmeldeinformationen (wie einem Benutzernamen und -passwort) an.

    Anmerkung

    Wenn Ihr Administrator Ihnen per E-Mail ein Einmalpasswort (OTP) geschickt hat und Sie sich zum ersten Mal anmelden, geben Sie dieses Passwort ein. Nachdem Sie sich angemeldet haben, müssen Sie ein neues Passwort für future Anmeldungen erstellen.

  3. Wenn du nach einem Bestätigungscode gefragt wirst, sieh in deiner E-Mail nach. Kopieren Sie dann den Code und fügen Sie ihn auf der Anmeldeseite ein.

    Anmerkung

    Bestätigungscodes werden normalerweise per E-Mail gesendet, aber die Zustellungsmethode kann variieren. Wenn Sie in Ihrer E-Mail keinen erhalten haben, erkundigen Sie sich bei Ihrem Administrator nach Einzelheiten zu Ihrem Bestätigungscode.

  4. Wenn MFA für Ihren Benutzer im IAM Identity Center aktiviert ist, authentifizieren Sie sich anschließend damit.

  5. Nach der Authentifizierung können Sie auf alle AWS-Konto Anwendungen zugreifen, die im Portal angezeigt werden.

    1. Um sich bei der anzumelden, AWS Management Console wählen Sie die AWS-Konto Kachel und dann das einzelne Konto aus, das Sie verwalten möchten.

      Die Berechtigungen Ihres Benutzers werden zusammen mit Links zum Öffnen der Verwaltungskonsole oder zum Abrufen von Anmeldeinformationen für den Befehlszeilen- oder programmatischen Zugriff angezeigt.

      Wählen Sie den Link Management Console aus.

    2. Um sich bei einer Anwendung anzumelden, wählen Sie das Symbol der Anwendung, auf die Sie zugreifen möchten.

Wenn Sie sich als Benutzer im IAM Identity Center anmelden, erhalten Sie Anmeldeinformationen für den Zugriff auf Ressourcen für einen bestimmten Zeitraum, der als Sitzung bezeichnet wird. Standardmäßig kann ein Benutzer AWS-Konto für eine Stunde bei einem angemeldet sein. Der maximale Wert, den der Administrator für eine Sitzung festlegen kann, beträgt 12 Stunden. Nachdem Ihre Sitzung beendet ist, können Sie sich erneut anmelden.
