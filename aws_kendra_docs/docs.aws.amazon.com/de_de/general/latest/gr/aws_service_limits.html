AWS-Service Quotas

Das AWS-Konto verfügt über Standardkontingente (früher als Limits bezeichnet) für jeden AWS-Service. Wenn nicht anders angegeben, gilt jedes Kontingent spezifisch für eine Region. Sie können Erhöhungen für einige Kontingente beantragen und andere Kontingente können nicht erhöht werden.

Service Quotas ist ein AWS-Service, der Ihnen hilft, Ihre Kontingente für viele AWS-Services von einem einzigen Standort aus zu verwalten. Sie können über die Service–Quotas-Konsole Kontingentwerte abfragen und außerdem Kontingenterhöhungen anfordern.

AWS Support kann Ihre Abfragen genehmigen, ablehnen oder teilweise genehmigen.

So zeigen Sie Servicekontingente an

Sie können Servicekontingente folgendermaßen anzeigen:

  • Öffnen Sie die Seite Service-Endpunkte und -kontingente der Dokumentation, suchen Sie den Service-Namen und klicken Sie auf den Link, um die Seite für diesen Service aufzurufen. Um die Servicekontingente für alle AWS-Services in der Dokumentation anzuzeigen, ohne die Seiten wechseln zu müssen, zeigen Sie die Informationen der Seite Service-Endpunkte und -kontingente stattdessen in der PDF-Datei an.

  • Öffnen Sie die Service Quotas-Konsole. Wählen Sie im Navigationsbereich AWS-Services und dann einen Service aus.

  • Verwenden Sie dieAWS CLI Befehle list-service-quotasund list-aws-default-service-quotas.

So fordern Sie eine Kontingenterhöhung an

Sie können eine Kontingenterhöhung über Service Quotas mit einer der folgenden Optionen anfordern. Erhöhungen werden nicht sofort gewährt. Es kann einige Tage dauern, bis Ihre Erhöhung wirksam wird.

  • Öffnen Sie die Service Quotas-Konsole. Wählen Sie im Navigationsbereich AWS-Services. Wählen Sie einen Service aus, wählen Sie ein Kontingent aus, und folgen Sie den Anweisungen, um eine Kontingenterhöhung anzufordern. Weitere Informationen finden Sie unter Anfordern einer Kontingenterhöhung im Service Quotas-Benutzerhandbuch.

  • Verwenden Sie den AWS CLI-Befehl request-service-quota-increase.

Wenn ein Service in Service Quotas noch nicht in verfügbar ist, erstellen Sie einen Fall zur Erhöhung des Servicelimits mit dem AWS Support Center Console. Wenn der Service in Service Quotas verfügbar ist, empfehlen wir Ihnen dringend, Service Quotas zu verwenden, anstatt einen Support-Fall zu erstellen.
