Endpunkte und Kontingente von Amazon Kendra

Im Folgenden werden die Service-Endpunkte und -kontingente für diesen Service beschrieben. Um programmgesteuert eine Verbindung zu einem AWS-Service herzustellen, verwenden Sie einen Endpunkt. Zusätzlich zu den standardmäßigen AWS-Endpunkten bieten einige AWS-Services FIPS-Endpunkte in ausgewählten Regionen. Weitere Informationen finden Sie unter AWS-Service-Endpunkte. Servicekontingente, auch als Limits bezeichnet, sind die maximale Anzahl von Serviceressourcen oder -vorgängen für Ihr AWS-Konto. Weitere Informationen finden Sie unter AWS-Service Quotas.
