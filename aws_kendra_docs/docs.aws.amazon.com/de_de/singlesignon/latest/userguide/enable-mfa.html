Multifaktor-Authentifizierung

Wenn Sie die Multi-Factor Authentication (MFA) aktivieren, müssen sich Benutzer mit ihrem Benutzernamen und Passwort am AWS Zugangsportal anmelden. Das ist der erste Faktor, den sie wissen. Benutzer müssen sich außerdem entweder mit einem Code oder einem Sicherheitsschlüssel anmelden. Dies ist der zweite Faktor, etwas, das sie haben oder etwas, das sie sind. Der zweite Faktor könnte entweder ein Authentifizierungscode sein, der von ihrem Mobilgerät generiert wird, oder alternativ durch Tippen auf einen mit ihrem Computer verbundenen Sicherheitsschlüssel. Zusammengenommen sorgen diese verschiedenen Faktoren für mehr Sicherheit, da sie den unbefugten Zugriff auf Ihre AWS Ressourcen verhindern, sofern keine gültige MFA-Herausforderung erfolgreich abgeschlossen wurde. Jeder Benutzer kann bis zu acht MFA-Geräte registrieren.

Wichtig

Aus Sicherheitsgründen empfehlen wir dringend, die Multifaktor-Authentifizierung zu aktivieren. MFA bietet eine einfache und sichere Möglichkeit, zusätzlich zum Standardauthentifizierungsmechanismus mit Benutzername und Passwort eine zusätzliche Schutzebene hinzuzufügen.
