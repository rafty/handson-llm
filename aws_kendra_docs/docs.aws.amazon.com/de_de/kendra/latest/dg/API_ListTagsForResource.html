ListTagsForResource

Ruft eine Liste von Tags ab, die mit einer bestimmten Ressource assoziiert sind. Indizes, FAQs und Datenquellen können mit Tags verknüpft sein.

Anforderungssyntax

{
   "ResourceARN": "string"
}
