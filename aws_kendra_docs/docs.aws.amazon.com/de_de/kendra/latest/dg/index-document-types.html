Typen von -Dokumenten

Ein Index kann mehrere Dokumente und verschiedene Dokumenttypen enthalten. Ein Index kann auch sowohl strukturierten als auch unstrukturierten Text enthalten.

Strukturierter Text

  • Häufig gestellte Fragen und Antworten

Dokumente im Format „Häufig gestellte Fragen“ und „Antworten“ werden verwendet, um Fragen wie „Wie hoch ist die Space Needle?“ zu beantworten. Sie können mehrere Fragen angeben, die dieselbe Antwort zurückgeben. Sie geben die Fragen und Antworten in einer CSV-Datei an, die in einemAmazon S3 Bucket gespeichert ist.

Wenn Sie eine JSON-Datei für Ihre häufig gestellten Fragen und Antworten verwenden möchten, werden JSON-Dateien als Klartext behandelt und sind eine Art von unstrukturiertem Text.

Ein Beispiel finden Sie unter Fragen und Antworten direkt zu einem Index hinzufügen.

Unstrukturierter Text

Amazon Kendraunterstützt die folgenden Arten von unstrukturiertem Text:

  • HTML-Dateien

  • Microsoft PowerPoint (PPT) -Präsentationen

  • MS WORD-Dokumente

  • Dokumente im Klartext

  • PDF-Dateien

  • Dateien mit kommagetrennten Werten (CSV)

  • Microsoft Excel-Dateien (MS EXCEL)

  • XML-Dateien

  • JSON-Dateien

  • Markdown-Dokumentationsdateien (MD)

  • RTF-Dateien (Rich Text Format)

  • Extensible Stylesheet Language Transformation (XSLT) -Dateien

Sie können Ihrem Index auf drei Arten unstrukturierte Dokumente hinzufügen:

  • Verwendung der BatchPutDocumentAPI.

  • Als Datenquelle. Weitere Informationen finden Sie unter Hinzufügen von Dokumenten aus einer Datenquelle.

  • Als Binärdaten. Weitere Informationen finden Sie unter Hinzufügen von Dokumenten mit der API.
