Dokumentattribute oder Felder

Einem Dokument sind Attribute oder Felder zugeordnet. Felder eines Dokuments sind die Eigenschaften eines Dokuments oder was in der Struktur eines Dokuments enthalten ist. Beispielsweise kann jedes Ihrer Dokumente Titel, Haupttext und Autor enthalten. Sie können auch Ihre eigenen benutzerdefinierten Felder zu Ihren Dokumenten hinzufügen. Benutzerdefinierte Felder sind Attribute, die Sie für Ihre eigenen Bedürfnisse festlegen. Wenn Ihr Index beispielsweise nach Steuerdokumenten sucht, können Sie ein benutzerdefiniertes Feld für den Typ des Steuerdokuments angeben, z. B. W-2, 1099 usw.

Bevor Sie ein Dokumentfeld in einer Abfrage verwenden können, muss es einem Indexfeld zugeordnet werden. Beispielsweise kann das Titelfeld dem Feld zugeordnet werden_document_title. Weitere Informationen finden Sie unter Zuweisen von Feldern. Um ein neues Feld hinzuzufügen, müssen Sie ein Indexfeld erstellen, dem das Feld zugeordnet werden soll. Sie erstellen Indexfelder mithilfe der Konsole oder mithilfe der UpdateIndexAPI.

Sie können Dokumentfelder verwenden, um Antworten zu filtern und Vorschläge für facettierte Suchen zu machen. Sie können beispielsweise eine Antwort filtern, um nur eine bestimmte Version eines Dokuments zurückzugeben, oder Sie können Suchanfragen filtern, um nur Steuerdokumente vom Typ 1099 zurückzugeben, die dem Suchbegriff entsprechen. Weitere Informationen finden Sie unter Filtern von Abfragen.

Sie können Dokumentfelder auch verwenden, um die Abfrageantwort manuell zu optimieren. Sie können beispielsweise die Wichtigkeit des Titelfeldes erhöhen, um die Gewichtung zu erhöhen, die dem FeldAmazon Kendra zugewiesen wird, wenn bestimmt wird, welche Dokumente in der Antwort zurückgegeben werden sollen. Weitere Informationen finden Sie unter Optimieren der Suchrelevanz.

Wenn Sie ein Dokument direkt zu einem Index hinzufügen, geben Sie die Felder im Eingabeparameter Dokument für dieBatchPutDocument API an. Sie geben die benutzerdefinierten Feldwerte in einemDocumentAttribute Objektarray an. Wenn Sie eine Datenquelle verwenden, hängt die Methode, mit der Sie die Dokumentfelder hinzufügen, von der Datenquelle ab. Weitere Informationen finden Sie unter Erstellen benutzerdefinierter Dokumentfelder.
