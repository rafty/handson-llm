TagResource

Fügt das angegebene Tag dem angegebenen Index, den häufig gestellten Fragen oder der Datenquellenressource hinzu. Wenn das Tag bereits existiert, wird der vorhandene Wert durch den neuen Wert ersetzt.

Anforderungssyntax

{
   "ResourceARN": "string",
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

Anfrageparameter

Informationen zu den Parametern, die allen Aktionen gemeinsam sind, finden Sie unter Allgemeine Parameter.

Die Anforderung akzeptiert die folgenden Daten im JSON-Format.

ResourceARN

Der Amazon-Ressourcenname (ARN) des Index, der häufig gestellten Fragen oder der zu kennzeichnenden Datenquelle.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Die Die Die Die Maximale beträgt 1011 Zeichen.

Erforderlich: Ja

Tags

Eine Liste von Tag-Schlüsseln, die dem Index, den häufig gestellten Fragen oder der Datenquelle hinzugefügt werden sollen. Wenn ein Tag bereits existiert, wird der vorhandene Wert durch den neuen Wert ersetzt.

Typ: Array von Tag-Objekten

Array-Mitglieder: Die Mindestanzahl beträgt 0 Elemente. Die Die Die Die Die Die Die Die Die maximale Anzahl beträgt 50

Erforderlich: Ja

Antwortelemente

Wenn die Aktion erfolgreich ist, gibt der Dienst eine HTTP 200-Antwort mit leerem HTTP-Textinhalt zurück.

Fehler

Hinweise zu den Fehlern, die allen Aktionen gemeinsam sind, finden Sie unterHäufige Fehler.

AccessDeniedException

Sie haben keinen ausreichenden Zugriff, um diese Aktion auszuführen. Bitte stellen Sie sicher, dass Sie über die erforderlichen Berechtigungsrichtlinien und Benutzerkonten verfügen, und versuchen Sie es erneut.

HTTP Status Code: 400

InternalServerException

Es ist ein Problem mit dem internen Server aufgetreten, der für Ihren Amazon Kendra-Service verwendet wird. Bitte warten Sie ein paar Minuten und versuchen Sie es erneut, oder wenden Sie sich an den Support, um Hilfe zu erhalten.

HTTP Status Code: 500

ResourceUnavailableException

Die Ressource, die Sie verwenden möchten, ist nicht verfügbar. Bitte überprüfen Sie, ob Sie die richtige Ressource angegeben haben, und versuchen Sie es erneut.

HTTP Status Code: 400

ThrottlingException

Die Anforderung wurde aufgrund der Drosselung von Anforderungen abgelehnt. Bitte reduzieren Sie die Anzahl der Anfragen und versuchen Sie es erneut.

HTTP Status Code: 400

ValidationException

Die Eingabe erfüllt nicht die vom Amazon Kendra-Service festgelegten Einschränkungen. Bitte geben Sie die richtige Eingabe und versuchen Sie es erneut.

HTTP Status Code: 400
