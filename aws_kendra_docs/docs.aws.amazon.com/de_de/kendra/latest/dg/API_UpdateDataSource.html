Anfrageparameter

Informationen zu den Parametern, die allen Aktionen gemeinsam sind, finden Sie unter Allgemeine Parameter.

Die Anforderung akzeptiert die folgenden Daten im JSON-Format.

Configuration

Konfigurationsinformationen, die für den Datenquellenkonnektor aktualisiert werden sollen.

Typ: DataSourceConfiguration Objekt

Required: No

CustomDocumentEnrichmentConfiguration

Konfigurationsinformationen, die für die Änderung der Metadaten und des Inhalts von Dokumenten während der Dokumentenerfassung verwendet werden sollen.

Weitere Informationen zum Erstellen, Ändern und Löschen von Dokumentmetadaten oder zum Vornehmen anderer Inhaltsänderungen bei der Aufnahme von Dokumenten in Amazon Kendra finden Sie unter Anpassen von Dokumentmetadaten während des Erfassungsprozesses.

Typ: CustomDocumentEnrichmentConfiguration Objekt

Required: No

Description

Eine neue Beschreibung für den Datenquellenkonnektor.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 0. Höchstlänge = 1 000 Zeichen.

Pattern: ^\P{C}*$

Erforderlich: Nein

Id

Der Bezeichner des Datenquellenkonnektor, der aktualisiert werden soll.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Höchstlänge = 100 Zeichen.

Pattern: [a-zA-Z0-9][a-zA-Z0-9_-]*

Erforderlich: Ja

IndexId

Der Bezeichner des Index, der für den Datenquellenkonnektor verwendet wird.

Typ: Zeichenfolge

Längenbeschränkungen: Maximale Länge beträgt 36Zeichen.

Pattern: [a-zA-Z0-9][a-zA-Z0-9-]*

Erforderlich: Ja

LanguageCode

Der Code für eine Sprache, die für den Datenquellenkonnektor verwendet werden soll. Auf diese Weise können Sie bei der Aktualisierung der Datenquelle eine Sprache für alle Dokumente unterstützen. Englisch wird standardmäßig unterstützt. Weitere Informationen zu unterstützten Sprachen, einschließlich ihrer Codes, finden Sie unter Hinzufügen von Dokumenten in anderen Sprachen als Englisch.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge beträgt 2 2 Zeichen. Höchstlänge beträgt 10 Zeichen.

Pattern: [a-zA-Z-]*

Erforderlich: Nein

Name

Ein neuer Name für den Datenquellenkonnektor.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Höchstlänge = 1000 Zeichen.

Pattern: [a-zA-Z0-9][a-zA-Z0-9_-]*

Erforderlich: Nein

RoleArn

Der Amazon-Ressourcenname (ARN) einer Rolle mit der Berechtigung für den Zugriff auf die Datenquelle und die benötigten Ressourcen. Weitere Informationen finden Sie unter IAM-Rollen für Amazon Kendra.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 0. Höchstlänge beträgt 1284 Zeichen.

Pattern: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Erforderlich: Nein

Schedule

Der Synchronisierungszeitplan, der für den Datenquellenkonnektor aktualisiert werden soll.

Typ: Zeichenfolge

Required: No

VpcConfiguration

Konfigurationsinformationen für eine Amazon Virtual Private Cloud zur Verbindung mit Ihrer Datenquelle. Weitere Informationen finden Sie unter Konfigurieren einer VPC.

Typ: DataSourceVpcConfiguration Objekt

Required: No

Antwortelemente

Wenn die Aktion erfolgreich ist, gibt der Dienst eine HTTP 200-Antwort mit leerem HTTP-Textinhalt zurück.

Fehler

Hinweise zu den Fehlern, die allen Aktionen gemeinsam sind, finden Sie unterHäufige Fehler.

AccessDeniedException

Sie haben keinen ausreichenden Zugriff zum Ausführen dieser Aktion. Bitte stellen Sie sicher, dass Sie über die erforderlichen Berechtigungsrichtlinien und Benutzerkonten verfügen, und versuchen Sie es erneut.

HTTP Status Code: 400

ConflictException

Bei der Anfrage ist ein Konflikt aufgetreten. Bitte korrigieren Sie alle Unstimmigkeiten mit Ihren Ressourcen und versuchen Sie es erneut.

HTTP Status Code: 400

InternalServerException

Es ist ein Problem mit dem internen Server aufgetreten, der für Ihren Amazon Kendra-Service verwendet wird. Bitte warten Sie ein paar Minuten und versuchen Sie es erneut, oder wenden Sie sich an den Support, um Hilfe zu erhalten.

HTTP Status Code: 500

ResourceNotFoundException

Die Ressource, die Sie verwenden möchten, existiert nicht. Bitte überprüfen Sie, ob Sie die richtige Ressource angegeben haben, und versuchen Sie es erneut.

HTTP Status Code: 400

ThrottlingException

Die Anforderung wurde aufgrund der Drosselung von Anforderungen abgelehnt. Reduzieren Sie die Anzahl der Anfragen und versuchen Sie es erneut.

HTTP Status Code: 400

ValidationException

Die Eingabe erfüllt nicht die vom Amazon Kendra-Dienst. Bitte geben Sie die korrekte Eingabe und versuchen Sie es erneut.

HTTP Status Code: 400
