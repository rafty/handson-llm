UntagResource

Entfernt ein Tag aus einem Index, häufig gestellten Fragen oder einer Datenquelle.

Anforderungssyntax

{
   "ResourceARN": "string",
   "TagKeys": [ "string" ]
}

Anfrageparameter

Informationen zu den Parametern, die allen Aktionen gemeinsam sind, finden Sie unter Allgemeine Parameter.

Die Anforderung akzeptiert die folgenden Daten im JSON-Format.

ResourceARN

Der Amazon-Ressourcenname (ARN) des Index, der häufig gestellten Fragen oder der Datenquelle, aus der das Tag entfernt sollen.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Die maximale Länge von 1011.

Erforderlich: Ja

TagKeys

Eine Liste von Tag-Schlüsseln, die aus dem Index, den häufig gestellten Fragen oder der Datenquelle entfernt sollen. Wenn für die Ressource kein Tag-Schlüssel vorhanden ist, wird er ignoriert.

Typ: Zeichenfolge-Array

Array-Mitglieder: Die Mindestanzahl beträgt 0 Elemente. Die maximale Anzahl von 200 Elementen.

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 128 Zeichen.

Erforderlich: Ja

Antwortelemente

Wenn die Aktion erfolgreich ist, gibt der Dienst eine HTTP 200-Antwort mit leerem HTTP-Textinhalt zurück.

Fehler

Hinweise zu den Fehlern, die allen Aktionen gemeinsam sind, finden Sie unterHäufige Fehler.

AccessDeniedException

Sie haben keinen ausreichenden Zugriff, um diese Aktion auszuführen. Bitte stellen Sie sicher, dass Sie über die erforderlichen Berechtigungsrichtlinien und Benutzerkonten verfügen, und versuchen Sie es erneut.

HTTP Status Code: 400

InternalServerException

Ein Problem ist mit dem internen Server aufgetreten, der für Ihren Amazon Kendra-Service verwendet wird. Bitte warten Sie ein paar Minuten und versuchen Sie es erneut, oder wenden Sie sich an den Support, um Hilfe zu erhalten.

HTTP Status Code: 500

ResourceUnavailableException

Die Ressource, die Sie verwenden möchten, ist nicht verfügbar. Bitte überprüfen Sie, ob Sie die richtige Ressource angegeben haben, und versuchen Sie es erneut.

HTTP Status Code: 400

ThrottlingException

Die Anforderung wurde aufgrund der Drosselung von Anforderungen abgelehnt. Bitte reduzieren Sie die Anzahl der Anfragen und versuchen Sie es erneut.

HTTP Status Code: 400

ValidationException

Die Eingabe erfüllt nicht die vom Amazon Kendra-Service definierten Einschränkungen. Bitte geben Sie die richtige Eingabe und versuchen Sie es erneut.

HTTP Status Code: 400
