Ihre IAM-Identity-Center-Identitätsquelle ändern

Warnung

Das Ändern Ihrer Identitätsquelle in den IAM Identity Center-Einstellungen kann sich auf die Beibehaltung von Benutzer- und Gruppeninformationen auswirken. Um dies sicher durchzuführen, wird empfohlen, dass Sie die Überlegungen zur Änderung Ihrer Identitätsquelle lesen. Wenn Sie Ihre Identitätsquelle ändern, wird eine neue Identitätsquellen-ID generiert. Vergewissern Sie sich, dass Sie die richtige ID verwenden, bevor Sie den Modus aufAWS_SSO In setzen UserGroupResolutionConfiguration.

Um die IAM-Identity-Center-Identitätsquelle zu ändern

  1. Öffnen Sie die IAM Identity Center>-Konsole.

  2. Wählen Sie Settings (Einstellungen) aus.

  3. Wählen Sie auf der Seite Einstellungen unter Identitätsquelle die Option Ändern aus.

  4. Wählen Sie auf der Seite „Identitätsquelle ändern“ Ihre bevorzugte Identitätsquelle aus und klicken Sie dann auf Weiter.
