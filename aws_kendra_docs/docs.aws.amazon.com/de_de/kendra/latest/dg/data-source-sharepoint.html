Microsoft SharePoint

SharePoint ist ein kollaborativer Dienst zum Erstellen von Websites, mit dem Sie Webinhalte anpassen und Seiten, Websites, Dokumentbibliotheken und Listen erstellen können. Sie können es verwendenAmazon Kendra, um Ihre SharePoint Datenquelle zu indexieren.

Amazon Kendraunterstützt derzeit SharePoint Online und SharePoint Server (Versionen 2013, 2016, 2019 und Subscription Edition).

Sie können entweder überAmazon Kendra die Amazon KendraKonsole, die API oder die TemplateConfigurationAPI eine Verbindung zu Ihrer SharePoint Datenquelle herstellen. SharePointConfiguration

Amazon Kendrahat zwei Versionen des SharePoint Connectors. Zu den unterstützten Funktionen jeder Version gehören:

SharePoint Anschluss V1.0//SharePointConfigurationAPI

  • Änderungsprotokoll

  • Feldzuordnungen

  • Filterung des Benutzerkontextes

  • Einschluss-/Ausschlussfilter

  • Virtual Private Cloud (VPC)

SharePoint Anschluss V2.0//TemplateConfigurationAPI

  • Feldzuordnungen

  • Filterung des Benutzerkontextes

  • Einschluss-/Ausschlussfilter

  • Virtual Private Cloud (VPC)

  • Alle Dokumente synchronisieren/Nur neue oder geänderte Dokumente synchronisieren/Nur neue, geänderte oder gelöschte Dokumente synchronisieren

Anmerkung

Die Support für SharePoint Connector V1.0/ SharePointConfiguration API wird voraussichtlich im Juni 2023 enden. Wir empfehlen die Verwendung von SharePoint Connector V2.0/ TemplateConfiguration API.

Informationen zur Fehlerbehebung bei IhremAmazon Kendra SharePoint Datenquellenconnector finden Sie unterFehlerbehebung bei Datenquellen.
