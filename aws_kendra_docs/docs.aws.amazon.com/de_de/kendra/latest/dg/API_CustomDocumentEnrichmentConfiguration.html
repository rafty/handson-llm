CustomDocumentEnrichmentConfiguration

Stellt die Konfigurationsinformationen zum Ändern der Metadaten und des Inhalts von Dokumenten während der Dokumentenerfassung bereit.

Weitere Informationen finden Sie unter Anpassen der Metadaten von Dokumenten während des Erfassungsprozesses.

Inhalt

InlineConfigurations

Konfigurationsinformationen zur Änderung von Dokumentattributen oder Metadatenfeldern und -inhalten bei der Erfassung von Dokumenten in Amazon Kendra.

Typ: Array von InlineCustomDocumentEnrichmentConfiguration-Objekten

Array-Mitglieder: Die Mindestanzahl beträgt 0 Elemente. Die maximale Anzahl beträgt 100 Elemente.

Required: No

PostExtractionHookConfiguration

Konfigurationsinformationen zum Aufrufen einer Lambda-Funktion in AWS Lambda auf den strukturierten Dokumenten mit ihren Metadaten und extrahiertem Text. Sie können eine Lambda-Funktion verwenden, um erweiterte Logik zum Erstellen, Ändern oder Löschen von Dokumentmetadaten und -inhalten anzuwenden. Weitere Informationen finden Sie unter Erweiterte Datenmanipulation.

Typ: HookConfiguration Objekt

Required: No

PreExtractionHookConfiguration

Konfigurationsinformationen zum Aufrufen einer Lambda-Funktion in AWS Lambda auf den original oder unformatierten Dokumenten mit ihren Metadaten und extrahiertem Text. Sie können eine Lambda-Funktion verwenden, um erweiterte Logik zum Erstellen, Ändern oder Löschen von Dokumentmetadaten und -inhalten anzuwenden. Weitere Informationen finden Sie unter Erweiterte Datenmanipulation.

Typ: HookConfiguration Objekt

Required: No

RoleArn

Der Amazon-Ressourcenname (ARN) einer Rolle mit der Berechtigung, PreExtractionHookConfiguration und PostExtractionHookConfiguration zum Ändern von Metadaten und Inhalten von Dokumenten während des Dokumenten-Erfassungsprozesse auszuführen. Weitere Informationen finden Sie unter IAM-Rollen für Amazon Kendra.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 0. Maximale Länge beträgt 2048 Zeichen.

Pattern: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Erforderlich: Nein
