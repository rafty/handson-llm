QueryResultItem

Ein einzelnes Abfrageergebnis.

Ein Abfrageergebnis enthält Informationen über ein Dokument, das von der Abfrage zurückgegeben wurde. Dazu gehören der ursprüngliche Speicherort des Dokuments, eine Liste der dem Dokument zugewiesenen Attribute und relevanter Text aus dem Dokument, der die Anfrage erfüllt.

Inhalt

AdditionalAttributes

Ein oder mehrere zusätzliche Attribute, die dem Abfrageergebnis zugeordnet sind.

Typ: Array von AdditionalResultAttribute-Objekten

Required: No

DocumentAttributes

Eine Reihe von Dokumentattributen, die einem Dokument in den Suchergebnissen zugewiesen wurden. Zum Beispiel der Autor des Dokuments (_author) oder die Quell-URI (_source_uri) des Dokuments.

Typ: Array von DocumentAttribute-Objekten

Required: No

DocumentExcerpt

Ein Auszug aus dem Text im Dokument. Enthält Informationen zur Markierung der relevanten Begriffe im Auszug.

Typ: TextWithHighlights Objekt

Required: No

DocumentId

Der Bezeichner für das Dokument.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 2048 Zeichen.

Required: No

DocumentTitle

Der Titel des Dokuments. Enthält den Text des Titels und Informationen zur Hervorhebung der relevanten Begriffe im Titel.

Typ: TextWithHighlights Objekt

Required: No

DocumentURI

Der URI des ursprünglichen Speicherorts des Dokuments.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 2048 Zeichen.

Pattern: ^(https?|ftp|file):\/\/([^\s]*)

Erforderlich: Nein

FeedbackToken

Ein Token, das ein bestimmtes Ergebnis einer bestimmten Abfrage identifiziert. Verwenden Sie dieses Token, um Klickfeedback für das Ergebnis zu geben. Weitere Informationen finden Sie unter Feedback einreichen.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 2048 Zeichen.

Pattern: ^\P{C}*.\P{C}*$

Erforderlich: Nein

Format

Wenn dasType Of-Dokument in der Antwort istANSWER, handelt es sich entweder um eineTABLE Antwort oder um eineTEXT Antwort. Wenn es sich um eine Tabellenantwort handelt, wird ein Tabellenauszug zurückgegebenTableExcerpt. Wenn es sich um eine Textantwort handelt, wird ein Textauszug zurückgegebenDocumentExcerpt.

Typ: Zeichenfolge

Zulässige Werte: TABLE | TEXT

Required: No

Id

Der Bezeichner für das Abfrageergebnis.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 73 Zeichen.

Required: No

ScoreAttributes

Gibt an, dass Amazon Kendra davon überzeugt ist, dass ein Ergebnis mit der von Ihnen angegebenen Abfrage übereinstimmt. Jedes Ergebnis wird in eine Ablage gelegt, die die KonfidenzVERY_HIGHHIGH,MEDIUM und angibtLOW. Anhand der Bewertung können Sie feststellen, ob eine Antwort die für Ihre Bewerbung erforderliche Sicherheit erfüllt.

Das Feld ist nurLOW dann auf gesetzt, wenn dasType Feld auf gesetzt istDOCUMENT und Amazon Kendra sich nicht sicher ist, ob das Ergebnis mit der Abfrage übereinstimmt.

Typ: ScoreAttributes Objekt

Required: No

TableExcerpt

Ein Auszug aus einer Tabelle in einem Dokument.

Typ: TableExcerpt Objekt

Required: No

Type

Die Art des Dokuments in der Antwort. Eine Antwort könnte beispielsweise eine Frage-Antwort enthalten, die für die Anfrage relevant ist.

Typ: Zeichenfolge

Zulässige Werte: DOCUMENT | QUESTION_ANSWER | ANSWER

Erforderlich: Nein
