Dokumente in anderen Sprachen als Englisch hinzufügen

Sie können Dokumente in mehreren Sprachen indexieren. Wenn Sie keine Sprache angeben,Amazon Kendra indexiert Dokumente standardmäßig in Englisch. Sie fügen den Sprachcode für ein Dokument als Feld in die Metadaten des Dokuments ein. Weitere Informationen zu dem Feld für ein Dokument finden Sie unter_language_code Feldzuordnungen und Benutzerdefinierte Attribute.

Sie können den Sprachcode für alle Ihre Dokumente in Ihrer Datenquelle angeben, wenn Sie anrufen CreateDataSource. Wenn für ein Dokument kein Sprachcode in einem Metadatenfeld angegeben ist, wird das Dokument mit dem Sprachcode indexiert, der für alle Dokumente auf Datenquellenebene angegeben ist. In der Konsole können Sie Dokumente in einer unterstützten Sprache nur auf Datenquellenebene indexieren. Gehen Sie zu Datenquellen und dann zur Seite Datenquellendetails angeben und wählen Sie eine Sprache aus der Dropdown-Liste Sprache aus.

Sie können Dokumente auch in einer unterstützten Sprache suchen oder abfragen. Weitere Informationen finden Sie unter Suchen in Sprachen.

Die folgenden Sprachen und ihre Codes werden unterstützt (Englisch oderen wird standardmäßig unterstützt, wenn Sie keine Sprache angeben). Diese Tabelle enthält Sprachen, die eine vollständige semantische SucheAmazon Kendra unterstützen, sowie Sprachen, die nur den einfachen Stichwortabgleich unterstützen. Sprachen, die eine vollständige semantische Suche unterstützen, sind mit einem Sternchen gekennzeichnet und in der folgenden Tabelle fett gedruckt. Englisch (Standardsprache) wird auch bei vollständiger semantischer Suche unterstützt.

*Die semantische Suche wird für die Sprache unterstützt.

Für Sprachen, die die semantische Suche unterstützen, werden die folgenden Funktionen unterstützt.

  • Relevanz von Dokumenten, die über den einfachen Stichwortabgleich hinausgehen.

  • Häufig gestellte Fragen, die über den einfachen Keyword-Matching

  • Extrahieren von Antworten aus Dokumenten auf der GrundlageAmazon Kendra des Leseverständnisses.

  • Konfidenzbereiche (sehr hoch, hoch, mittel und niedrig) der Suchergebnisse.

Für Sprachen, die keine semantische Suche unterstützen, wird der einfache Stichwortabgleich für die Relevanz von Dokumenten und häufig gestellte Fragen unterstützt.

Synonyme (einschließlich benutzerdefinierter Synonyme), inkrementelles Lernen und Feedback sowie Abfragevorschläge werden nur für Englisch (Standardsprache) unterstützt.
