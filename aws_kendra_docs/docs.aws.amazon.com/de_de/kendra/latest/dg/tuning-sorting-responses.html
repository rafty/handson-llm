Optimieren der Antworten

Sie können die Auswirkung eines Felds oder Attributs auf die Suchrelevanz durch Relevanzoptimierung ändern. Um die Relevanzoptimierung schnell zu testen, verwenden Sie die Abfrage-API, um Optimierungskonfigurationen in der Abfrage zu übergeben. Dann können Sie die verschiedenen Suchergebnisse sehen, die Sie aus verschiedenen Konfigurationen erhalten. Die Relevanzoptimierung auf Abfrageebene wird in der Konsole nicht unterstützt. Sie können auch Felder oder Attribute optimieren, die nur aufStringList Indexebene vom Typ sind. Weitere Informationen finden Sie unter Suchrelevanz einstellen.

Standardmäßig werden Abfrageantworten nach dem Relevanzwert sortiert, der für jedes Ergebnis in der AntwortAmazon Kendra bestimmt wird.

Sie können die Ergebnisse für jedes integrierte oder benutzerdefinierte Attribut/Feld der folgenden Typen optimieren:

  • Wert des Datums

  • Langer Wert

  • String-Wert

Attribute des folgenden Typs können nicht sortiert werden:

  • String-Listenwerte

Ranken und Optimieren von Dokumentergebnissen (AWSSDK)

Setzen Sie denSearchable Parameter auf true, um die Konfiguration der Dokumentmetadaten zu verbessern.

Um ein Attribut in einer Abfrage zu optimieren, legen Sie denDocumentRelevanceOverrideConfigurations Parameter derQuery API fest und geben Sie den Namen des zu optimierenden Attributs an.

Das folgende JSON-Beispiel zeigt einDocumentRelevanceOverrideConfigurations Objekt, das die Optimierung für das Attribut „department“ im Index überschreibt.

"DocumentRelevanceOverrideConfigurations" : [
    "Name": "department",
    "Relevance": {
        "Importance": 1,
        "ValueImportanceMap": {
            "IT": 3,
            "HR": 7
        }
    }
]

Antworten sortieren

Amazon Kendraverwendet das Sortierattribut oder -feld als Teil der Kriterien für die von der Abfrage zurückgegebenen Dokumente. Beispielsweise enthalten die von einer nach „_created_at“ sortierten Abfrage zurückgegebenen Ergebnisse möglicherweise nicht dieselben Ergebnisse wie eine nach „_version“ sortierte Abfrage.

Standardmäßig werden Abfrageantworten nach dem Relevanzwert sortiert, der für jedes Ergebnis in der AntwortAmazon Kendra bestimmt wird. Um die Sortierreihenfolge zu ändern, machen Sie ein Dokumentattribut sortierbar und konfigurieren SieAmazon Kendra es dann so, dass dieses Attribut zum Sortieren von Antworten verwendet wird.

Sie können Ergebnisse nach jedem integrierten oder benutzerdefinierten Attribut/Feld der folgenden Typen sortieren:

  • Wert des Datums

  • Langer Wert

  • String-Wert

Attribute des folgenden Typs können nicht sortiert werden:

  • String-Listenwerte

Sie können in jeder Abfrage nur nach einem Dokumentattribut/Feld sortieren. Abfragen liefern 100 Ergebnisse. Wenn es weniger als 100 Dokumente gibt, für die das Sortierattribut gesetzt ist, werden Dokumente ohne Wert für das Sortierattribut am Ende der Ergebnisse zurückgegeben, sortiert nach Relevanz für die Abfrage.

Um Dokumentergebnisse zu sortieren (AWSSDK)

  1. Um die UpdateIndexAPI zu verwenden, um ein Attribut sortierbar zu machen, setzen Sie denSortable Parameter auftrue. Das folgende JSON-Beispiel verwendetDocumentMetadataConfigurationUpdates, um dem Index ein Attribut namens „Department“ hinzuzufügen und es sortierbar zu machen.

    "DocumentMetadataConfigurationUpdates": [
   {
       "Name": "Department",
       "Type": "STRING_VALUE",
       "Search": {
           "Sortable": "true"
       }
   }
]

  2. Um ein sortierbares Attribut in einer Abfrage zu verwenden, legen Sie denSortingConfiguration Parameter der Abfrage-API fest. Geben Sie den Namen des zu sortierenden Attributs an und geben Sie an, ob die Antwort in aufsteigender oder absteigender Reihenfolge sortiert werden soll.

    Das folgende JSON-Beispiel zeigt denSortingConfiguration Parameter, den Sie verwenden, um die Ergebnisse einer Abfrage nach dem Attribut „Department“ in aufsteigender Reihenfolge zu sortieren.

       "SortingConfiguration": { 
      "DocumentAttributeKey": "Department",
      "SortOrder": "ASC"
   }
So sortieren Sie Dokumentergebnisse (Konsole)

  1. Um ein Attribut in der Konsole sortierbar zu machen, wählen Sie Sortierbar in der Attributdefinition. Sie können ein Attribut sortierbar machen, wenn Sie das Attribut erstellen, oder Sie können es später ändern.

  2. Um eine Abfrageantwort in der Konsole zu sortieren, wählen Sie im Menü Sortieren das Attribut aus, um die Antwort zu sortieren. In der Liste werden nur Attribute angezeigt, die während der Datenquellenkonfiguration als sortierbar markiert wurden.
