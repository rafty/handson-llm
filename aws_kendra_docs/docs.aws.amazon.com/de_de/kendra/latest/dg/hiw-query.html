Abfragen

Um Antworten zu erhalten, fragen Benutzer einen Index ab. Benutzer können in ihren Abfragen natürliche Sprache verwenden. Die Antwort enthält Informationen wie den Titel, einen Textauszug und die Position der Dokumente im Index, die die beste Antwort bieten.

Amazon Kendraverwendet alle Informationen, die Sie zu Ihren Dokumenten angeben, nicht nur den Inhalt der Dokumente, um festzustellen, ob ein Dokument für die Anfrage relevant ist. Wenn Ihr Index beispielsweise Informationen darüber enthält, wann Dokumente zuletzt aktualisiert wurden, können Sie festlegen, dass Dokumenten,Amazon Kendra die kürzlich aktualisiert wurden, eine höhere Relevanz zugewiesen werden soll.

Eine Abfrage kann auch Kriterien für das Filtern der Antwort enthalten, sodass nur DokumenteAmazon Kendra zurückgegeben werden, die die Filterkriterien erfüllen. Wenn Sie beispielsweise ein Indexfeld mit dem Namen Abteilung erstellt haben, können Sie die Antwort filtern, sodass nur Dokumente zurückgegeben werden, bei denen das Abteilungsfeld auf legal gesetzt ist. Weitere Informationen finden Sie unter Filterung und Facettensuche.

Sie können die Ergebnisse einer Abfrage beeinflussen, indem Sie die Relevanz einzelner Felder im Index optimieren. Durch Tuning wird die Bedeutung eines Feldes für die Ergebnisse geändert. Wenn Sie beispielsweise die Wichtigkeit von Dokumenten mit der Kategorie Neu erhöhen, ist es wahrscheinlicher, dass Dokumente mit dieser Kategorie in der Antwort enthalten sind. Weitere Informationen finden Sie unter Optimieren von Suchresultaten.

Weitere Informationen zur Verwendung von Abfragen finden Sie unterIndizes durchsuchen.
