Erste Schritte (AWS CLI)

Das folgende Verfahren zeigt, wie Sie einenAmazon Kendra Index mit dem erstellenAWS CLI. Die Prozedur erstellt eine Datenquelle und einen Index und führt eine Abfrage für den Index aus.

Um einenAmazon Kendra Index zu erstellen (CLI)

  1. Mach dasVoraussetzungen.

  2. Verwenden Sie den folgenden Befehl, um einen Index zu erstellen.

    aws kendra create-index \
 --name cli-getting-started-index \
 --description "Index for CLI getting started guide." \
 --role-arn arn:aws:iam::account id:role/KendraRoleForGettingStartedIndex

  3. Warten SieAmazon Kendra, bis der Index erstellt wurde. Überprüfen Sie den Fortschritt mithilfe des Befehls. Sobald das Statusfeld lautetACTIVE, fahren Sie mit dem nächsten Schritt fort.

    aws kendra describe-index \
 --id index id

  4. Verwenden Sie an der Eingabeaufforderung den folgenden Befehl, um eine Datenquelle zu erstellen.

    aws kendra create-data-source \
 --index-id index id \
 --name data source name \
 --role-arn arn:aws:iam::account id:role/KendraRoleForGettingStartedDataSource \
 --type S3 \
 --configuration '{"S3Configuration":{"BucketName":"S3 bucket name"}}'

    Wenn Sie mithilfe eines Vorlagenschemas eine Verbindung zu Ihrer Datenquelle herstellen, konfigurieren Sie das Vorlagenschema.

    aws kendra create-data-source \
 --index-id index id \
 --name data source name \
 --role-arn arn:aws:iam::account id:role/KendraRoleForGettingStartedDataSource \
 --type TEMPLATE \
 --configuration '{"TemplateConfiguration":{"Template":{JSON schema}}}'

  5. Das Erstellen der Datenquelle wirdAmazon Kendra eine Weile dauern. Verwenden Sie den folgenden Befehl, um den Fortschritt zu überprüfen. Sobald der Status lautetACTIVE, fahren Sie mit dem nächsten Schritt fort.

    aws kendra describe-data-source \
 --id data source ID \
 --index-id index ID

  6. Verwenden Sie den folgenden Befehl, um die Datenquelle zu synchronisieren.

    aws kendra start-data-source-sync-job \
 --id data source ID \
 --index-id index ID

  7. Amazon Kendraindexiert Ihre Datenquelle. Der Zeitaufwand hängt von der Anzahl der Dokumente ab. Sie können den Status des Synchronisierungsjobs mithilfe des Befehls überprüfen. Sobald der Status lautetACTIVE, fahren Sie mit dem nächsten Schritt fort.

    aws kendra describe-data-source \
 --id data source ID \
 --index-id index ID

  8. Verwenden Sie den folgenden Befehl, um eine Abfrage zu stellen.

    aws kendra query \
 --index-id index ID \
 --query-text "search term"

    Die Ergebnisse der Suche werden im JSON-Format angezeigt.
