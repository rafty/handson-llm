Abfrageantworten und Antworttypen

Amazon Kendraunterstützt verschiedene Abfrageantworten und Antworttypen.

Antworten abfragen

Ein Aufruf der Abfrage-API gibt Informationen über die Ergebnisse einer Suche zurück. Die Ergebnisse befinden sich in einer Reihe vonQueryResultItem Objekten (ResultItems). JedesQueryResultItem enthält eine Zusammenfassung des Ergebnisses. Mit dem Abfrageergebnis verknüpfte Dokumentattribute sind enthalten.

Zusammenfassende Informationen

Die zusammenfassenden Informationen variieren je nach Art des Ergebnisses. In jedem Fall enthält es Dokumenttext, der mit dem Suchbegriff übereinstimmt. Es enthält auch Markierungsinformationen, mit denen Sie den Suchtext in der Ausgabe Ihrer Anwendung hervorheben können. Zum Beispiel, wenn der Suchbegriff lautet, wie hoch ist die Space Needle? , die zusammenfassenden Informationen enthalten die Textposition für die Wörter Höhe und Leertaste. Weitere Informationen zu Antworttypen finden Sie unterAbfrageantworten und Antworttypen.

Attribute des Dokuments

Jedes Ergebnis enthält Dokumentattribute für das Dokument, das einer Abfrage entspricht. Einige der Attribute sind vordefiniert, wieDocumentIdDocumentTitle, undDocumentUri. Andere sind benutzerdefinierte Attribute, die Sie definieren. Sie können Dokumentattribute verwenden, um die Antwort aus derQuery API zu filtern. Beispielsweise möchten Sie möglicherweise nur die Dokumente eines bestimmten Autors oder einer bestimmten Version eines Dokuments. Weitere Informationen finden Sie unter Filterung und Facettensuche. Sie geben Dokumentattribute an, wenn Sie Dokumente zu einem Index hinzufügen. Weitere Informationen finden Sie unter Benutzerdefinierte Felder oder Attribute.

Arten von Antworttypen

Amazon Kendragibt drei Arten von Abfrageantworten zurück.

  • Antwort (einschließlich Tabellenantwort)

  • Dokument

  • Frage und Antwort

Der Typ der Antwort wird imType Antwortfeld des QueryResultItemObjekts zurückgegeben.

Antwort

Amazon Kendrahat eine oder mehrere Fragenantworten in der Antwort erkannt. Ein Faktoid ist die Antwort auf eine Frage wer, was, wann oder wo, wie zum Beispiel Wo ist das nächstgelegene Servicecenter für mich? Amazon Kendragibt Text im Index zurück, der der Abfrage am besten entspricht. Der Text befindet sich imAnswerText Feld und enthält Markierungsinformationen für den Suchbegriff im Antworttext. AnswerTextenthält den vollständigen Dokumentauszug mit hervorgehobenem Text, währendDocumentExcerpt der gekürzte (290 Zeichen) Dokumentauszug mit hervorgehobenem Text eingeschlossen ist.

Amazon Kendragibt nur eine Antwort pro Dokument zurück, und das ist die Antwort mit der höchsten Sicherheit. Um mehrere Antworten aus einem Dokument zurückzugeben, müssen Sie das Dokument in mehrere Dokumente aufteilen.

Dokument

Amazon Kendragibt Dokumente mit Rangfolge für diejenigen zurück, die dem Suchbegriff entsprechen. Das Ranking basiert auf dem Vertrauen,Amazon Kendra das in die Richtigkeit des Suchergebnisses gesetzt wird. Informationen über das passende Dokument werden in der zurückgegeben QueryResultItem. Es enthält den Titel des Dokuments. Der Auszug enthält hervorgehobene Informationen für den Suchtext und den Abschnitt mit dem passenden Text im Dokument. Die URI für passende Dokumente befindet sich imSourceURI Dokumentattribut. Das folgende JSON-Beispiel zeigt die Dokumentzusammenfassung für ein passendes Dokument.

Frage und Antwort

Eine Frage und eine Antwort werden zurückgegeben, wenn eine Frage mit einer der häufig gestellten Fragen in Ihrem IndexAmazon Kendra übereinstimmt. Die Antwort enthält die passende Frage und Antwort im QueryResultItemFeld. Es enthält auch Highlight-Informationen für Abfragebegriffe, die in der Abfragezeichenfolge erkannt wurden. Das folgende JSON zeigt eine Frage- und Antwortantwort. Beachten Sie, dass die Antwort den Fragetext enthält.

{
    'AnswerText': {
        'TextWithHighlights': [
            
        ],
        'Text': '605feet'
    },
    'DocumentExcerpt': {
        'Highlights': [
            {
                'BeginOffset': 0,
                'EndOffset': 8,
