Datenquellen

Eine Datenquelle ist ein Datenspeicher oder Speicherort, der eineAmazon Kendra Verbindung zu Ihren Dokumenten oder Inhalten herstellt und diese indexiert. Sie können beispielsweise so konfigurieren,Amazon Kendra dass eine Verbindung zu Microsoft hergestellt wird SharePoint , um Ihre in dieser Quelle gespeicherten Dokumente zu crawlen und zu indexieren. Sie können Webseiten auch indexieren, indem Sie die URLs fürAmazon Kendra das Crawlen angeben. Sie können eine Datenquelle automatisch mit einemAmazon Kendra Index synchronisieren, sodass hinzugefügte, aktualisierte oder gelöschte Dokumente in der Datenquelle auch im Index hinzugefügt, aktualisiert oder gelöscht werden.

Anmerkung

Um einen Index zu erstellen, benötigen Sie keine Datenquelle. Sie können Dokumente direkt zu einem Index hinzufügen. Weitere Informationen finden Sie unter Direktes Hinzufügen von Dokumenten zu einem Index.
