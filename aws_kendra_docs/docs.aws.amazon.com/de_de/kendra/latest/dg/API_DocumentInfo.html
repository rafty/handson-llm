DocumentInfo

Identifiziert ein Dokument, für das Statusinformationen abgerufen werden sollen

Inhalt

Attributes

Attribute, die eine bestimmte Version eines zu prüfenden Dokuments identifizieren.

Die einzige gültige Attribute sind:

  • Version

  • Datenquellen-ID

  • jobExecutionId

Die Attribute erfolgen nach diesen Regeln:

  • dataSourceIdundjobExecutionId müssen zusammen verwendet werden.

  • versionwird ignoriert, wenndataSourceId undjobExecutionId nicht angegeben werden.

  • WenndataSourceId und angegebenjobExecutionId werden, aber nicht,version ist die Version standardmäßig auf „0" gesetzt.

Typ: Array von DocumentAttribute-Objekten

Required: No

DocumentId

Die Kennung des Dokuments.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 2048 Zeichen.

Erforderlich: Ja
