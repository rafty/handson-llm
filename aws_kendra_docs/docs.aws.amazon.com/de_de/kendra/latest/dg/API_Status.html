Status

Bietet Informationen über den Status der zur Indexierung eingereichten Dokumente.

Inhalt

DocumentId

Die ID des Dokuments.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 2048 Zeichen.

Required: No

DocumentStatus

Der aktuelle Status eines Dokuments.

Wenn das Dokument zum Löschen eingereicht wurde, lautet der StatusNOT_FOUND nach dem Löschen des Dokuments.

Typ: Zeichenfolge

Zulässige Werte: NOT_FOUND | PROCESSING | INDEXED | UPDATED | FAILED | UPDATE_FAILED

Required: No

FailureCode

Bietet die Quelle des Fehlers.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 2048 Zeichen.

Required: No

FailureReason

Enthält detaillierte Informationen darüber, warum das Dokument nicht indexiert werden konnte. Verwenden Sie diese Informationen, um den Fehler zu korrigieren, bevor Sie das Dokument erneut zur Indexierung einreichen.

Typ: Zeichenfolge

Längenbeschränkungen: Minimale Länge von 1. Maximale Länge beträgt 2048 Zeichen.

Required: No
