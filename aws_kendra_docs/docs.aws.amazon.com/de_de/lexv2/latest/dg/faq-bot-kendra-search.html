Beispiel: Einen FAQ-Bot für einen Amazon Kendra Kendra-Index erstellen

In diesem Beispiel wird ein Amazon Lex V2-Bot erstellt, der einen Amazon Kendra Kendra-Index verwendet, um Antworten auf die Fragen der Benutzer zu geben. Der Bot zu häufig gestellten Fragen verwaltet den Dialog für den Benutzer. Er verwendet die Absicht AMAZON.KendraSearchIntent, um den Index abzufragen und die Antwort für den Benutzer bereitzustellen. Hier finden Sie eine Zusammenfassung, wie Sie Ihren FAQ-Bot mithilfe eines Amazon Kendra Kendra-Index erstellen:

  1. Erstellen Sie einen Bot, mit dem Ihre Kunden interagieren werden, um Antworten von diesem Bot zu erhalten.

  2. Erstellen Sie eine benutzerdefinierte Absicht. Da esAMAZON.FallbackIntent sich bei denAMAZON.KendraSearchIntent und um Backup-Intents handelt, benötigt Ihr Bot mindestens eine weitere Absicht, die mindestens eine Äußerung enthalten muss. Diese Absicht ermöglicht die Entwicklung des Bots, wird anderweitig jedoch nicht verwendet. Ihr FAQ-Bot wird daher mindestens drei Absichten enthalten, wie in der Abbildung unten dargestellt:

  3. Fügen Sie Ihrem Bot dieAMAZON.KendraSearchIntent Absicht hinzu und konfigurieren Sie ihn so, dass er mit Ihrem Amazon Kendra Kendra-Index funktioniert.

  4. Testen Sie den Bot, indem Sie eine Abfrage stellen und überprüfen, ob es sich bei den Ergebnissen Ihres Amazon Kendra Kendra-Index um Dokumente handelt, die die Anfrage beantworten.

Voraussetzungen

Damit Sie dieses Beispiel verwenden können, müssen Sie einen Amazon-Kendra-Index erstellen. Weitere Informationen finden Sie unter Erste Schritte mit der Amazon Kendra-Konsole im Amazon Kendra Developer Guide. Wählen Sie für dieses Beispiel den Beispieldatensatz (Beispiel-AWS-Dokumentation) als Ihre Datenquelle aus.

Um einen FAQ-Bot zu erstellen:

  1. Melden Sie sich sich Sie sich bei derAWS Management Console Sie sich Sie Sie Sie Sie Sie sich Amazon Lex Sie sich Sie sich Sie sich Sie sich https://console.aws.amazon.com/lex/

  2. Wählen Sie im Navigationsbereich Bots.

  3. Wählen Sie Create Bot.

    1. Wählen Sie als Erstellungsmethode die Option Leeren Bot erstellen aus.

    2. Geben Sie dem Bot im Abschnitt Bot-Konfiguration einen Namen, der seinen Zweck angibt, z. B.KendraTestBot, und eine optionale Beschreibung. Der Name muss in Ihrem Konto eindeutig sein.

    3. Wählen Sie im Abschnitt IAM-Berechtigungen die Option Rolle mit grundlegenden Amazon Lex-Berechtigungen erstellen aus. Dadurch wird eine AWS Identity and Access Management(IAM-) Rolle mit den Berechtigungen erstellt, die Amazon Lex V2 benötigt, um Ihren Bot auszuführen.

    4. Wählen Sie im Abschnitt Gesetz zum Schutz der Privatsphäre von Kindern im Internet (COPPA) die Option Nein aus.

    5. Belassen Sie in den Abschnitten Sitzungstimeout im Leerlauf und Erweiterte Einstellungen die Standardeinstellungen und wählen Sie Weiter.

    6. Jetzt befinden Sie sich im Abschnitt Sprache zum Bot hinzufügen. Wählen Sie im Menü unter Sprachinteraktion die Option Keine aus. Dies ist nur eine textbasierte Anwendung. Überbliche Einstellungen für die übrigen Felder

    7. Wählen Sie Done (Erledigt) aus. Amazon Lex V2 erstellt Ihren Bot und eine Standardabsicht, die aufgerufen wird NewIntent, und führt Sie auf die Seite, auf der Sie diese Absicht konfigurieren können

Um erfolgreich einen Bot zu erstellen, müssen Sie mindestens eine Absicht erstellen, die von demAMAZON.FallbackIntent und dem getrennt istAMAZON.KendraSearchIntent. Diese Absicht ist erforderlich, um Ihren Amazon Lex V2-Bot zu erstellen, wird jedoch nicht für die Antwort auf häufig gestellte Fragen verwendet. Diese Absicht muss mindestens eine beispielhafte Äußerung enthalten, und die Äußerung darf sich auf keine der Fragen beziehen, die Ihr Kunde stellt.

Gehen Sie wie folgt vor, um die erforderliche Absicht zu erstellen:

  1. Geben Sie im Abschnitt Absichtsdetails einen Namen für die Absicht, z.RequiredIntent B.

  2. Geben Sie im Abschnitt Beispieläußerungen eine Äußerung in das Feld neben Äußerung hinzufügen ein, z.Required utterance B. Wählen Sie dann Äußerung hinzufügen.

  3. Wählen Sie Save intent (Absicht speichern).

Erstellen Sie die Absicht, einen Amazon Kendra Kendra-Index und die Antwortnachricht, die zurückgegeben werden soll, zu durchsuchen.

So erstellen Sie Sie Sie Sie sich einen AMAZON- KendraSearchIntent Absichts- und Antwortnachricht:

  1. Wählen Sie im Navigationsbereich Zurück zur Absichtsliste aus, um zur Seite mit den Absichten für Ihren Bot zurückzukehren. Wählen Sie Absicht hinzufügen und wählen Sie im Dropdown-Menü die Option Integrierte Absicht verwenden aus.

  2. Wählen Sie in dem sich öffnenden Feld das Menü unter Integrierte Absicht aus. Geben Sie esAMAZON.KendraSearchIntent in die Suchleiste ein und wählen Sie es dann aus der Liste aus.

  3. Geben Sie der Absicht einen Namen, z.KendraSearchIntent B.

  4. Wählen Sie Sie sich im Amazon-Kendra-Menü den Index, den Sie durchsuchen möchten, möchten Sie suchen möchten, können Sie ihn durchsuchen möchten. Der Index, den Sie im Abschnitt Voraussetzungen erstellt haben, sollte verfügbar sein.

  5. Wählen Sie Hinzufügen.

  6. Scrollen Sie im Intent-Editor nach unten zum Abschnitt Versand, wählen Sie den Rechtspfeil aus, um den Abschnitt zu erweitern, und fügen Sie die folgende Meldung in das Feld unter Bei erfolgreicher Erfüllung ein: 

    I found a link to a document that could help you: ((x-amz-lex:kendra-search-response-document-link-1)).

    Weitere Informationen zur Amazon Kendra Search Response finden Sie unter Verwenden der Suchantwort.

  7. Wählen Sie Save intent (Absicht speichern) und anschließend Build (Erstellen), um den Bot zu erstellen. Wenn der Bot bereit ist, wird das Banner oben auf dem Bildschirm grün und es wird eine Erfolgsmeldung angezeigt.

Verwenden Sie schließlich das Konsolentestfenster, um Antworten von Ihrem Bot zu testen.

Um deinen FAQ-Bot zu testen:

  1. Nachdem der Bot erfolgreich erstellt wurde, wählen Sie Test.

  2. Geben SieWhat is Amazon Kendra? das Testfenster der Konsole ein. Stellen Sie sicher, dass der Bot mit einem Link reagiert.

  3. Weitere Informationen zur Konfiguration finden SieAMAZON.KendraSearchIntent unter AMAZON unter AMAZON. KendraSearchIntentund KendraConfiguration.
