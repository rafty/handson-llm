Mengunggah, mengunduh, dan bekerja dengan objek di Amazon S3

Untuk menyimpan data Anda di Amazon S3, Anda bekerja dengan sumber daya yang dikenal sebagai bucket dan objek. bucket adalah wadah untuk objek. objek adalah file dan metadata opsional apa pun yang menjelaskan file tersebut.

Untuk menyimpan objek di Amazon S3, Anda membuat bucket dan kemudian mengunggah objek ke bucket. Bila objek ada di dalam bucket, Anda bisa membukanya, mengunduhnya, dan menyalinnya. Bila Anda tidak lagi membutuhkan objek atau bucket, Anda bisa membersihkan sumber daya ini.

penting

Di konsol Amazon S3, saat Anda memilih Buka atau Unduh Sebagai untuk objek, operasi ini membuat URL yang telah ditandatangani sebelumnya. Selama lima menit, objek Anda akan dapat diakses oleh siapa saja yang memiliki akses ke URL yang telah ditandatangani sebelumnya. Untuk informasi lebih lanjut tentang URLs presigned, lihat Menggunakan URLs presigned.

Dengan Amazon S3, Anda hanya membayar untuk apa yang Anda gunakan. Untuk informasi selengkapnya tentang fitur dan harga Amazon S3, lihat Amazon S3. Jika Anda adalah pelanggan Amazon S3, Anda dapat memulai dengan Amazon S3 secara gratis. Untuk informasi selengkapnya, lihat AWS Tingkat Gratis.
