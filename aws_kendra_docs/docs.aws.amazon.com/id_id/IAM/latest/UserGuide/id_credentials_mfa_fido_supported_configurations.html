Konfigurasi yang mendukung untuk menggunakan kunci keamanan FIDO

Anda dapat menggunakan WebAuthn metode Multi-Factor Authentication (MFA) dengan IAM menggunakan konfigurasi yang didukung saat ini. Ini termasuk perangkat FIDO Certified yang didukung oleh IAM dan browser yang mendukung WebAuthn.

Perangkat FIDO didukung olehAWS

IAM saat ini mendukung perangkat keamanan Bersertifikat FIDO yang terhubung ke perangkat Anda melalui USB, Bluetooth, atau NFC. Saat ini kami tidak mendukung autentikator platform seperti TouchID, FaceID, atau Windows Hello.
