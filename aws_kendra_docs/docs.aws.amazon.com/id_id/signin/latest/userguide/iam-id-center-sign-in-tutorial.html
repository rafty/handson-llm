Masuk ke portal AWS akses

Seorang pengguna di IAM Identity Center adalah anggota dari. AWS Organizations Pengguna di IAM Identity Center dapat mengakses beberapa Akun AWS aplikasi bisnis dengan masuk ke portal AWS akses dengan URL login tertentu. Untuk informasi selengkapnya tentang URL masuk tertentu, lihat. AWSakses portal

Sebelum Anda masuk ke pengguna IAM Identity Center, kumpulkan informasi wajib berikut. Akun AWS

  • Nama pengguna perusahaan

  • Kata sandi perusahaan

  • URL masuk tertentu

catatan

Setelah Anda masuk, sesi portal AWS akses Anda berlaku selama 8 jam. Anda diminta untuk masuk lagi setelah 8 jam.

Untuk masuk ke portal AWS akses

  1. Di jendela browser, tempelkan URL masuk yang diberikan melalui email, misalnya. https://your_subdomain.awsapps.com/start Kemudian, tekan Enter.

  2. Masuk menggunakan kredensi perusahaan Anda (seperti nama pengguna dan kata sandi).

    catatan

    Jika administrator mengirimi Anda email kata sandi satu kali (OTP) dan ini adalah pertama kalinya Anda masuk, masukkan kata sandi tersebut. Setelah masuk, Anda harus membuat kata sandi baru untuk login di future.

  3. Jika Anda diminta untuk kode verifikasi, periksa email Anda untuk itu. Kemudian copy dan paste kode ke halaman login.

    catatan

    Kode verifikasi biasanya dikirim melalui email, tetapi metode pengirimannya mungkin berbeda. Jika belum menerimanya di email Anda, tanyakan kepada administrator Anda untuk detail tentang kode verifikasi Anda.

  4. Jika MFA diaktifkan untuk pengguna Anda di IAM Identity Center, Anda kemudian mengautentikasi menggunakannya.

  5. Setelah otentikasi, Anda dapat mengakses aplikasi apa pun Akun AWS dan yang muncul di portal.

    1. Untuk masuk ke AWS Management Console pilih Akun AWS ubin dan kemudian pilih akun individual yang akan dikelola.

      Izin yang dimiliki pengguna Anda ditampilkan bersama dengan tautan untuk membuka konsol manajemen atau untuk mendapatkan kredensi baris perintah atau akses terprogram.

      Pilih tautan Management console.

    2. Untuk masuk ke aplikasi, pilih ikon aplikasi yang ingin Anda akses.

Masuk sebagai pengguna di IAM Identity Center memberi Anda kredensi untuk mengakses sumber daya selama durasi waktu yang ditentukan, yang disebut sesi. Secara default, pengguna dapat masuk ke Akun AWS selama satu jam. Nilai maksimum yang dapat diatur administrator untuk sesi adalah 12 jam. Setelah sesi berakhir, Anda dapat masuk kembali.
