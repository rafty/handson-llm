Masuk ke AWS Management Console

Ketika Anda masuk ke AWS Management Console dari URL AWS masuk utama (https://signin.aws.amazon.com), Anda harus memilih jenis pengguna Anda, baik pengguna Root atau pengguna IAM.

Pengguna root memiliki akses akun yang tidak dibatasi dan dikaitkan dengan orang yang membuat file. Akun AWS Pengguna root kemudian membuat jenis pengguna lain, seperti pengguna IAM danpengguna di AWS IAM Identity Center (penerus AWS Single Sign-On), dan menugaskan mereka akses kredenial.

IAM pengguna adalah identitas dalam akun Anda Akun AWS yang memiliki izin khusus tertentu. Ketika pengguna IAM masuk, mereka dapat menggunakan URL login yang menyertakan alias mereka Akun AWS atau alias, seperti https://account_alias_or_id.signin.aws.amazon.com/console/ alih-alih URL masuk utamaAWS. https://signin.aws.amazon.com/

Jika Anda tidak yakin jenis pengguna Anda, lihattipe pengguna.
