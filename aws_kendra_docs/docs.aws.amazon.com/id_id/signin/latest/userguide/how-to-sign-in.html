Cara masuk ke AWS

Cara Anda masuk AWS tergantung pada jenis AWS pengguna Anda. Ada berbagai jenis AWS pengguna. Anda bisa menjadi pengguna root akun, pengguna IAM, pengguna di IAM Identity Center, identitas federasi, atau menggunakan AWS Builder ID. Untuk informasi selengkapnya, lihat tipe pengguna.

Anda dapat mengakses AWS dengan masuk dengan salah satu metode berikut:

  • Masuk ke AWS Management Consolesebagai pengguna root atau pengguna IAM

  • Masuk ke portal AWS aksessebagai pengguna di IAM Identity Center

  • Masuk sebagai identitas federasi

  • Masuk melalui AWS Command Line Interfacedan metode terprogram lainnya seperti API dan SDK (Software Development Kit)

  • Masuk dengan AWS Builder ID

Jika Anda ingin membuat yang baruAkun AWS, lihat Bagian 1: Mengatur yang baru Akun AWS di Panduan AWS Pengaturan.
