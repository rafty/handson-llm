AWS Service quotas

Akun AWS Anda memiliki kuota default, yang sebelumnya disebut sebagai batas, untuk setiap layanan AWS. Kecuali dinyatakan lain, setiap kuota bersifat khusus per Wilayah. Anda dapat meminta peningkatan untuk beberapa kuota dan kuota lainnya tidak dapat ditingkatkan.

Service Quotas adalah layanan AWS yang membantu mengelola kuota Anda untuk banyak layanan AWS dari satu lokasi. Selain mencari nilai kuota, Anda juga dapat meminta peningkatan kuota dari konsol Service Quotas.

AWS Supportmungkin menyetujui, menolak, atau menyetujui sebagian permintaan Anda.

Untuk melihat service quotas

Anda dapat melihat service quotas menggunakan opsi berikut:

  • Buka halaman Titik akhir dan kuota layanan dalam dokumentasi, cari nama layanan, dan klik tautan untuk membuka halaman layanan tersebut. Untuk melihat service quotas untuk semua layanan AWS dalam dokumentasi tanpa beralih halaman, lihat informasi di halaman Service Endpoints and Quotas di PDF sebagai gantinya.

  • Buka Konsol Service Quotas. Di panel navigasi, pilih AWS services (Layanan AWS) dan pilih layanan.

  • Gunakan list-service-quotasdan list-aws-default-service-kuotaAWS CLI perintah.

Untuk meminta peningkatan kuota

Anda dapat meminta kenaikan kuota menggunakan Service Quotas, menggunakan salah satu opsi berikut. Peningkatan tidak diberikan segera. Mungkin perlu beberapa hari agar peningkatan Anda berlaku.

  • Buka Konsol Service Quotas. Di panel navigasi, pilih Layanan AWS. Pilih layanan, pilih kuota, dan ikuti petunjuk untuk meminta peningkatan kuota. Untuk informasi selengkapnya, lihat Meminta Peningkatan Kuota dalam Panduan Pengguna Service Quotas.

  • Gunakan perintah request-service-quota-increase AWS CLI.

Jika layanan belum tersedia dalam Service Quotas, buat kasus kenaikan batas layanan menggunakanAWS Support Center Console. Jika layanan tersedia dalam Service Quotas, kami sangat menyarankan agar Anda menggunakan Service Quotas daripada membuat kasus dukungan.
