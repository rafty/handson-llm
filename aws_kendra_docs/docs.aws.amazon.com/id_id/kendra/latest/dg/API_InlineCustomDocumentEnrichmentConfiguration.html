InlineCustomDocumentEnrichmentConfiguration

Menyediakan informasi konfigurasi untuk menerapkan logika dasar untuk mengubah metadata dokumen dan konten saat menelan dokumen ke Amazon Kendra. Untuk menerapkan logika lanjutan, untuk melampaui apa yang dapat Anda lakukan dengan logika dasar, lihat HookConfiguration.

Untuk informasi selengkapnya, lihat Menyesuaikan metadata dokumen selama proses konsumsi.

Daftar Isi

Condition

Konfigurasi kondisi yang digunakan untuk atribut dokumen target atau bidang metadata saat memasukkan dokumen ke Amazon Kendra.

Tipe: Objek DocumentAttributeCondition

Wajib: Tidak

DocumentContentDeletion

TRUEuntuk menghapus konten jika kondisi yang digunakan untuk atribut target terpenuhi.

Tipe: Boolean

Wajib: Tidak

Target

Konfigurasi atribut dokumen target atau bidang metadata saat memasukkan dokumen ke Amazon Kendra. Anda juga bisa memasukkan nilai.

Tipe: Objek DocumentAttributeTarget

Wajib: Tidak
