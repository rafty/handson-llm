Amazon S3skema Template

Anda menyertakan JSON yang berisi skema sumber data sebagai bagian dari konfigurasi template. Anda memberikan nama bucket S3 sebagai bagian dari konfigurasi koneksi atau detail endpoint repositori. Juga tentukan jenis sumber data sebagaiS3, dan konfigurasi lain yang diperlukan. Anda kemudian menentukanTEMPLATE sebagaiType ketika Anda menelepon CreateDataSource.
