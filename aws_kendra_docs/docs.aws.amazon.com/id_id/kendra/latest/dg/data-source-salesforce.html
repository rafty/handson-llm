Salesforce

Salesforce adalah alat manajemen hubungan pelanggan (CRM) untuk mengelola tim dukungan, penjualan, dan pemasaran. Anda dapat menggunakanAmazon Kendra untuk mengindeks objek standar Salesforce Anda dan bahkan objek kustom.

Anda dapat terhubungAmazon Kendra ke sumber data Salesforce menggunakan Amazon Kendrakonsol, TemplateConfigurationAPI, atau SalesforceConfigurationAPI.

Amazon Kendramemiliki dua versi konektor Salesforce. Fitur yang didukung dari setiap versi meliputi:

Konektor Salesforce V1.0 SalesforceConfiguration/API

  • Pemetaan lapangan

  • Penyaringan konteks pengguna

  • Filter inklusi

Konektor Salesforce TemplateConfigurationV2.0/API

  • Pemetaan lapangan

  • Penyaringan konteks pengguna

  • Filter inklusi

  • Dukungan lampiran entitas

  • Identitas merangkak

  • Dukungan VPC

  • Sinkronkan semua dokumen/Sinkronkan hanya dokumen baru, dimodifikasi, atau dihapus

catatan

Support untuk konektor Salesforce V1.0 SalesforceConfiguration /API dijadwalkan berakhir pada Juni 2023. Kami merekomendasikan menggunakan konektor Salesforce V2.0/ TemplateConfiguration API.

Untuk mengatasi masalah konektor sumber dataAmazon Kendra Salesforce Anda, lihatMengatasi masalah sumber data.
