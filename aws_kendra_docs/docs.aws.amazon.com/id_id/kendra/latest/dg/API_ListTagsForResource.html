ListTagsForResource

Mendapatkan daftar tanda yang terkait dengan sumber daya tertentu. Indeks, Pertanyaan yang Sering Diajukan, dan sumber data dapat memiliki tanda yang terkait dengannya.

Sintaksis Permintaan

{
   "ResourceARN": "string"
}

Parameter Permintaan

Untuk informasi tentang parameter yang umum untuk semua tindakan, lihat Parameter Umum.

Permintaan menerima data berikut dalam format JSON.

ResourceARN

Amazon Resource Name (ARN) dari indeks, Pertanyaan yang Sering Diajukan, atau sumber data untuk mendapatkan daftar tanda.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 1011.

Wajib: Ya

Sintaksis Respons

{
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

Elemen Respons

Jika tindakan berhasil, layanan mengirimkan kembali respons HTTP 200.

Layanan mengembalikan data berikut dalam format JSON.

Tags

Daftar tanda yang terkait dengan indeks, Pertanyaan yang Sering Diajukan, atau sumber data.

Tipe: Array objek Tag

Anggota Array: Jumlah minimum 0 item. Jumlah maksimum 200 item.

Kesalahan

Untuk informasi tentang kesalahan yang umum untuk semua tindakan, lihat Kesalahan Umum.

AccessDeniedException

Anda tidak memiliki akses yang memadai untuk melakukan tindakan ini. Pastikan Anda memiliki kebijakan izin dan akun pengguna yang diperlukan dan coba lagi.

Kode Status HTTP: 400

InternalServerException

Masalah terjadi dengan server internal yang digunakan untuk layanan Amazon Kendra Anda. Tunggu beberapa menit dan coba lagi, atau hubungi Support untuk bantuan.

Kode Status HTTP: 500

ResourceUnavailableException

Sumber daya yang ingin Anda gunakan tidak tersedia. Silakan periksa Anda telah menyediakan sumber daya yang benar dan coba lagi.

Kode Status HTTP: 400

ThrottlingException

Permintaan ditolak karena throttling permintaan. Kurangi jumlah permintaan dan coba lagi.

Kode Status HTTP: 400

ValidationException

Input gagal untuk memenuhi batasan yang ditentukan oleh layanan Amazon Kendra. Berikan input yang benar dan coba lagi.

Kode Status HTTP: 400
