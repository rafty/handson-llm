CreateDataSource

Menciptakan sebuah konektor sumber data yang ingin Anda gunakan dengan indeks Amazon Kendra.

Anda menentukan nama, jenis konektor sumber data, dan deskripsi untuk sumber data Anda. Anda juga menentukan informasi konfigurasi untuk konektor sumber data.

CreateDataSource adalah operasi tersinkron. Operasi ini mengembalikan 200 jika sumber data berhasil dibuat. Jika tidak, pengecualian akan ditampilkan.

Untuk contoh membuat indeks dan sumber data menggunakan Python SDK, lihat Memulai dengan Python SDK. Untuk contoh pembuatan indeks dan sumber data menggunakan Java SDK, lihat Memulai dengan Java SDK.

Parameter Permintaan

Untuk informasi tentang parameter yang umum untuk semua tindakan, lihat Parameter Umum.

Permintaan menerima data berikut dalam format JSON.

ClientToken

Token yang Anda berikan untuk mengidentifikasi permintaan untuk membuat sebuah konektor sumber data. Beberapa panggilan keCreateDataSource API dengan token klien yang sama akan membuat satu konektor sumber data saja.

Jenis: String

Panjang Batasan: Panjang minimum 1. Panjang maksimum 100.

Wajib: Tidak

Configuration

Informasi konfigurasi untuk terhubung ke repositori sumber data Anda.

Anda tidak dapat menentukan parameter Configuration ketika parameter Type diatur ke CUSTOM. Jika melakukannya, Anda akan mendapatkan pengecualian ValidationException.

Parameter Configuration diperlukan untuk semua sumber data lainnya.

Tipe: Objek DataSourceConfiguration

Wajib: Tidak

CustomDocumentEnrichmentConfiguration

Informasi konfigurasi untuk mengubah metadata dokumen dan konten selama proses konsumsi dokumen.

Untuk informasi selengkapnya tentang cara membuat, memodifikasi, dan menghapus metadata dokumen, atau membuat perubahan konten lainnya saat Anda memasukkan dokumen ke Amazon Kendra, lihat Menyesuaikan metadata dokumen selama proses konsumsi.

Tipe: Objek CustomDocumentEnrichmentConfiguration

Wajib: Tidak

Description

Deskripsi untuk konektor sumber data.

Jenis: String

Batasan Panjang: Panjang minimum sebesar 0. Panjang maksimum sebesar 1000.

Pola: ^\P{C}*$

Wajib: Tidak

IndexId

Pengenal indeks yang ingin Anda gunakan dengan konektor sumber data.

Jenis: String

Batas Panjang: Panjang tetap 36.

Pola: [a-zA-Z0-9][a-zA-Z0-9-]*

Diperlukan: Ya

LanguageCode

Kode untuk bahasa. Ini memungkinkan Anda untuk mendukung bahasa untuk semua dokumen saat membuat konektor sumber data. Bahasa Inggris didukung secara default. Untuk informasi selengkapnya tentang bahasa yang didukung, termasuk kodenya, lihat Menambahkan dokumen dalam bahasa selain bahasa Inggris.

Jenis: String

Panjang Batasan: Panjang minimum 2. Panjang maksimum 10.

Pola: [a-zA-Z-]*

Wajib: Tidak

Name

Nama untuk konektor sumber data.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 1000.

Pola: [a-zA-Z0-9][a-zA-Z0-9_-]*

Diperlukan: Ya

RoleArn

Amazon Resource Name (ARN) dari peran IAM dengan izin untuk mengakses sumber data dan sumber daya yang diperlukan. Untuk informasi selengkapnya, lihat IAM role akses untuk Amazon Kendra. .

Anda tidak dapat menentukan parameter RoleArn ketika parameter Type diatur ke CUSTOM. Jika melakukannya, Anda akan mendapatkan pengecualian ValidationException.

Parameter RoleArn diperlukan untuk semua sumber data lainnya.

Jenis: String

Batasan Panjang: Panjang minimum 0. Panjang maksimum 1284.

Pola: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Wajib: Tidak

Schedule

Mengatur frekuensi untuk Amazon Kendra untuk memeriksa dokumen dalam repositori sumber data Anda dan memperbarui indeks. Jika Anda tidak menetapkan jadwal Amazon Kendra tidak akan memperbarui indeks secara berkala. Anda dapat memanggilStartDataSourceSyncJob API untuk memperbarui indeks.

Tentukan string string string string string string string string string string string string string string string string string string string string string string string string string string string string yang kosong untuk menunjukkan bahwa indeks diperbarui sesuai permintaan.cron-

Anda tidak dapat menentukan parameter Schedule ketika parameter Type diatur ke CUSTOM. Jika melakukannya, Anda akan mendapatkan pengecualian ValidationException.

Tipe: String

Wajib: Tidak

Tags

Daftar pasangan nilai-kunci yang mengidentifikasi atau mengkategorikan konektor sumber data. Anda juga dapat menggunakan tanda untuk membantu mengontrol akses ke konektor sumber data. Kunci dan nilai tanda dapat terdiri dari huruf Unicode, angka, white space, dan salah satu simbol berikut: _ . : / = + - @.

Tipe: Array objek Tag

Anggota Array: Jumlah minimum 0 item. Jumlah maksimum 200 item.

Wajib: Tidak

Type

Jenis repositori sumber data. Sebagai contoh, SHAREPOINT.

Jenis: String

Nilai Valid: S3 | SHAREPOINT | DATABASE | SALESFORCE | ONEDRIVE | SERVICENOW | CUSTOM | CONFLUENCE | GOOGLEDRIVE | WEBCRAWLER | WORKDOCS | FSX | SLACK | BOX | QUIP | JIRA | GITHUB | ALFRESCO | TEMPLATE

Wajib: Ya

VpcConfiguration

Informasi konfigurasi untuk Amazon Virtual Private Cloud untuk terhubung ke sumber data Anda. Untuk informasi selengkapnya, lihat Mengonfigurasi VPC.

Tipe: Objek DataSourceVpcConfiguration

Wajib: Tidak

Sintaksis Respons

{
   "Id": "string"
}

Elemen Respons

Jika tindakan berhasil, layanan mengirimkan kembali respons HTTP 200.

Layanan mengembalikan data berikut dalam format JSON.

Id

Pengenal konektor sumber data.

Jenis: String

Panjang Batasan: Panjang minimum 1. Panjang maksimum 100.

Pola: [a-zA-Z0-9][a-zA-Z0-9_-]*

Kesalahan

Untuk informasi tentang kesalahan yang umum untuk semua tindakan, lihat Kesalahan Umum.

AccessDeniedException

Anda tidak memiliki akses yang memadai untuk melakukan tindakan ini. Pastikan Anda memiliki kebijakan izin dan akun pengguna yang diperlukan dan coba lagi.

Kode Status HTTP: 400

ConflictException

Konflik terjadi dengan permintaan. Harap perbaiki ketidakkonsistenan dengan sumber daya Anda dan coba lagi.

Kode Status HTTP: 400

InternalServerException

Masalah terjadi dengan server internal yang digunakan untuk layanan Amazon Kendra Anda. Tunggu beberapa menit dan coba lagi, atau hubungi Support untuk bantuan.

Kode Status HTTP: 500

ResourceAlreadyExistException

Sumber daya yang ingin Anda gunakan sudah ada. Silakan periksa Anda telah menyediakan sumber daya yang benar dan coba lagi.

Kode Status HTTP: 400

ResourceNotFoundException

Sumber daya yang ingin Anda gunakan tidak ada. Silakan periksa Anda telah menyediakan sumber daya yang benar dan coba lagi.

Kode Status HTTP: 400

ServiceQuotaExceededException

Anda telah melampaui batas yang ditetapkan untuk layanan Amazon Kendra Anda. Silakan lihat Kuota untuk informasi lebih lanjut, atau hubungi Support untuk menanyakan tentang peningkatan batas.

Kode Status HTTP: 400

ThrottlingException

Permintaan ditolak karena throttling permintaan. Kurangi jumlah permintaan dan coba lagi.

Kode Status HTTP: 400

ValidationException

Input gagal untuk memenuhi batasan yang ditetapkan oleh layanan Amazon Kendra. Berikan input yang benar dan coba lagi.

Kode Status HTTP: 400
