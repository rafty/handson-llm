DocumentAttributeValue

Nilai atribut dokumen. Anda hanya dapat memberikan satu nilai untuk atribut dokumen.

Daftar Isi

DateValue

Tanggal diekspresikan sebagai string ISO 8601.

Penting bahwa zona waktu disertakan dalam format tanggal ISO 8601. Sebagai contoh, 2012-03-25T12:30:10+ 01:00 adalah format tanggal-waktu ISO 8601 untuk 25 Maret 2012 pukul 12:30 PM (ditambah 10 detik) di Waktu Eropa Tengah.

Tipe: Timestamp

Wajib: Tidak

StringListValue
