ServiceNow

ServiceNow menyediakan sistem manajemen layanan berbasis cloud untuk membuat dan mengelola alur kerja tingkat organisasi, seperti layanan TI, sistem tiket, dan dukungan. Anda dapat menggunakanAmazon Kendra untuk mengindeks ServiceNow katalog Anda, artikel pengetahuan, insiden, dan lampiran mereka.

Anda dapat terhubungAmazon Kendra ke sumber ServiceNow data Anda menggunakan Amazon Kendrakonsol, TemplateConfigurationAPI, atau ServiceNowConfigurationAPI.

Amazon Kendramemiliki dua versi ServiceNow konektor. Fitur yang didukung dari setiap versi meliputi:

ServiceNow konektor V1.0 ServiceNowConfiguration/API

  • Pemetaan lapangan

  • ServiceNow versi contoh: London, Lainnya

  • Pola inklusi: Katalog layanan, artikel pengetahuan, lampiran

ServiceNow konektor TemplateConfigurationV2.0/API

  • Pemetaan lapangan

  • Pemfilteran konteks pengguna

  • Virtual private cloud (VPC)

  • Sinkronkan semua dokumen/Sinkronkan hanya dokumen baru, dimodifikasi, dan dihapus

  • ServiceNow versi contoh: Roma, Sandiego, Tokyo, Lainnya

  • Pola inklusi: Katalog layanan, artikel pengetahuan, insiden, lampiran

catatan

Support untuk ServiceNow konektor V1.0 ServiceNowConfiguration /API dijadwalkan berakhir pada Juni 2023. Kami merekomendasikan menggunakan ServiceNow konektor V2.0/ TemplateConfiguration API.

Untuk mengatasi masalah konektor sumberAmazon Kendra ServiceNow data Anda, lihatMengatasi masalah sumber data.
