Mengubah sumber identitas IAM Identity Center

Awas

Mengubah sumber identitas Anda di Pengaturan Pusat Identitas IAM dapat memengaruhi pelestarian informasi pengguna dan grup. Untuk melakukan ini dengan aman, disarankan Anda meninjau Pertimbangan untuk mengubah sumber identitas Anda. Saat Anda mengubah sumber identitas, ID sumber identitas baru akan dibuat. Periksa Anda menggunakan ID yang benar sebelum Anda mengatur mode keAWS_SSO dalam UserGroupResolutionConfiguration.

Untuk mengubah sumber identitas IAM Identity Center

  1. Buka IAM Identity Center> konsol.

  2. Pilih Pengaturan.

  3. Pada halaman Pengaturan, di bawah Sumber identitas, pilih Ubah.

  4. Pada halaman Ubah sumber identitas, pilih sumber identitas pilihan Anda, lalu pilih Berikutnya.
