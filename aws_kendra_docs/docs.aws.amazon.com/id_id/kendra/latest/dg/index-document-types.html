Jenis dokumen SSM

Indeks dapat berisi beberapa dokumen dan beberapa jenis dokumen. Indeks juga dapat mencakup teks terstruktur dan tidak terstruktur.

Teks terstruktur

  • Pertanyaan dan jawaban yang sering diajukan

Dokumen format pertanyaan dan jawaban yang sering diajukan digunakan untuk menjawab pertanyaan seperti Seberapa tinggi Space Needle? Anda dapat menentukan beberapa pertanyaan yang memberikan jawaban yang sama. Anda menentukan pertanyaan dan jawaban dalam file nilai yang dipisahkan oleh koma (CSV) yang disimpan diAmazon S3 bucket.

Jika Anda ingin menggunakan file JSON untuk serangkaian pertanyaan dan jawaban yang sering diajukan, file JSON diperlakukan sebagai teks biasa dan merupakan jenis teks yang tidak terstruktur.

Misalnya, Menambahkan pertanyaan dan jawaban langsung ke indeks.

Teks tidak terstruktur

Amazon Kendramendukung tipe teks tidak terstruktur berikut:

  • File HTML

  • Microsoft PowerPoint (PPT) presentasi

  • Dokumen MS WORD

  • Dokumen teks biasa

  • PDF

  • File Comma Separated Values (CSV)

  • Berkas Microsoft Excel (MS EXCEL)

  • File XMLM

  • File JSON

  • Penurunan harga Dokumentasi (MD) file

  • File Format Teks Kaya (RTF)

  • Extensible Stylesheet Bahasa Transformasi (XSLT) file

Anda dapat menambahkan dokumen tidak terstruktur ke indeks Anda dengan tiga cara:

  • Menggunakan BatchPutDocumentAPI.

  • Sebagai sumber data. Untuk informasi selengkapnya, lihat Menambahkan dokumen dari sumber data.

  • Sebagai data biner. Untuk informasi selengkapnya, lihat Menambahkan dokumen dengan API.
