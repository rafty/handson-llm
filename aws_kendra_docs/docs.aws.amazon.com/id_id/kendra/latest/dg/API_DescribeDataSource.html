Elemen Respons

Jika tindakan berhasil, layanan mengirimkan kembali respons HTTP 200.

Layanan mengembalikan data berikut dalam format JSON.

Configuration

Rincian konfigurasi untuk konektor sumber data data. Ini menunjukkan bagaimana sumber data dikonfigurasi. Opsi konfigurasi untuk sumber data bergantung pada penyedia sumber data bergantung pada penyedia sumber data.

Tipe: Objek DataSourceConfiguration

CreatedAt

Tanggal dan waktu Unix saat konektor sumber data dibuat.

Tipe: Timestamp

CustomDocumentEnrichmentConfiguration

Informasi konfigurasi untuk mengubah metadata dokumen dan konten selama proses konsumsi dokumen saat Anda menjelaskan sumber data.

Untuk informasi selengkapnya tentang cara membuat, memodifikasi, dan menghapus metadata dokumen, atau membuat perubahan konten lainnya saat Anda memasukkan dokumen ke Amazon Kendra, lihat Menyesuaikan metadata dokumen selama proses konsumsi.

Tipe: Objek CustomDocumentEnrichmentConfiguration

Description

Deskripsi untuk konektor sumber data data.

Jenis: String

Batasan Panjang: Panjang minimum sebesar 0. Panjang maksimum sebesar 1000.

Pola: ^\P{C}*$

ErrorMessage

Ketika nilai bidang Status adalah FAILED, bidang ErrorMessage berisi deskripsi kesalahan yang menyebabkan sumber data gagal.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 2048.

Pola: ^\P{C}*$

Id

Pengenal konektor sumber data sumber data.

Jenis: String

Panjang Batasan: Panjang minimum 1. Panjang maksimum 100.

Pola: [a-zA-Z0-9][a-zA-Z0-9_-]*

IndexId

Pengenal indeks yang digunakan dengan konektor sumber data.

Jenis: String

Batas Panjang: Panjang tetap 36.

Pola: [a-zA-Z0-9][a-zA-Z0-9-]*

LanguageCode

Kode untuk bahasa. Ini menunjukkan bahasa yang didukung untuk semua dokumen di sumber data. Bahasa Inggris didukung secara default. Untuk informasi selengkapnya tentang bahasa yang didukung, termasuk kodenya, lihat Menambahkan dokumen dalam bahasa selain bahasa Inggris.

Jenis: String

Batasan Panjang: Panjang minimum 2. Panjang maksimum 10.

Pola: [a-zA-Z-]*

Name

Nama untuk konektor sumber data data.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 1000.

Pola: [a-zA-Z0-9][a-zA-Z0-9_-]*

RoleArn

Amazon Resource Name (ARN) dari peran dengan izin untuk mengakses sumber data dan sumber data yang diperlukan.

Jenis: String

Batasan Panjang: Panjang minimum 0. Panjang maksimum 1284.

Pola: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Schedule

Jadwal Amazon Kendra untuk memperbarui indeks.

Jenis: String

Status

Status sumber data saat ini dari konektor sumber data saat ini. Sumber data siap digunakan saat statusnya ACTIVE. Jika statusnya adalah FAILED, bidang ErrorMessage berisi alasan mengapa sumber data gagal.

Jenis: String

Nilai Valid: CREATING | DELETING | FAILED | UPDATING | ACTIVE

Type

Jenis sumber data. Sebagai contoh, SHAREPOINT.

Jenis: String

Nilai Valid: S3 | SHAREPOINT | DATABASE | SALESFORCE | ONEDRIVE | SERVICENOW | CUSTOM | CONFLUENCE | GOOGLEDRIVE | WEBCRAWLER | WORKDOCS | FSX | SLACK | BOX | QUIP | JIRA | GITHUB | ALFRESCO | TEMPLATE

UpdatedAt

Tanggal dan waktu Unix saat konektor sumber data terakhir diperbarui.

Tipe: Timestamp

VpcConfiguration

Informasi konfigurasi untuk Amazon Virtual Private Cloud untuk terhubung ke sumber data Anda. Untuk informasi selengkapnya, lihat Mengonfigurasi VPC.

Tipe: Objek DataSourceVpcConfiguration

Kesalahan

Untuk informasi tentang kesalahan yang umum untuk semua tindakan, lihat Kesalahan Umum.

AccessDeniedException

Anda tidak memiliki akses yang memadai untuk melakukan tindakan ini. Pastikan Anda memiliki kebijakan izin dan akun pengguna yang diperlukan dan coba lagi.

Kode Status HTTP: 400

InternalServerException

Masalah terjadi dengan server internal yang digunakan untuk layanan Amazon Kendra Anda. Tunggu beberapa menit dan coba lagi, atau hubungi Support untuk mendapatkan bantuan.
