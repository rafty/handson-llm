DescribeIndex

Mendapat informasi tentang indeks Amazon Kendra ada.

Sintaksis Permintaan

{
   "Id": "string"
}

Parameter Permintaan

Untuk informasi tentang parameter yang umum untuk semua tindakan, lihat Parameter Umum.

Permintaan menerima data berikut dalam format JSON.

Id

Pengenal indeks tempat Anda ingin mendapatkan informasi.

Jenis: String

Batas Panjang: Panjang tetap 36.

Pola: [a-zA-Z0-9][a-zA-Z0-9-]*

Diperlukan: Ya

Elemen Respons

Jika tindakan berhasil, layanan mengirimkan kembali respons HTTP 200.

Layanan mengembalikan data berikut dalam format JSON.

CapacityUnits

Untuk indeks Edisi Korporasi, Anda dapat memilih menggunakan kapasitas tambahan untuk memenuhi kebutuhan aplikasi Anda. Ini berisi unit kapasitas yang digunakan untuk indeks. Kapasitas penyimpanan kueri atau dokumen nol menunjukkan bahwa indeks menggunakan kapasitas default. Untuk informasi lebih lanjut tentang kapasitas default untuk indeks dan menyesuaikan ini, lihat Menyesuaikan kapasitas.

Tipe: Objek CapacityUnitsConfiguration

CreatedAt

Tanggal dan waktu Unix saat indeks dibuat.

Tipe: Timestamp

Description

Deskripsi untuk indeks.

Jenis: String

Batasan Panjang: Panjang minimum sebesar 0. Panjang maksimum sebesar 1000.

Pola: ^\P{C}*$

DocumentMetadataConfigurations

Informasi konfigurasi untuk metadata dokumen atau bidang. Metadata dokumen adalah bidang atau atribut yang terkait dengan dokumen Anda. Misalnya, nama departemen perusahaan yang terkait dengan setiap dokumen.

Tipe: Array objek DocumentMetadataConfiguration

Anggota Array: Jumlah minimum 0 item. Jumlah maksimum 500 item.

Edition

Edisi Amazon Kendra yang digunakan untuk indeks. Anda menentukan edisi ketika membuat indeks.

Jenis: String

Nilai Valid: DEVELOPER_EDITION | ENTERPRISE_EDITION

ErrorMessage

Ketika nilaiStatus bidangFAILED,ErrorMessage bidang berisi pesan yang menjelaskan alasannya.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 2048.

Pola: ^\P{C}*$

Id

Pengenal indeks.

Jenis: String

Batas Panjang: Panjang tetap 36.

Pola: [a-zA-Z0-9][a-zA-Z0-9-]*

IndexStatistics

Memberikan informasi tentang jumlah pertanyaan dan jawaban Pertanyaan yang Sering Diajukan serta jumlah dokumen teks yang diindeks.

Tipe: Objek IndexStatistics

Name

Nama indeks.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 1000.

Pola: [a-zA-Z0-9][a-zA-Z0-9_-]*

RoleArn

Amazon Resource Name (ARN) dari IAM role yang memberikan izin pada Amazon Kendra untuk menulis ke Amazon CloudWatch Logs Anda.

Jenis: String

Batasan Panjang: Panjang minimum 0. Panjang maksimum 1284.

Pola: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

ServerSideEncryptionConfiguration

Pengenal kunci utamaAWS KMS pelanggan (CMK) yang digunakan untuk mengenkripsi data Anda. Amazon Kendra tidak mendukung CMK asimetris.

Tipe: Objek ServerSideEncryptionConfiguration

Status

Status indeks saat ini. Ketika nilainya adalah ACTIVE, indeks siap digunakan. Jika nilai bidang Status adalah FAILED, bidang ErrorMessage berisi pesan yang menjelaskan alasannya.

Jenis: String

Nilai Valid: CREATING | ACTIVE | DELETING | FAILED | UPDATING | SYSTEM_UPDATING

UpdatedAt

Unix saat indeks terakhir diperbarui.

Tipe: Timestamp

UserContextPolicy

Kebijakan konteks pengguna untuk indeks Amazon Kendra.

Jenis: String

Nilai Valid: ATTRIBUTE_FILTER | USER_TOKEN

UserGroupResolutionConfiguration

Apakah Anda telah mengaktifkan konfigurasi untuk mengambil tingkat akses grup dan pengguna dari sumberAWS IAM Identity Center (successor to AWS Single Sign-On) identitas.

Tipe: Objek UserGroupResolutionConfiguration

UserTokenConfigurations

Konfigurasi token pengguna untuk indeks Amazon Kendra.

Tipe: Array objek UserTokenConfiguration

Anggota Array: Jumlah maksimum 1 item.

Kesalahan

Untuk informasi tentang kesalahan yang umum untuk semua tindakan, lihat Kesalahan Umum.

AccessDeniedException

Anda tidak memiliki akses yang memadai untuk melakukan tindakan ini. Pastikan Anda memiliki kebijakan izin dan akun pengguna yang diperlukan dan coba lagi.

Kode Status HTTP: 400

InternalServerException

Masalah terjadi dengan server internal yang digunakan untuk layanan Amazon Kendra Anda. Tunggu beberapa menit dan coba lagi, atau hubungi Support.

Kode Status HTTP: 500

ResourceNotFoundException

Sumber daya yang ingin Anda gunakan tidak ada. Silakan periksa Anda telah menyediakan sumber daya yang benar dan coba lagi.

Kode Status HTTP: 400

ThrottlingException

Permintaan ditolak karena throttling permintaan. Kurangi jumlah permintaan dan coba lagi.

Kode Status HTTP: 400

ValidationException

Input gagal untuk memenuhi batasan yang ditetapkan oleh layanan Amazon Kendra. Berikan input yang benar dan coba lagi.

Kode Status HTTP: 400
