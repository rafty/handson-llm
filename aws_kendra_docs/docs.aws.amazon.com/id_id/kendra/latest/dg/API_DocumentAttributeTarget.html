DocumentAttributeTarget

Atribut dokumen target atau bidang metadata yang ingin Anda ubah saat memasukkan dokumen ke Amazon Kendra.

Misalnya, Anda dapat menghapus nomor identifikasi pelanggan yang terkait dengan dokumen, yang disimpan di bidang metadata dokumen yang disebut 'Customer_ID'. Anda menetapkan kunci target sebagai 'Customer_ID' dan flag penghapusan keTRUE. Ini menghapus semua nilai ID pelanggan di bidang 'Customer_ID'. Ini akan menggosok informasi identitas pribadi dari setiap metadata dokumen.

Amazon Kendra tidak dapat membuat bidang target jika belum dibuat sebagai bidang indeks. Setelah Anda membuat bidang indeks Anda, Anda dapat membuat bidang metadata dokumen menggunakanDocumentAttributeTarget. Amazon Kendra kemudian akan memetakan bidang metadata yang baru dibuat ke bidang indeks Anda.

Anda juga dapat menggunakan ini dengan DocumentAttributeCondition.

Daftar Isi

TargetDocumentAttributeKey

Pengenal dari atribut dokumen target atau bidang metadata.

Misalnya, 'Departemen' bisa menjadi pengidentifikasi untuk atribut target atau bidang metadata yang menyertakan nama departemen yang terkait dengan dokumen.

Jenis: String

Panjang Batasan: Panjang minimum 1. Panjang maksimum 200.

Pola: [a-zA-Z0-9_][a-zA-Z0-9_-]*

Wajib: Tidak

TargetDocumentAttributeValue

Nilai target yang ingin Anda buat untuk atribut target.

Misalnya, 'Keuangan' bisa menjadi nilai target untuk kunci atribut target 'Department'.

Tipe: Objek DocumentAttributeValue

Wajib: Tidak

TargetDocumentAttributeValueDeletion

TRUEuntuk menghapus nilai target yang ada untuk kunci atribut target yang Anda tentukan. Anda tidak dapat membuat nilai target dan mengatur ini keTRUE. Untuk membuat nilai target (TargetDocumentAttributeValue), atur ini keFALSE.

Tipe: Boolean

Wajib: Tidak
