Atribut dokumen atau bidang

Sebuah dokumen memiliki atribut atau bidang yang terkait dengannya. Bidang dokumen adalah sifat-sifat dokumen atau apa yang terkandung dalam struktur dokumen. Misalnya, setiap dokumen Anda mungkin berisi judul, teks badan, dan penulis. Anda juga dapat menambahkan bidang kustom untuk dokumen Anda sendiri. bidang kustom yang Anda tentukan untuk kebutuhan Anda sendiri. Misalnya, jika indeks Anda mencari dokumen pajak, Anda mungkin menentukan bidang kustom untuk jenis dokumen pajak seperti W-2, 1099, dan seterusnya.

Sebelum Anda dapat menggunakan bidang dokumen dalam kueri, itu harus dipetakan ke bidang indeks. Misalnya, bidang judul dapat dipetakan ke bidang_document_title. Untuk informasi selengkapnya, lihat Memetakan bidang. Untuk menambahkan bidang baru, Anda harus membuat bidang indeks untuk memetakan bidang. Anda membuat bidang kustom menggunakan konsol atau menggunakan UpdateIndexAPI.

Anda dapat menggunakan bidang dokumen untuk menyaring respons dan membuat saran pencarian faset. Misalnya, Anda dapat menyaring respons untuk hanya mengembalikan versi tertentu dari dokumen, atau Anda dapat menyaring pencarian untuk hanya mengembalikan 1099 jenis dokumen pajak yang sesuai dengan istilah pencarian. Untuk informasi selengkapnya, lihat Memfilter kueri.

Anda juga dapat menggunakan bidang dokumen untuk menyetel respons kueri secara manual. Misalnya, Anda dapat memilih untuk meningkatkan pentingnya bidang judul untuk meningkatkan bobot yang diberikan keAmazon Kendra bidang saat menentukan dokumen mana yang akan dikembalikan dalam respons. Untuk informasi selengkapnya, lihat Menyetel relevansi penelusuran.

Jika Anda menambahkan dokumen langsung ke indeks, Anda menentukan bidang dalam parameter masukan Dokumen keBatchPutDocument API. Anda menentukan nilai bidang kustom dalam jajaranDocumentAttribute objek. Jika Anda menggunakan sumber data, metode yang Anda gunakan untuk menambahkan bidang dokumen tergantung pada sumber data. Untuk informasi selengkapnya, lihat Membuat bidang dokumen kustom.
