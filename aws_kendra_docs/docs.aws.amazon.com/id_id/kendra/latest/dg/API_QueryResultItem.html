QueryResultItem

Hasil kueri tunggal.

Hasil kueri berisi informasi tentang dokumen yang dikembalikan oleh kueri. Ini termasuk lokasi asli dokumen, daftar atribut yang ditetapkan ke dokumen, dan teks yang relevan dari dokumen yang memenuhi kueri.

Daftar Isi

AdditionalAttributes

Satu atau lebih atribut tambahan yang terkait dengan hasil kueri.

Tipe: Array objek AdditionalResultAttribute

Wajib: Tidak

DocumentAttributes

Array atribut dokumen yang akan diberikan ke sebuah dokumen di hasil penelusuran. Sebagai contoh, penulis dokumen (_author) atau sumber URI () atau sumber URI (_source_uri) dokumen.

Tipe: Array objek DocumentAttribute

Wajib: Tidak

DocumentExcerpt

Ekstrak teks dalam dokumen. Berisi informasi tentang menyoroti istilah yang relevan dalam kutipan.

Tipe: Objek TextWithHighlights

Wajib: Tidak

DocumentId

Pengidentifikasi untuk dokumen tersebut.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 2048.

Wajib: Tidak

DocumentTitle

Judul dokumen. Berisi teks judul dan informasi untuk menyoroti istilah yang relevan dalam judul.

Tipe: Objek TextWithHighlights

Wajib: Tidak

DocumentURI

URI dari lokasi asli dokumen.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 2048.

Pola: ^(https?|ftp|file):\/\/([^\s]*)

Wajib: Tidak

FeedbackToken

Token yang mengidentifikasi hasil tertentu dari kueri tertentu. Gunakan token ini untuk memberikan umpan balik klik tayang bagi hasilnya. Untuk informasi selengkapnya, lihat Mengirimkan umpan balik.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 2048.

Pola: ^\P{C}*.\P{C}*$

Wajib: Tidak

Format

Jika dokumen dalam respon adalahANSWER, maka itu adalahTABLE jawaban atauTEXT jawaban.Type Jika jawaban tabel, kutipan meja dikembalikanTableExcerpt. Jika jawaban teks, kutipan teks dikembalikanDocumentExcerpt.

Jenis: String

Nilai yang Valid: TABLE | TEXT

Wajib: Tidak

Id

Pengidentifikasi untuk hasil kueri.

Jenis: String

Batasan Panjang: Panjang minimum 1. Panjang maksimum 73.

Wajib: Tidak

ScoreAttributes

Menunjukkan kepastian yang dimiliki Amazon Kendra bahwa hasil cocok dengan kueri yang Anda berikan. Setiap hasil ditempatkan ke kotak yang menunjukkan kepastian, VERY_HIGH, HIGH, MEDIUM dan LOW. Anda dapat menggunakan skor untuk menentukan apakah respons memenuhi kepastian yang diperlukan untuk aplikasi Anda.

Bidang ini hanya diatur ke LOW saat bidang Type diatur ke DOCUMENT dan Amazon Kendra tidak yakin bahwa hasilnya cocok dengan kueri.

Tipe: Objek ScoreAttributes

Wajib: Tidak

TableExcerpt

Kutipan dari tabel dalam dokumen.

Tipe: Objek TableExcerpt

Wajib: Tidak

Type

Jenis dokumen dalam respon. Misalnya, respons dapat mencakup jawaban pertanyaan yang relevan dengan kueri.

Jenis: String

Nilai yang Valid: DOCUMENT | QUESTION_ANSWER | ANSWER

Diperlukan: Tidak
