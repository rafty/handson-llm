Memulai dengan Amazon Lex V2

Amazon Lex V2 menyediakan operasi API yang dapat Anda integrasikan dengan aplikasi yang ada. Untuk daftar operasi yang didukung, lihat Referensi API. Anda dapat menggunakan salah satu opsi berikut:

  • AWS SDK — Saat menggunakan SDK, permintaan Anda ke Amazon Lex V2 ditandatangani dan diautentikasi secara otomatis menggunakan kredensi yang Anda berikan. Kami menyarankan agar Anda menggunakan SDK untuk membuat aplikasi Anda.

  • AWS CLI- Anda dapat menggunakanAWS CLI untuk mengakses fitur Amazon Lex V2 tanpa harus menulis kode apapun.

  • AWS Console — Konsol adalah cara paling mudah untuk memulai pengujian dan menggunakan Amazon Lex V2

Jika Anda baru mengenal Amazon Lex V2, kami sarankan agar Anda membacaCara kerjanya terlebih dahulu.
