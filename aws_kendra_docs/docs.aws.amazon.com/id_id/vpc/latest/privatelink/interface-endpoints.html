Mengkonfigurasi sebuah titik antarmuka

Setelah membuat endpoint VPC antarmuka, Anda dapat memperbarui konfigurasinya.

Tugas
  • Menambahkan atau menghapus subnet
  • Kelompok keamanan asosiasi
  • Mengedit kebijakan VPC
  • Aktifkan nama DNS pribadi
  • Kelola tag

Menambahkan atau menghapus subnet

Anda dapat memilih satu subnet per Availability Zone untuk titik akhir antarmuka. Jika Anda menambahkan subnet, kami membuat antarmuka jaringan titik akhir di subnet dan ditetapkan alamat IP privat dari rentang alamat IP subnet. Jika Anda menghapus subnet, kami menghapus antarmuka jaringan endpoint.

Untuk mengubah subnet menggunakan konsol

  1. Buka konsol Amazon VPC di https://console.aws.amazon.com/vpc/.

  2. Di panel navigasi, pilih Titik akhir.

  3. Pilih antarmuka titik akhir.

  4. Pilih Tindakan, Kelola subnet.

  5. Pilih atau batalkan pilihan subnet sesuai kebutuhan.

  6. Pilih Ubah subnet.

Untuk mengubah subnet menggunakan baris perintah

  • modify-vpc-endpoint (AWS CLI)

  • Edit-EC2VpcEndpoint (Alat untuk Windows PowerShell)

Kelompok keamanan asosiasi

Anda dapat mengubah grup keamanan yang terkait dengan antarmuka jaringan untuk titik akhir antarmuka Anda. Aturan grup keamanan mengontrol lalu lintas yang diizinkan ke antarmuka jaringan titik akhir dari sumber daya di VPC Anda.

Untuk mengubah grup keamanan menggunakan konsol

  1. Buka konsol Amazon VPC di https://console.aws.amazon.com/vpc/.

  2. Di panel navigasi, pilih Titik akhir.

  3. Pilih antarmuka titik akhir.

  4. Pilih Tindakan, Kelola grup keamanan.

  5. Pilih atau batalkan pilihan grup keamanan sesuai kebutuhan.

  6. Pilih Ubah grup keamanan.

Untuk mengubah grup keamanan menggunakan baris perintah

  • modify-vpc-endpoint (AWS CLI)

  • Edit-EC2VpcEndpoint (Alat untuk Windows PowerShell)

Mengedit kebijakan VPC

Anda dapat mengedit kebijakan VPC, yang mengontrol akses ke layanan VPC, yang mengontrol akses ke layanan titik akhir. Setelah Anda memperbarui kebijakan titik akhir, memerlukan waktu beberapa menit sampai perubahan diterapkan. Untuk informasi selengkapnya, lihat Kebijakan VPC endpoint.

Untuk mengubah kebijakan titik akhir menggunakan konsol

  1. Buka konsol Amazon VPC di https://console.aws.amazon.com/vpc/.

  2. Di panel navigasi, pilih Titik akhir.

  3. Pilih antarmuka titik akhir.

  4. Pilih Tindakan, Kelola kebijakan.

  5. Pilih Akses Penuh untuk memungkinkan akses penuh ke layanan, atau pilih Kustom dan lampirkan kebijakan khusus.

  6. Pilih Save (Simpan).

Untuk mengubah kebijakan titik akhir menggunakan baris perintah

  • modify-vpc-endpoint (AWS CLI)

  • Edit-EC2VpcEndpoint (Alat untuk Windows PowerShell)

Aktifkan nama DNS pribadi

Anda dapat mengaktifkan nama DNS pribadi untuk titik akhir VPC Anda. Untuk menggunakan nama DNS pribadi, Anda harus mengaktifkan nama host DNS dan resolusi DNS untuk VPC Anda. Setelah Anda mengaktifkan nama DNS pribadi, mungkin diperlukan beberapa menit agar alamat IP pribadi tersedia. Data DNS yang kami buat saat Anda mengaktifkan nama DNS pribadi bersifat pribadi. Oleh karena itu, nama DNS pribadi tidak dapat diselesaikan secara publik.

Untuk mengubah opsi nama DNS pribadi menggunakan konsol

  1. Buka konsol Amazon VPC di https://console.aws.amazon.com/vpc/.

  2. Di panel navigasi, pilih Titik akhir.

  3. Pilih antarmuka titik akhir.

  4. Pilih Tindakan, Ubah nama DNS pribadi.

  5. Pilih atau hapus Aktifkan untuk titik akhir ini sesuai kebutuhan.

  6. Jika layanannya adalah Amazon S3, memilih Enable for endpoint ini pada langkah sebelumnya juga akan memilih Enable DNS privat hanya untuk titik akhir masuk. Jika Anda lebih suka fungsionalitas DNS pribadi standar, hapus Aktifkan DNS pribadi hanya untuk titik akhir masuk. Jika Anda tidak memiliki titik akhir gateway untuk Amazon S3 selain titik akhir antarmuka untuk Amazon S3, dan Anda memilih Aktifkan DNS pribadi hanya untuk titik akhir masuk, Anda akan menerima kesalahan saat menyimpan perubahan pada langkah berikutnya. Untuk informasi selengkapnya, lihat DNS pribadi.

  7. Pilih Save changes (Simpan perubahan).

Untuk mengubah opsi nama privat menggunakan baris perintah

  • modify-vpc-endpoint (AWS CLI)

  • Edit-EC2VpcEndpoint (Alat untuk Windows PowerShell)

Kelola tag

Anda dapat menandai titik akhir antarmuka untuk membantu mengidentifikasi atau mengkategorikannya sesuai dengan kebutuhan organisasi.

Untuk mengelola tag menggunakan konsol

  1. Buka konsol Amazon VPC di https://console.aws.amazon.com/vpc/.

  2. Di panel navigasi, pilih Titik akhir.

  3. Pilih antarmuka titik akhir.

  4. Pilih Tindakan, Kelola tag.

  5. Untuk setiap tag untuk menambahkan pilih Tambahkan tag baru dan masukkan kunci tag dan nilai tag.

  6. Untuk menghapus tanda, pilih Hapus di sebelah kanan kunci dan nilai tag.

  7. Pilih Save (Simpan).

Untuk mengelola tag menggunakan baris perintah

  • membuat-tag dan menghapus-tag (AWS CLI)

  • Baru-EC2tag dan menghapus-EC2TAG (Alat untuk Windows PowerShell)
