Autentikasi multi-faktor

Saat Anda mengaktifkan autentikasi multi-faktor (MFA), pengguna harus masuk ke portal AWS akses dengan nama pengguna dan kata sandi mereka. Ini adalah faktor pertama, sesuatu yang mereka tahu. Pengguna juga harus masuk dengan kode atau kunci keamanan. Ini adalah faktor kedua, sesuatu yang mereka miliki atau sesuatu yang mereka miliki. Faktor kedua bisa berupa kode otentikasi yang dihasilkan dari perangkat seluler mereka atau sebagai alternatif dengan mengetuk kunci keamanan yang terhubung ke komputer mereka. Secara keseluruhan, beberapa faktor ini memberikan peningkatan keamanan dengan mencegah akses tidak sah ke AWS sumber daya Anda kecuali jika tantangan MFA yang valid telah berhasil diselesaikan. Setiap pengguna dapat mendaftarkan hingga delapan perangkat MFA.

penting

Sebagai praktik keamanan terbaik, kami sangat menyarankan agar Anda mengaktifkan autentikasi multi-faktor. MFA menyediakan cara sederhana dan aman untuk menambahkan lapisan perlindungan tambahan di atas mekanisme otentikasi default nama pengguna dan kata sandi.
