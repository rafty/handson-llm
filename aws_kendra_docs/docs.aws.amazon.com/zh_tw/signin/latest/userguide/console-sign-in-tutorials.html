登入到 AWS Management Console

當您AWS Management Console從主要AWS登入網址 (https://signin.aws.amazon.com) 登入時，您必須選擇您的使用者類型，即根使用者或 IAM 使用者。

root 使用者擁有不受限制的帳戶存取權，並且與建立. AWS 帳戶 然後，root 使用者會建立其他類型的使用者，例如 IAM 使用者和AWS IAM Identity Center (AWS Single Sign-On 的繼任者) 的使用者，並為他們指派存取登入資料。

IAM 使用者是您AWS 帳戶內具備特定自訂許可的身分。IAM 使用者登入時，他們可以使用包含其AWS 帳戶或別名的登入 URL，例如， https://account_alias_or_id.signin.aws.amazon.com/console/而不是主要AWS登入 URL https://signin.aws.amazon.com/。

如果您不確定自己是哪種使用者類型，請參閱使用者類型。
