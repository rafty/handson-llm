如何登入 AWS

登入方式AWS取決於您的AWS使用者類型。有不同的AWS使用者類型。您可以是帳戶根使用者、IAM 使用者、IAM Identity Center 中的使用者、聯合身分或使用AWS產生器 ID。如需詳細資訊，請參閱使用者類型。

您可以使AWS用下列任何一種方法登入來存取：

  • 登入到 AWS Management Console以根使用者身分或 IAM 使用者身分

  • 登入AWS存取入口網站以 IAM Identity Center 中的使用者身分驗證

  • 以同盟身分登入

  • 透過登入 AWS Command Line Interface和其他程序化方法，如 API 和 SDK（軟件開發工具包）

  • 使用AWS生成器 ID 登錄

如果您要建立新的項目AWS 帳戶，請參閱《AWS設定指南》AWS 帳戶中的第 1 部分：設定新項目。
