Amazon Lex V2

Amazon Lex V2 提供的 API 操作可讓您與現有的應用程式整合。對於支持的操作的清單，請參閱 API 參考。您可以使用下列任一選項：

  • AWS 開發套件 — 使用開發套件時，系統會使用您提供的登入資料自動簽署和驗證 Amazon Lex V2 的請求。我們建議您使用 SDK 建立您的應用程式。

  • AWS CLI— 您可以使AWS CLI用 Amazon Lex V2 功能，無需編寫任何程式碼。

  • AWS Lex V2，主控台是開始測試和使用 Amazon Lex V2 的最簡單方式

如果您是 Amazon Lex V2 的新手，建議您運作方式先閱讀。
