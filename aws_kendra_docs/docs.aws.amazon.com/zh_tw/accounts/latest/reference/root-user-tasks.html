需要根使用者憑證的任務

重要

下列 AWS Identity and Access Management (IAM) 動作將於 2023 年 7 月終止標準支援：aws-portal:ModifyAccount和aws-portal:ViewAccount。請參閱，用精細AWS Billing動作，以便您可以存取AWS Billing、AWS和主控台AWS。

如果您AWS於 2023 年 3 月 6 日之前建立了帳戶或AWS Organizations管理帳戶，則精細動作將在 2023 年 7 月 6 日之前建立了帳戶或管理帳戶，精細動作 建議您新增精細動作，但不要移除具有 aws-portal 或 purchase-orders 字首的現有許可。

如果您AWS於 2023 年 3 月 6 日或之後建立了帳戶或AWS Organizations管理帳戶，則精細動作會立即生效。

建議您在中配置管理使用者，AWS IAM Identity Center (successor to AWS Single Sign-On)以執行每日工作和存取AWS資源。不過，只有當您以帳號的 root 使用者身分登入時，才能執行下列工作。

任務

  • 變更您的帳戶設定。這包括帳戶名稱、電子郵件地址、root 使用者密碼和 root 使用者存取金鑰。其他帳戶設定，例如聯絡資訊、付款貨幣偏好設定AWS 區域，以及不需要 root 使用者憑證。

  • 還原 IAM 使用者許可。如果唯一的 IAM 管理員不小心撤銷了自己的許可，您可以以 root 使用者身分登入以編輯政策並還原這些許可。

  • 針對 BBilling and Cost Management 主控台啟用 IAM 存取權。

  • 檢視特定稅務發票。擁有 aws-Portal 的 IAM 使用者：ViewBilling權限可以檢視和下載來自AWS歐洲的加值稅發票，但不能檢視和下載 AWS Inc. 或亞馬遜網際網路服務私人有限公司 (AISPL)。

  • 關閉您的 AWS 帳戶.

  • 在預留執行個體市場註冊為賣方。

  • 將 Amazon S3 儲存貯體設定為啟用 MFA (多重因素認證)。

  • 編輯或刪除 Amazon Simple Storage Service (Amazon S3) 儲存貯體政策，其中包含無效的虛擬私有雲端 (VPC) ID 或 VPC 端點識別碼。

  • 註冊 AWS GovCloud (US)。

  • 請求AWS GovCloud (US)帳號根使用者存取金鑰AWS Support。
