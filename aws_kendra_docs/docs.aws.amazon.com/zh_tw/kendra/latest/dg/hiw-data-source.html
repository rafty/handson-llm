資料來源

資料來源是Amazon Kendra連線至文件或內容並編製索引的資料儲存庫或位置。例如，您可以設定為連線Amazon Kendra SharePoint 至 Microsoft，以編目和索引儲存在此來源中的文件。您也可以透過提供要檢索的 URLAmazon Kendra 來編製網頁索引。您可以自動將資料來源與Amazon Kendra索引同步，以便在索引中新增、更新或刪除資料來源中的文件也會新增、更新或刪除。

支援的資料來源包括：

  • 露天

  • Amazon S3水桶

  • Amazon RDS對於 MySQL,Amazon RDS 對於 PostgreSQL,Amazon Aurora MySQL,Amazon Aurora PostgreSQL 數據庫

  • Amazon FSx

  • Amazon Kendra網絡爬蟲

  • Amazon WorkDocs

  • Box (立方體)

  • 匯合

  • 自訂資料來源

  • Dropbox

  • GitHub

  • Gmail

  • 谷歌工作區硬盤

  • 吉拉

  • 微軟交易所

  • 微軟 OneDrive

  • 微軟 SharePoint

  • 微軟團隊

  • 微軟亞默

  • 麻煩

  • 銷售人員

  • ServiceNow

  • 鬆弛

  • ZENDESK

如需支援的文件類型清單，Amazon Kendra請參閱文件類型。

注意

如要建立索引，您不需要資料來源。您可以直接將文件新增至索引。如需詳細資訊，請參閱直接將文件新增至索引。

若要使用資料來源索引文件。

  1. 建立索引。

  2. 建立資料來源。

如需Amazon Kendra主控台或AWS CLI 的逐步解說，請參閱入門。
