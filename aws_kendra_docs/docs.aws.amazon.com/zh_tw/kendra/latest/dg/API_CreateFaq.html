CreateFaq

使用存放在 Amazon S3 儲存貯體中的指定常見問答集檔案集 (FAQ)。

將常見問答集的常見問答集是一種非同步操作。

如需使用 Python 和 Java SDK 將常見問題集新增至索引的範例，請參閱使用常見問題集檔案。

請求語法

{
   "ClientToken": "string",
   "Description": "string",
   "FileFormat": "string",
   "IndexId": "string",
   "LanguageCode": "string",
   "Name": "string",
   "RoleArn": "string",
   "S3Path": { 
      "Bucket": "string",
      "Key": "string"
   },
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

請求參數

如需所有動作共用參數的相關資訊，請參閱一般參數。

請求接受採用 JSON 格式的下列資料。

ClientToken

您提供用來識別建立常見問題集之要求的權杖。對具有相同客戶端令牌的CreateFaqRequest API 進行多次調用將僅創建一個常見問題解答。

類型：字串

長度限制：長度下限為 1。長度上限為 100。

必要：否

Description

FAQ 的描述。

類型：字串

長度限制：長度下限為 0。長度上限為 1000。

模式：^\P{C}*$

必要：否

FileFormat

常見問答集的格式。您可以在基本的 CSV 格式、標頭中包含自訂屬性的 CSV 格式，以及包含自訂屬性的 JSON 格式中選擇。

格式必須符合在S3Path參數中識別的 S3 儲存貯體中檔案格式。

如需詳細資訊，請參閱新增問題和答案。

類型：字串

有效值： CSV | CSV_WITH_HEADER | JSON

必要：否

IndexId

常見問答集的索引標識符。

類型：字串

長度上限為 36。

模式：[a-zA-Z0-9][a-zA-Z0-9-]*

必要：是

LanguageCode

一種語言的代碼。這可讓您支援常見問題集文件的語言。預設支援英文。如需支援語言 (包括其代碼) 的詳細資訊，請參閱以英文以外的語言新增文件。

類型：字串

長度上限為 2。長度上上上上限為 10。

模式：[a-zA-Z-]*

必要：否

Name

常見問答集的名稱。

類型：字串

長度限制：長度下限為 1。長度上限為 100。

模式：[a-zA-Z0-9][a-zA-Z0-9_-]*

必要：是

RoleArn

角色的 Amazon Resource Name (ARN)，具備可存取包含常見問答集的 S3 儲存貯體許可。如需詳細資訊，請參閱 Amazon Kendra 的 IAM 角色。

類型：字串

長度限制：長度下限為 0。長度上上上上限為 1284。

模式：arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

必要：是

S3Path

S3 中常見問答集檔案的路徑。

類型：S3Path 物件

必要：是

Tags

識別常見問答集的鍵值組清單。您可以使用標籤，識別和組織您的資源，並控制對您資源的存取。

類型：Tag 物件陣列

陣列成員：項目數下限為 0。項目數上限為 200。

必要：否

回應語法

{
   "Id": "string"
}

回應元素

如果動作成功，則服務傳回 HTTP 200 回應。

服務會傳回下列 JSON 格式的資料。

Id

常見問答集的識別碼。

類型：字串

長度限制：長度下限為 1。長度上限為 100。

模式：[a-zA-Z0-9][a-zA-Z0-9_-]*

錯誤

如需所有動作常見的錯誤相關資訊，請參閱常見錯誤。

AccessDeniedException

您沒有執行此動作的存取。請確定您擁有必要的權限原則和使用者帳戶，然後再試一次。

HTTP 狀態碼：400

ConflictException

與請求發生衝突。請修正任何與資源不一致的問題，然後再試一次。

HTTP 狀態碼：400

InternalServerException

用於 Amazon Kendra 服務的內部伺服器發生問題。請稍候幾分鐘後再試一次，或聯絡 Sup port 尋求協助。

HTTP 狀態碼：500

ResourceNotFoundException

您要使用的資源不存在。請檢查您是否提供了正確的資源，然後再試一次。

HTTP 狀態碼：400

ServiceQuotaExceededException

您已超過 Amazon Kendra 服務的設定限制。如需詳細資訊，請參閱配額，或聯絡 Sup port 部門以詢問有關增加限制的問題。

HTTP 狀態碼：400

ThrottlingException

由於請求調節，因此請求遭到拒絕。請減少要求數目，然後再試一次。

HTTP 狀態碼：400

ValidationException

輸入不符合 Amazon Kendra 服務設定的限為 Amazon Kendra 服務。請提供正確的輸入，然後重試。

HTTP 狀態碼：400
