變更您的 IAM Identity Center Identity Center 來源

警告

在 IAM 身分中心設定中變更身分識別來源可能會影響使用者和群組資訊的保留。若要安全地執行此操作，建議您檢閱變更身分識別來源的考量事項。當您變更身分識別來源時，會產生新的身分識別來源 ID。AWS_SSO在將模式設定為「中」之前，請檢查您是否使用正確的 ID UserGroupResolutionConfiguration。

變更您的 IAM Identity Center Identity Center Identity Center

  1. 開啟 IAM 身分中心 > 主控台。

  2. 選擇 Settings (設定)。

  3. 在 [設定] 頁面的 [身分識別來源] 下，選擇 [變更]。

  4. 在 [變更身分識別來源] 頁面上，選取偏好的身分識別來源，然後選擇 [下一步]。
