Google 雲端硬碟

Google 雲端硬盤是一種基於雲的文件存儲服務。您可以使用在 Google 雲端硬碟資料Amazon Kendra來源的共用雲端硬碟、「我的雲端硬碟」和「與我共享」資料夾中儲存的文件編製索引。您可以為 Google Workspace 文件和文件類型中列出的文件編製索引。您也可以使用包含和排除篩選器，依檔案名稱、檔案類型和檔案路徑為內容建立索引。

您可以使用Amazon Kendra主控台、API 或 TemplateConfigurationAPI 連線Amazon Kendra到 Google 雲端硬碟資料來源。GoogleDriveConfiguration

Amazon Kendra有兩個版本的谷歌驅動器連接器。每個版本的支援功能包括：

谷歌驅動器連接器 V1.0/GoogleDriveConfigurationAPI

  • 欄位映射

  • 包含/排除過濾器

谷歌驅動器連接器 V2.0/TemplateConfigurationAPI

  • 使用者內容篩選

  • Virtual Private Cloud (VPC) (虛擬私有雲端 (VPC))

  • 包含/排除過濾器

  • 同步所有文檔/僅同步新的，修改或刪除的文檔

注意

Support 谷歌驅動器連接器 V1.0 /谷歌 DriveConfiguration API 定於 2023 年 6 月結束。我們建議使用谷歌驅動器連接器 V2.0/ TemplateConfiguration API。
