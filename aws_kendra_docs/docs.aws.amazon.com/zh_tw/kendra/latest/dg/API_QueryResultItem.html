QueryResultItem

單一查詢結果。

查詢結果包含查詢所傳回之文件的相關資訊。這包括文件的原始位置、指派給文件的屬性清單，以及符合查詢的文件中的相關文字。

內容

AdditionalAttributes

與查詢結果相關聯的一或多個其他屬性。

類型：AdditionalResultAttribute 物件陣列

必要：否

DocumentAttributes

指派給搜尋結果中文件的文件屬性陣列。例如，文件作者 (_author) 或文件的來源 URI (_source_uri)。

類型：DocumentAttribute 物件陣列

必要：否

DocumentExcerpt

文件中文字的摘錄。包含有關在摘錄中反白顯示相關術語的資訊。

類型：TextWithHighlights 物件

必要：否

DocumentId

文件的識別碼。

類型：String

長度限制：長度下限為 1。長度上限為 2048。

必要：否

DocumentTitle

文件標題。包含標題的文字和資訊，以反白標題中的相關術語。

類型：TextWithHighlights 物件

必要：否

DocumentURI

文件原始位置的 URI。

類型：String

長度限制：長度下限為 1。長度上限為 2048。

模式：^(https?|ftp|file):\/\/([^\s]*)

必要：否

FeedbackToken

標識特定查詢的特定結果的令牌。使用此權杖提供結果的點選回饋。如需詳細資訊，請參閱提交意見反應。

類型：String

長度限制：長度下限為 1。長度上限為 2048。

模式：^\P{C}*.\P{C}*$

必要：否

Format

如果響應中Type的文檔是ANSWER，那麼它是一個TABLE答案或TEXT答案。如果這是表格答案，則會在中返回表摘錄TableExcerpt。如果是文字答案，則會在中傳回文字摘錄DocumentExcerpt。

類型：String

有效值： TABLE | TEXT

必要：否

Id

查詢結果的識別碼。

類型：String

長度限制：長度下限為 1。長度上限為 73。

必要：否

ScoreAttributes

表示 Amazon Kendra 對結果與您提供的查詢相符的信心。每個結果都會放入指示信賴度、VERY_HIGHHIGH、MEDIUM和的資料桶中LOW。您可以使用分數來判斷回應是否符合應用程式所需的可信度。

只有LOW當欄位設定為DOCUMENT且 Amazon Kendra 不確定結果與查詢相符時，此Type欄位才會設定為。

類型：ScoreAttributes 物件

必要：否

TableExcerpt

文件中表格的摘錄。

類型：TableExcerpt 物件

必要：否

Type

回應中的文件類型。例如，回應可能包含與查詢相關的問題答案。

類型：String

有效值： DOCUMENT | QUESTION_ANSWER | ANSWER

必要：否
