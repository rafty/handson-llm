Salesforce

Salesforce 是一種客戶關係管理（CRM）工具，用於管理支持，銷售和營銷團隊。您可以使Amazon Kendra用索引您的 Salesforce 標準物件，甚至是自訂物件。

您可以使用Amazon Kendra主控台、API 或 TemplateConfigurationAPI 連線Amazon Kendra到 Salesforce 資料來源。SalesforceConfiguration

Amazon Kendra有兩個版本的 Salesforce 連接器。每個版本的支援功能包括：

銷售力連接器 1.0 版/SalesforceConfiguration應用程式介面

  • 欄位映射

  • 使用者內容篩選

  • 包含/排除過濾器

銷售力量連接器 2.0 版/TemplateConfigurationAPI

  • 欄位映射

  • 使用者內容篩選

  • 包含/排除過濾器

  • 實體附件支援

  • 身分檢索

  • VPC 支援

  • 同步所有文檔/僅同步新的，修改或刪除的文檔

注意

Support 於 Salesforce 連接器 1.0/ SalesforceConfiguration API 的支援預定於 2023 年 6 月結束。我們建議您使用連接器 2.0/ TemplateConfiguration API。

如需Amazon Kendra Salesforce 資料來源連接器的疑難排解，請參閱對資料來源進行。
