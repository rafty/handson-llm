Confluence

Confluence 是一種協作工作管理工具，專為共享，存儲和工作項目規劃，軟件開發和產品管理而設計。您可以用Amazon Kendra來索引您的 Confluence 空間、頁面 (包括巢狀頁面)、部落格，以及已編製索引的頁面和部落格的留言和附件。

Amazon Kendra同時支持匯流服務器和匯流雲。

注意

默認情況下，Amazon Kendra不索引匯流檔案和個人空間。您可以選擇在建立資料來源時對其建立索引。如果您不Amazon Kendra想索引空間，請在匯合中將其標記為私有。

您可以使用Amazon Kendra主控台、API 或 TemplateConfigurationAPI 連線Amazon Kendra到您的匯流資料來源。ConfluenceConfiguration

Amazon Kendra具有兩個版本的匯流連接器。每個版本的支援功能包括：

匯流連接器 ConfluenceConfiguration

  • 欄位映射

  • 使用者內容篩選

  • 包含/排除過濾器

  • (僅限 Confluence 伺服器) Virtual private cloud (VPC)

匯流連接器 V2.0 TemplateConfiguration

  • 欄位映射

  • 使用者內容篩選

  • Virtual Private Cloud (VPC) (虛擬私有雲端 (VPC))

  • 同步所有文檔/僅同步新的，修改或刪除的文檔

  • 包含/排除模式

注意

匯流連接器 V1.0/ ConfluenceConfiguration API 的 Support 定於 2023 年 6 月結束。我們建議使用匯流連接器 V2.0/ TemplateConfiguration API。

若要疑難排解您的Amazon Kendra匯流資料來源連接器，請參閱對資料來源進行。
