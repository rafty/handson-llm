S3Path

在 Amazon S3 儲存貯體中尋找特定檔案所需的資訊。

內容

Bucket

包含檔案的 S3 儲存貯體名稱。

類型：String

長度限制：長度下限為 3。長度上限為 63。

模式：[a-z0-9][\.\-a-z0-9]{1,61}[a-z0-9]

必要：是

Key

檔案名稱。

類型：String

長度限制：長度下限為 1。長度上限為 1024。

必要：是
