請求參數

如需有關所有動作共用之參數的資訊，請參閱一般參數。

請求接受採用 JSON 格式的下列資料。

CapacityUnits

設定索引應使用的其他文件儲存和查詢容量單位的數目。您每天最多可以變更索引的容量 5 次，或進行 5 次 API 呼叫。

如果您使用額外的儲存單元，則無法將儲存容量降低到滿足索引儲存需求所需的儲存容量。

類型：CapacityUnitsConfiguration 物件

必要：否

Description

索引的新的描述。

類型：字串

長度限制：長度下限為 0。長度上限為 1000。

模式：^\P{C}*$

必要：否

DocumentMetadataConfigurationUpdates

您要針對索引更新的文件中繼資料組態。文件中繼資料是與文件相關聯的欄位或屬性。例如，與每份文件相關聯的公司部門名稱。

類型：DocumentMetadataConfiguration 物件陣列

陣列成員：項目數下限為 0。項目數上限為 500。

必要：否

Id

您要更新之索引的識別碼。

類型：字串

長度限制：固定長度為 36。

模式：[a-zA-Z0-9][a-zA-Z0-9-]*

必要：是

Name

您要更新的索引名稱。

類型：字串

長度限制：長度下限為 1。長度上限為 1000。

模式：[a-zA-Z0-9][a-zA-Z0-9_-]*

必要：否

RoleArn

一種AWS Identity and Access Management（IAM）角色，可授予 Amazon Kendra 存取 Amazon CloudWatch logs 和指標的許可。

類型：字串

長度限制：長度下限為 0。長度上限為 1284。

模式：arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

必要：否

UserContextPolicy

使用者內容政策。

類型：字串

有效值： ATTRIBUTE_FILTER | USER_TOKEN

必要：否

UserGroupResolutionConfiguration

啟用從AWS IAM Identity Center (successor to AWS Single Sign-On)身分識別來源擷取群組和使用者的存取層級。若要對此進行設定，請參閱UserGroupResolutionConfiguration。

類型：UserGroupResolutionConfiguration 物件

必要：否

UserTokenConfigurations

使用者權杖組態。

類型：UserTokenConfiguration 物件陣列

陣列成員：最多 1 個項目。

必要：否

回應元素

如果動作成功，則服務會傳回具空 HTTP 內文的 HTTP 200 回應。

錯誤

如需所有動作常見的錯誤相關資訊，請參閱常見錯誤。

AccessDeniedException

您沒有足夠的存取執行此動作的許可。請確定您擁有必要的權限原則和使用者帳戶，然後再試一次。

HTTP 狀態碼：400

ConflictException

與請求發生衝突。請修正任何與資源不一致的問題，然後再試一次。

HTTP 狀態碼：400

InternalServerException

用於 Amazon Kendra 服務的內部伺服器發生問題。請稍候幾分鐘，然後再試一次，或聯絡 Sup port 尋求協助。

HTTP 狀態碼：500

ResourceNotFoundException

您要使用的資源不存在。請檢查您是否提供了正確的資源，然後再試一次。

HTTP 狀態碼：400

ServiceQuotaExceededException

您已超過 Amazon Kendra 服務的設定限制。如需詳細資訊，請參閱配額，或聯絡 Sup port 部門以詢問有關增加限制的問題。

HTTP 狀態碼：400

ThrottlingException

由於請求調節，因此請求遭到拒絕。請減少申請數目，然後再試一次。

HTTP 狀態碼：400

ValidationException

輸入不符合 Amazon Kendra 服務規定的限制。請提供正確的輸入，然後重試。

HTTP 狀態碼：400
