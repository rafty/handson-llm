直接將常見問答集 (FAQ) 新增至索引

您可以使用主控台或CreateFaq API 將常見問題 (FAQ) 直接新增至索引。將常見問答集新增至索引是一種非同步操作。您可以將常見問題集的資料放在儲存Amazon Simple Storage Service貯體中的檔案中。您可以使用 CSV 或 JSON 文件作為常見問題解答的輸入：

  • 基本 CSV — 一個 CSV 檔案，其中每一行都包含問題、答案和可選 URL，其中包含有關答案的詳細資訊。

  • 自訂 CSV — 包含問題、答案和標題的 CSV 檔案，可定義自訂屬性，您可以使用這些屬性來複製、顯示或排序常見問題集回應。您也可以定義存取控制屬性，將常見問題集回應限制給特定使用者和群組。

  • JSON — 包含問題和答案的 JSON 檔案。也可以選擇包含自訂和存取控制屬性。您可以定義要刻面、顯示和排序常見問題集回應的屬性。或者，您可以使用存取控制屬性，將常見問題集回應限制給特定使用者和群組。

例如，以下是一個基本的 CSV 文件，該文件提供有關美國華盛頓州斯波坎免費診所和美國密蘇里州山景城的問題的答案。

How many free clinics are in Spokane WA?, 13, https://www.freeclinics.com/
How many free clinics are there in Mountain View Missouri?, 7, https://www.freeclinics.com/

當您使用自訂 CSV 或 JSON 檔案進行輸入時，您可以為常見問題集問題宣告自訂屬性。例如，您可以建立自訂屬性，將每個 FAQ 問題指派給部門。在回應中傳回常見問題集時，您可以使用該部門作為縮小搜尋範圍的方面。

自訂屬性必須對應至索引欄位。您可以使用內建欄位或指定自訂索引欄位。在主控台中，您可以使用 Facet 定義頁面來建立索引欄位。在使用 API 時，您必須先使用UpdateIndex API 建立索引欄位。

FAQ 檔案中的屬性類型必須與關聯索引欄位的類型相符。例如，內建_authors欄位是STRING_LIST類型欄位。因此，您必須在 FAQ 檔案中以字串清單形式提供_authors欄位的值。您可以使用主控台中的 Facet 定義頁面或使用DescribeIndex API 來檢查索引欄位的類型。

當您建立對應至自訂屬性的索引欄位時，您可以將其標示為可顯示、可列印或排序。您無法使自訂屬性可搜尋。

除了自訂屬性之外，您也可以在自訂 CSV 或 JSON 檔案中使用下列內建屬性：

  • _authors(字串清單) — 常見問題解答的作者清單。

  • _category(字串) — 將常見問題解答的答案與其他類似文件分組的類別。

  • _created_at(ISO 801 編碼的字串)-常見問答集問答集問題的建立日期和時間。日期和時間必須格式化為 ISO 8601 編碼的字符串。

    如有需要，您還可以 ISO 8601 日期時間格式中包含 ISO 8601 日期時間格式。例如，2012-03-25T12 30:30 (加上 10 秒)，是以 ISO 8601 0101 年 3 月 25 日中午 12:30 (加上 10 秒) 的歐洲中午 12:30 (加上 10 秒)。

  • _last_updated_at(ISO 8601 編碼字串)-常見問題問題的更新日期和時間。日期和時間必須結束為 ISO 8601 編碼的字串。

    例如，2012-03-25T12 30:30 (加上 10 秒)，是以 ISO 8601 0101 年 3 月 25 日中午 12:30 (加上 10 秒) 的歐洲中午 12:30 (加上 10 秒)。

  • _source_uri(字串) — 文件的 URL，其中包含有關常見問題答案的詳細資訊。

  • _version(字串) — 常見問題問題的版本。

  • _view_count(長) — 在搜尋結果中檢視常見問題的次數。

CSV 檔案

當您想為常見問題解答使用簡單的結構時，請使用基本的 CSV 文件。在基本 CSV 檔案中，每一行都有兩個或三個欄位：一個問題、一個答案，以及指向包含更多資訊的文件的選用來源 URI。

對於逗號分隔值 (CSV) 檔案，檔案的內容必須遵循 RFC 4180 一般格式和 MIME 類型。

以下是基本 CSV 格式的常見問題集檔案。

How many free clinics are in Spokane WA?, 13, https://www.freeclinics.com/
How many free clinics are there in Mountain View Missouri?, 7, https://www.freeclinics.com/

自訂 CSV 檔案

當您想要將自訂屬性新增至常見問題集時，請使用自訂 CSV 檔案。對於自訂 CSV 檔案，您可以在 CSV 檔案中使用標題列來定義其他屬性。

CSV 檔案必須包含以下兩種必需屬性：

  • _question— 常見問題

  • _answer— 常見問題的答案

您的檔案可以包含內建屬性和自訂屬性。以下是自訂 CSV 檔案的範例。

_question,_answer,_last_updated_at,custom_string
How many free clinics are in Spokane WA?, 13, 2012-03-25T12:30:10+01:00, Note: Some free clinics require you to meet certain criteria in order to use their services
How many free clinics are there in Mountain View Missouri?, 7, 2012-03-25T12:30:10+01:00, Note: Some free clinics require you to meet certain criteria in order to use their services

對於逗號分隔值 (CSV) 檔案，自訂屬性檔案的內容必須遵循 RFC 4180 一般格式和 MIME 類型。

有四種類型的自訂屬性：

  • 日期：ISO 8601 編碼的日期和時間值。

    例如，2012-03-25T12 30:30 (加上 10 秒)，是以 ISO 8601 0101 年 3 月 25 日中午 12:30 (加上 10 秒) 的歐洲中午 12:30 (加上 10 秒)。

  • 長 — 數字，例如1234。

  • 字串 — 字串值。如果您的字串包含逗號，請以雙引號 (「) 括住整個值 (例如，"custom attribute, and more")。

  • 字串清單-字串值清單。以逗號分隔的清單形式列出值，其中包含引號 (「"item1, item2, item3")。如果清單只包含單一項目，您可以省略引號 (例如item1)。

自訂 CSV 檔案可以包含使用者內容欄位。您可以使用這些欄位來限制特定使用者和群組對常見問題集的存取。若要篩選使用者前後關聯，使用者必須在查詢中提供使用者和群組資訊。否則，將返回所有相關常見問題解答。如需詳細資訊，請參閱篩選使用者內容。

常見問題解答有四種用戶上下文過濾器：

  • _acl_user_allow允許清單中的使用者可以在查詢回應中看到常見問題集。常見問題集不會傳回給其他使用者。

  • _acl_user_deny拒絕清單中的使用者無法在查詢回應中看到常見問題集。當常見問題集與查詢相關時，會傳回給所有其他使用者。

  • _acl_group_allow屬於允許群組成員的使用者可以在查詢回應中看到常見問題集。常見問題集不會傳回給身為其他群組成員的使用者。

  • _acl_group_deny屬於拒絕群組成員的使用者無法在查詢回應中看到常見問題集。常見問題集會在與查詢相關時傳回給其他群組。

在以引號括住的逗號分隔清單中，提供允許和拒絕清單的值 (例如，"user1,user2,user3")。您可以將使用者或群組包含在允許清單或拒絕清單中，但不能同時包含兩者。如果您同時包含使用者或群組，您會收到錯誤。

以下是包含使用者內容資訊的自訂 CSV 檔案範例。

_question, _answer, _acl_user_allow, _acl_user_deny, _acl_group_allow, _acl_group_deny
How many free clinics are in Spokane WA?, 13, userID4565, "userID6201, userID7552", groupBasicRate, "groupBasicPlusRate,groupPremiumRate"

JSON 檔案

您可以使用 JSON 檔案來提供索引的問題、答案和屬性。您可以將任何內建屬性和自訂屬性新增至常見問題集。這是 JSON 檔案的結構定義。

{
    "SchemaVersion": 1,
    "FaqDocuments": [
        {
            "Question": string,
            "Answer": string,
            "Attributes": {
                string: object
                additional attributes
            },
            "AccessControlList": [
               {
                   "Name": string,
                   "Type": enum( "GROUP" | "USER" ),
                   "Access": enum( "ALLOW" | "DENY" )
               },
               additional user context
            ]
        },
        additional FAQ documents
    ]
}

此範例 JSON 檔案顯示兩個常見問題集文件。其中一個只有必要的問題和答案。另一個也有其他屬性和使用者前後關聯資訊。

{
    "SchemaVersion": 1,
    "FaqDocuments": [
        {
            "Question": "How many free clinics are in Spokane WA?",
            "Answer": "13"
        },
        {
            "Question": "How many free clinics are there in Mountain View Missouri?",
            "Answer": "7",
            "Attributes": {
                "_source_uri": "https://www.freeclinics.com",
                "_category": "Charitable Clinics"
            },
            "AccessControlList": [
               {
                   "Name": "user@amazon.com",
                   "Type": "USER",
                   "Access": "ALLOW"
               },
               {
                   "Name": "Admin",
                   "Type": "GROUP",
                   "Access": "ALLOW"
               }
            ]
        }
    ]
}

JSON 檔案有四種類型的自訂屬性：

  • 日期 — 具有 ISO 8601 編碼日期和時間值的 JSON 字串值。

    ISO 8601 日期時間格式中必須包含時區。例如，2012-03-25T12 30:30 (加上 10 秒)，是以 ISO 8601 0101 年 3 月 25 日中午 12:30 (加上 10 秒) 的歐洲中午 12:30 (加上 10 秒)。

  • 長 — JSON 數字值，例如1234。

  • 字串-JSON 字串值 (例如，"custom attribute")。

  • 字串清單-字串值的 JSON 陣列 (例如，["item1,item2,item3"])。

除了內建和自訂屬性之外，您還可以在 JSON 檔案中提供常見問題集的使用者內容資訊。您可以提供使用者內容資訊，以根據使用者和群組限制對常見問題集內容的存取。您可以將使用者或群組包含在允許清單或拒絕清單中，但不能同時包含兩者。如果您同時包含使用者或群組，您會收到錯誤。若要篩選使用者前後關聯，您必須在查詢中提供使用者和群組資訊。否則，將返回所有相關常見問題解答。如需詳細資訊，請參閱篩選使用者內容。

此 JSON 範例會將使用者內容新增至常見問題集。

"AccessControlList": [
                {
                    "Name": "group or user name",
                    "Type": "GROUP | USER",
                    "Access": "ALLOW | DENY"
                },
                additional user context
            ]

使用常見問題解答檔

將常見問題集輸入檔案存放在 S3 儲存貯體後，您可以使用主控台或CreateFaq API 將問題和答案放入索引中。如果您想要更新常見問題集，請刪除常見問題集並重新建立。您可以使用DeleteFaq API 刪除常見問題集。

您必須提供可存取包含來源檔案之 S3 儲存貯體的IAM角色。您可以在主控台或RoleArn參數中指定角色。下列是將 FAQ 檔案新增至索引的程式範例。

Python
import boto3

kendra = boto3.client("kendra")

# Provide the index ID
index_id = "index-id"
# Provide the IAM role ARN required to index documents in an S3 bucket
role_arn = "arn:aws:iam::${accountId}:role/${roleName}"

# Provide the S3 bucket path information to the FAQ file
faq_path = {
    "Bucket": "bucket-name",
    "Key": "FreeClinicsUSA.csv"
}

response = kendra.create_faq(
    S3Path =  faq_path,
    Name = "FreeClinicsUSA",
    IndexId = index_id,
    RoleArn = role_arn
)

print(response)
Java
package com.amazonaws.kendra;

import software.amazon.awssdk.services.kendra.KendraClient;
import software.amazon.awssdk.services.kendra.model.CreateFaqRequest;
import software.amazon.awssdk.services.kendra.model.CreateFaqResponse;
import software.amazon.awssdk.services.kendra.model.S3Path;

public class AddFaqExample {
    public static void main(String[] args) {
        KendraClient kendra = KendraClient.builder().build();

        String indexId = "yourIndexId";
        String roleArn = "your role for accessing S3 files";

        CreateFaqRequest createFaqRequest = CreateFaqRequest
            .builder()
            .indexId(indexId)
            .name("FreeClinicsUSA")
            .roleArn(roleArn)
            .s3Path(
                S3Path
                    .builder()
                    .bucket("an-aws-kendra-test-bucket")
                    .key("FreeClinicsUSA.csv")
                    .build())
            .build();

        CreateFaqResponse response = kendra.createFaq(createFaqRequest);

        System.out.println(String.format("The result of creating FAQ: %s", response));

    }
}

英語以外的語言常見問題解答文件

您可以使用支援的語言編製常見問題集的索引。 Amazon Kendra如果您未指定語言，則默認情況下會用英語索引常見問答集。您可以在呼叫CreateFaq作業時指定語言代碼，或在常見問題集中繼資料中包含常見問題集的語言代碼做為欄位。如果常見問題集在中繼資料欄位中指定的中繼資料中沒有語言代碼，常見問題集會使用您呼叫CreateFAQ作業時指定的語言代碼編製索引。若要在主控台中以支援的語言編製常見問題集文件的索引，請前往常見問題集並選取新增常見問題 您可以從下拉菜單中選擇一種語言語言。
