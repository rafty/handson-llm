AWS Service Quotas

對於每個 AWS 服務，您的 AWS 帳戶有預設配額，先前稱為限制。除非另有說明，否則每個配額都是區域特定規定。您可以要求提高某些配額，而其他配額無法提高。

「Service Quotas」是一項AWS服務，可協助您從單一位置管理許多AWS服務的配額。除了查詢配額，您也可以從 Service Quotas 主控台請求提高配額。

AWS Support可能會核准、拒絕或部分核准您的請求。

若要檢視 Service Quotas

您可以使用下列選項檢視服務配額：

  • 開啟文件中的 服務端點和配額 頁面，搜尋服務名稱，然後按一下連結，前往該服務的頁面。若要在不切換頁面的情況下檢視文件中所有AWS服務的服務配額，請改為在 PDF 的「服務端點與配額」頁面中檢視資訊。

  • 開啟 Service Quotas 主控台。在導覽窗格中，選擇AWS服務，然後選擇一個服務。

  • 使用list-service-quotas和 list-aws-default-service-配額AWS CLI命令。

請求提高配額

您可以使用下列其中一個 Service Quotas 請求提高配額。提升不會立即授予。您的增長可能需要幾天的時間才會生效。

  • 開啟 Service Quotas 主控台。在導覽窗格中，選擇 AWS services (AWS 服務)。選取服務、選取配額，然後依照指示請求增加配額。如需詳細資訊，請參閱《Service Quotas 使用者指南》中的請求增加配額。

  • 使用 request-service-quota-increase AWS CLI 命令。

如果 Service Quotas 中尚未提供的服務配額，請使用AWS Support Center Console. 如果 Service Quotas 中有提供服務，我們強烈建議您使用 Service Quotas，而不是建立支援案例。
