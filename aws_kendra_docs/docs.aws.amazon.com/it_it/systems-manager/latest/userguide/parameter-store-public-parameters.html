Utilizzo dei parametri pubblici

Alcuni Servizi AWS pubblicano informazioni sugli artefatti comuni come parametri pubblici AWS Systems Manager. Ad esempio, il servizio Amazon Elastic Compute Cloud (Amazon EC2) pubblica le informazioni su Amazon Machine Images (AMIs) sotto forma di parametri pubblici.

Queste informazioni possono essere chiamate dagli script e dal codice utilizzando le operazioni API DescribeParameters, GetParametersByPath, GetParameter e GetParameters.
