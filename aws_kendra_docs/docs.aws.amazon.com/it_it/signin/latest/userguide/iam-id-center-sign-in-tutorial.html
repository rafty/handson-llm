Accedi al portale di AWS accesso

Un utente di IAM Identity Center è membro diAWS Organizations. Un utente in IAM Identity Center può accedere a più Account AWS applicazioni aziendali accedendo al portale di AWS accesso con un URL di accesso specifico. Per ulteriori informazioni sull'URL di accesso specifico, consultaPortale di accesso AWS.

Prima di accedere a un Account AWS come utente in IAM Identity Center, raccogli le seguenti informazioni necessarie.

  • Nome utente aziendale

  • Password aziendale

  • URL di accesso specifico

Nota

Dopo l'accesso, la sessione del portale di AWS accesso è valida per 8 ore. È necessario effettuare nuovamente l'accesso dopo 8 ore.

Per accedere al portale di AWS accesso

  1. Nella finestra del browser, incolla l'URL di accesso che ti è stato fornito tramite e-mail, ad esempiohttps://your_subdomain.awsapps.com/start. Quindi, premere Invio.

  2. Accedere utilizzando le credenziali aziendali (come nome utente e password).

    Nota

    Se l'amministratore ti ha inviato un'e-mail con una password monouso (OTP) ed è la prima volta che accedi, inserisci quella password. Dopo aver effettuato l'accesso, devi creare una nuova password per gli accessi future.

  3. Se ti viene richiesto un codice di verifica, controlla la tua email. Quindi copia e incolla il codice nella pagina di accesso.

    Nota

    I codici di verifica di solito vengono inviati via e-mail, ma il metodo di consegna può variare. Se non ne hai ricevuto uno nella tua e-mail, contatta l'amministratore per i dettagli sul codice di verifica.

  4. Se l'autenticazione MFA è abilitata per il tuo utente in IAM Identity Center, esegui l'autenticazione utilizzando l'autenticazione.

  5. Dopo l'autenticazione, puoi accedere Account AWS a qualsiasi applicazione visualizzata nel portale.

    1. Per accedere, AWS Management Console seleziona il Account AWS riquadro e quindi seleziona il singolo account da gestire.

      Le autorizzazioni di cui dispone l'utente vengono visualizzate insieme ai collegamenti per aprire la console di gestione o per ottenere le credenziali per l'accesso da riga di comando o programmatico.

      Seleziona il collegamento alla console di gestione.

    2. Per accedere a un'applicazione, scegli l'icona dell'applicazione a cui desideri accedere.

L'accesso come utente a IAM Identity Center fornisce le credenziali per accedere alle risorse per un determinato periodo di tempo, chiamato sessione. Per impostazione predefinita, un utente può accedere a un account Account AWS per un'ora. Il valore massimo che l'amministratore può impostare per una sessione è 12 ore. Al termine della sessione, puoi accedere nuovamente.
