Accedi al AWS Management Console

Quando accedi all'URL AWS Management Console di AWS accesso principale (https://signin.aws.amazon.com) devi scegliere il tuo tipo di utente, utente root o utente IAM.

L'utente root ha accesso illimitato all'account ed è associato alla persona che ha creato ilAccount AWS. L'utente root crea quindi altri tipi di utenti, come utenti IAM eutenti in Centro identità AWS IAM (successore di AWS Single Sign-On), e assegna loro le credenziali di accesso.

Un utente IAM è un'identità all'interno del tuo Account AWS che dispone di autorizzazioni personalizzate specifiche. Quando un utente IAM effettua l'accesso, può utilizzare un URL di accesso che include il proprio alias Account AWS o, ad esempio, https://account_alias_or_id.signin.aws.amazon.com/console/ al posto dell'URL di AWS accesso principale. https://signin.aws.amazon.com/

Se non sei sicuro del tipo di utente che sei, consultaTipi di utenti.
