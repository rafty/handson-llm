Come accedere a AWS

Il modo in cui effettui l'accesso AWS dipende dal tipo di AWS utente che sei. Esistono diversi tipi di AWS utenti. Puoi essere un utente root dell'account, un utente IAM, un utente di IAM, un utente di IAM, un'identità federata o utilizzare AWS Builder ID. Per ulteriori informazioni, consulta Tipi di utenti.

Puoi accedere AWS accedendo con uno dei seguenti metodi:

  • Accedi al AWS Management Consolecome utente root o utente IAM

  • Accedi al portale di AWS accessocome utente nel Centro identità IAM

  • Accedi come identità federata

  • Accedi tramite AWS Command Line Interfacee altri metodi programmatici come API e SDK (kit di sviluppo software)

  • Accedi con AWS Builder ID

Se vuoi crearne uno nuovoAccount AWS, vedi Parte 1: Configurare un nuovo Account AWS nella Guida all'AWSinstallazione.
