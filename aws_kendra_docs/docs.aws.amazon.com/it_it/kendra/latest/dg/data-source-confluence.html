Confluence

Confluence è uno strumento collaborativo di gestione del lavoro progettato per condividere, archiviare e lavorare sulla pianificazione dei progetti, sullo sviluppo del software e sulla gestione dei prodotti. Puoi utilizzarloAmazon Kendra per indicizzare gli spazi Confluence, le pagine (comprese le pagine annidate), i blog, i commenti e gli allegati di pagine e blog indicizzati.

Amazon Kendrasupporta sia Confluence Server che Confluence Cloud.

Nota

Per impostazione predefinita,Amazon Kendra non indicizza gli archivi e gli spazi personali di Confluence. Puoi scegliere di indicizzarli quando crei la fonte dei dati. Se non vuoiAmazon Kendra indicizzare uno spazio, contrassegnalo come privato in Confluence.

Puoi connettertiAmazon Kendra alla tua fonte di dati Confluence utilizzando la Amazon Kendraconsole, l'TemplateConfigurationAPI o l'ConfluenceConfigurationAPI.

Amazon Kendraha due versioni del connettore Confluence. Le funzionalità supportate di ogni versione includono:

Connettore Confluence V1.0//ConfluenceConfigurationAPI

  • Mappature di campi

  • Filtraggio del contesto utente

  • Filtri di inclusione/esclusione

  • (Solo per Confluence Server) Virtual private cloud (VPC, Virtual private cloud (Cloud privato virtuale (VPC)

Connettore Confluence V2.0//TemplateConfigurationAPI

  • Mappature di campi

  • Filtraggio del contesto utente

  • Virtual Private Cloud (VPC) (Cloud privato virtuale (VPC)

  • Sincronizza tutti i documenti/Sincronizza solo i documenti nuovi, modificati o eliminati

  • Modelli di inclusione/esclusione

Nota

Support per il connettore Confluence ConfluenceConfiguration V1.0/API dovrebbe terminare entro giugno 2023. Si consiglia di utilizzare il connettore Confluence TemplateConfiguration V2.0/API.

Per la risoluzione dei problemi del connettore di origine datiAmazon Kendra Confluence, consultaRisoluzione dei problemi delle origini dati.
