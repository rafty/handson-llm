Tipi di documenti

Un indice può contenere più documenti e più tipi di documenti. Un indice può includere anche testo strutturato e non strutturato.

Testo strutturato

  • Domande e risposte frequenti

I documenti in formato di domande e risposte frequenti vengono utilizzati per rispondere a domande come Quanto è alto lo Space Needle? Puoi specificare più domande che restituiscono la stessa risposta. Specifichi le domande e le risposte in un file con valori separati da virgole (CSV) memorizzato in unAmazon S3 bucket.

Se desideri utilizzare un file JSON per la tua serie di domande e risposte frequenti, i file JSON vengono trattati come testo normale e sono un tipo di testo non strutturato.

Per un esempio, vedi Aggiungere domande e risposte direttamente a un indice.

Testo non strutturato

Amazon Kendrasupporta i seguenti tipi di testo non strutturato:

  • File HTML

  • Presentazioni Microsoft PowerPoint (PPT)

  • Documenti MS WORD

  • Documenti in testo normale

  • PDF

  • File con valori separati da virgola (CSV)

  • File di Microsoft Excel (MS EXCEL)

  • File XML

  • File JSON

  • File di documentazione Markdown (MD)

  • File RTF (Rich Text Format)

  • File XSLT (Extensible Stylesheet Language Transformation)

Puoi aggiungere documenti non strutturati all'indice in tre modi:

  • Utilizzo dell'BatchPutDocumentAPI.

  • Come fonte di dati. Per ulteriori informazioni, consulta Aggiungere documenti da un'origine dati.

  • Come dati binari. Per ulteriori informazioni, consulta la sezione Aggiungere documenti con l'API di.
