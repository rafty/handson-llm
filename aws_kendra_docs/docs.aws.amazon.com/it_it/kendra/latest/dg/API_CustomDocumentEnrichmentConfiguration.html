CustomDocumentEnrichmentConfiguration

Fornisce le informazioni sulla configurazione per modificare i metadati e il contenuto del documento durante il processo di importazione del documento.

Per ulteriori informazioni, consulta la sezione Personalizzazione dei metadati del documento durante il processo di importazione.

Indice

InlineConfigurations

Informazioni sulla configurazione per modificare gli attributi del documento o i campi e il contenuto dei metadati durante l'importazione di documenti in Amazon Kendra.

Tipo: matrice di oggetti InlineCustomDocumentEnrichmentConfiguration

Membri dell'array: numero minimo di 0 elementi. Numero massimo di 100 elementi.

Required: No

PostExtractionHookConfiguration

Informazioni sulla configurazione per richiamare una funzione Lambda in AWS Lambda sui documenti strutturati con metadati e testo estratti. Puoi utilizzare una funzione Lambda per applicare una logica avanzata per la creazione, la modifica o l'eliminazione di metadati e contenuti di documenti. Per ulteriori informazioni, consulta la sezione Manipolazione avanzata dei dati.

Tipo: HookConfiguration oggetto

Required: No

PreExtractionHookConfiguration

Informazioni sulla configurazione per richiamare una funzione Lambda in AWS Lambda sui documenti originali o grezzi prima di estrarne i metadati e il testo. Puoi utilizzare una funzione Lambda per applicare una logica avanzata per la creazione, la modifica o l'eliminazione di metadati e contenuti di documenti. Per ulteriori informazioni, consulta la sezione Manipolazione avanzata dei dati.

Tipo: HookConfiguration oggetto

Required: No

RoleArn

Il nome della risorsa Amazon (ARN) di un ruolo con l'autorizzazione all'esecuzione di PreExtractionHookConfiguration e PostExtractionHookConfiguration per modificare i metadati e il contenuto del documento durante il processo di importazione del documento. Per ulteriori informazioni, consulta la sezione Ruoli IAM per Amazon Kendra.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 0. Lunghezza massima di 1284 caratteri.

Modello: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Required: No
