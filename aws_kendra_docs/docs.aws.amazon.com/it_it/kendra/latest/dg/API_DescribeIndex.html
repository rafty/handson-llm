DescribeIndex

Permette di ottenere informazioni su un indice Amazon Kendra esistente.

Sintassi della richiesta

{
   "Id": "string"
}

Parametri della richiesta

Per informazioni sui parametri comuni a tutte le azioni, vedere Parametri comuni.

La richiesta accetta i seguenti dati in formato JSON.

Id

L'identificatore dell'indice sul quale si desidera ottenere informazioni.

Tipo: String

Limitazioni di lunghezza: lunghezza fissa di 36.

Pattern: [a-zA-Z0-9][a-zA-Z0-9-]*

Campo obbligatorio: sì

Elementi di risposta

Se l'operazione riesce, il servizio restituisce una risposta HTTP 200.

I dati seguenti vengono restituiti in formato JSON mediante il servizio.

CapacityUnits

Per gli indici Enterprise Edition, puoi scegliere di utilizzare una capacità aggiuntiva per soddisfare le esigenze della tua applicazione. Contiene le unità di capacità utilizzate per l'indice. Una capacità di archiviazione di una query o di un documento pari a zero indica che l'indice utilizza la capacità predefinita. Per ulteriori informazioni sulla capacità predefinita di un indice e sulla relativa regolazione, consulta Regolazione della capacità.

Tipo: CapacityUnitsConfiguration oggetto

CreatedAt

Il nome della creazione dell'indice.

Tipo: Timestamp

Description

La descrizione dell'indice.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 0. Lunghezza massima di 1000.

Pattern: ^\P{C}*$

DocumentMetadataConfigurations

Informazioni di configurazione per i metadati o i campi del documento. I metadati dei documenti sono campi o attributi associati ai tuoi documenti. Ad esempio, il nome del reparto aziendale associato a ciascun documento.

Tipo: matrice di oggetti DocumentMetadataConfiguration

Membri dell'array: numero minimo di 0 elementi. Numero massimo di 500 elementi.

Edition

L'edizione Amazon Kendra utilizzata per l'indice. Siete voi a decidere l'edizione al momento della creazione dell'indice.

Tipo: String

Valori validi: DEVELOPER_EDITION | ENTERPRISE_EDITION

ErrorMessage

Quando il valore delStatus campo èFAILED, ilErrorMessage campo contiene un messaggio che spiega il motivo.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. La lunghezza massima è 2048 caratteri.

Pattern: ^\P{C}*$

Id

L'identificatore dell'indice.

Tipo: String

Limitazioni di lunghezza: lunghezza fissa di 36.

Pattern: [a-zA-Z0-9][a-zA-Z0-9-]*

IndexStatistics

Fornisce informazioni sul numero di domande e risposte alle domande frequenti e sul numero di documenti di testo indicizzati.

Tipo: IndexStatistics oggetto

Name

Il nome dell'indice.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. Lunghezza massima di 1000.

Pattern: [a-zA-Z0-9][a-zA-Z0-9_-]*

RoleArn

L'ARN (Amazon Resource Name) del ruolo IAM che dà ad Amazon Kendra il nome della risorsa Amazon (ARN) del ruolo IAM che autorizza Amazon Kendra a scrivere nei log Amazon Cloudwatch.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 0. La lunghezza massima è 1284 caratteri.

Pattern: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

ServerSideEncryptionConfiguration

L'identificatore del cliente (CMK) delAWS KMS cliente (CMK) del cliente (CMK) del cliente (CMK) del cliente (CMK) del cliente (CMK) Amazon Kendra non supporta chiavi CMK asimmetriche.

Tipo: ServerSideEncryptionConfiguration oggetto

Status

Stato attuale dell'indice. Quando il valore èACTIVE, l'indice è pronto per l'uso. Se il valore delStatus campo èFAILED, ilErrorMessage campo contiene un messaggio che spiega il motivo.

Tipo: String

Valori validi: CREATING | ACTIVE | DELETING | FAILED | UPDATING | SYSTEM_UPDATING

UpdatedAt

L'Unix al momento dell'ultimo aggiornamento.

Tipo: Timestamp

UserContextPolicy

La politica del contesto utente per l'indice Amazon Kendra.

Tipo: String

Valori validi: ATTRIBUTE_FILTER | USER_TOKEN

UserGroupResolutionConfiguration

Se hai abilitato la configurazione per recuperare i livelli di accesso di gruppi e utenti da un'origine diAWS IAM Identity Center (successor to AWS Single Sign-On) identità.

Tipo: UserGroupResolutionConfiguration oggetto

UserTokenConfigurations

La configurazione del token utente per l'indice Amazon Kendra.

Tipo: matrice di oggetti UserTokenConfiguration

Membri dell'array: numero massimo di 1 elementi.

Errori

Per informazioni sugli errori comuni a tutte le operazioni, consultare Errori comuni.

AccessDeniedException

Non disponi di accesso sufficiente per eseguire questa azione. Assicurati di disporre delle politiche di autorizzazione e degli account utente richiesti e riprova.

Codice di stato HTTP: 400

InternalServerException

Si è verificato un problema con il server interno utilizzato per il servizio Amazon Kendra. Attendi qualche minuto e riprova oppure contatta l'Support per ricevere assistenza.

Codice di stato HTTP: 500

ResourceNotFoundException

La risorsa che desideri utilizzare non esiste. Verifica di aver fornito la risorsa corretta e riprova.

Codice di stato HTTP: 400

ThrottlingException

La richiesta è stata negata a causa del throttling della richiesta. Ridurre il numero di richieste e riprovare.

Codice di stato HTTP: 400

ValidationException

L'input non riesce a soddisfare i vincoli imposti dal servizio Amazon Kendra. Fornire l'input corretto e riprovare.

Codice di stato HTTP: 400
