StartDataSourceSyncJob

Avvia un processo di sincronizzazione per un connettore dell'origine dati. Se un processo di sincronizzazione è già in corso, Amazon Kendra restituisce un'ResourceInUseExceptioneccezione.

Sintassi della richiesta

{
   "Id": "string",
   "IndexId": "string"
}

Parametri della richiesta

Per informazioni sui parametri comuni a tutte le azioni, vedere Parametri comuni.

La richiesta accetta i seguenti dati in formato JSON.

Id

L'identificatore del connettore dell'origine dati da sincronizzare.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. Lunghezza massima di 100.

Pattern: [a-zA-Z0-9][a-zA-Z0-9_-]*

Campo obbligatorio: sì

IndexId

L'identificatore dell'indice utilizzato con il connettore dell'origine dati.

Tipo: String

Limitazioni di lunghezza: lunghezza fissa di 36.

Pattern: [a-zA-Z0-9][a-zA-Z0-9-]*

Campo obbligatorio: sì

Sintassi della risposta

{
   "ExecutionId": "string"
}

Elementi di risposta

Se l'operazione riesce, il servizio restituisce una risposta HTTP 200.

I dati seguenti vengono restituiti in formato JSON mediante il servizio.

ExecutionId

Identifica un particolare processo di sincronizzazione.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. La lunghezza massima è 2048 caratteri.

Errori

Per informazioni sugli errori comuni a tutte le operazioni, consultare Errori comuni.

AccessDeniedException

Non disponi di diritti di accesso sufficienti per eseguire questa azione. Assicurati di disporre delle politiche di autorizzazione e degli account utente richiesti e riprova.

Codice di stato HTTP: 400

ConflictException

Si è verificato un conflitto con la richiesta. Correggi eventuali incongruenze con le tue risorse e riprova.

Codice di stato HTTP: 400

InternalServerException

Si è verificato un problema con il server interno utilizzato per il servizio Amazon Kendra. Attendi qualche minuto e riprova oppure contatta l'Support per ricevere assistenza.

Codice di stato HTTP: 500

ResourceInUseException

La risorsa che desideri utilizzare è attualmente in uso. Verifica di aver fornito la risorsa corretta e riprova.

Codice di stato HTTP: 400

ResourceNotFoundException

La risorsa che desideri utilizzare non esiste. Verifica di aver fornito la risorsa corretta e riprova.

Codice di stato HTTP: 400

ThrottlingException

La richiesta è stata negata a causa del throttling della richiesta. Ridurre il numero di richieste e riprovare.

Codice di stato HTTP: 400

ValidationException

L'input non riesce a soddisfare i vincoli stabiliti dal servizio Amazon Kendra. Fornire l'input corretto e riprovare.

Codice di stato HTTP: 400
