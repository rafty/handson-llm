Guida introduttiva a una fonte diAmazon S3 dati (console)

Puoi usare laAmazon Kendra console per iniziare a usare unAmazon S3 bucket come archivio dati. Quando usi la console, specifichi tutte le informazioni di connessione necessarie per indicizzare il contenuto del bucket. Per ulteriori informazioni, consulta Amazon S3.

Utilizza la procedura seguente per creare un'origine dati bucket S3 di base utilizzando la configurazione predefinita. La procedura presuppone che sia stato creato un indice seguendo i passaggi del passaggio 1 diGuida introduttiva all'utilizzo dellaAmazon Kendra console.

Per creare un'origine dati bucket S3 utilizzando laAmazon Kendra console

  1. AccedereAWS Management Console e aprire laAmazon Kendra console all'indirizzo https://console.aws.amazon.com/kendra/home.

  2. Dall'elenco degli indici scegli l'indice a cui desideri aggiungere l'origine dati.

  3. Scegli Aggiungi origini dati.

  4. Dall'elenco dei connettori per sorgenti dati, scegli Amazon S3.

  5. Nella pagina Definisci attributi, assegna un nome all'origine dei dati e, facoltativamente, una descrizione. Lascia vuoto il campo Tag. Seleziona Next (Successivo) per continuare.

  6. Nel campo Inserisci la posizione dell'origine dati digitare il nome del bucket S3 che contiene i documenti. Puoi inserire direttamente il nome oppure cercare il nome scegliendo Sfoglia. Il bucket deve trovarsi nella stessa regione dell'indice.

  7. Nel IAMruolo scegli Crea un nuovo ruolo e quindi digita un nome per il ruolo. Per ulteriori informazioni, consulta IAMi ruoli per le fonti diAmazon S3 dati.

  8. Nella sezione Imposta la pianificazione dell'esecuzione della sincronizzazione, scegli Esegui su richiesta.

  9. Seleziona Next (Successivo) per continuare.

  10. Nella pagina Revisione e creazione, rivedi i dettagli della tua fonte di dati S3. Se desideri apportare modifiche, scegli il pulsante Edit (Modifica) accanto all'elemento che desideri modificare. Quando sei soddisfatto delle tue scelte, scegli Crea per creare la tua fonte di dati S3.

Dopo aver scelto Crea,Amazon Kendra inizia a creare l'origine dati. può impiegare diversi minuti per creare l'origine dati. Al termine, lo stato dell'origine dati cambia da Creazione a Attivo.

Dopo aver creato l'origine dati, è necessario sincronizzare l'Amazon Kendraindice con l'origine dati. Scegli Sincronizza ora per avviare il processo di sincronizzazione. La sincronizzazione dell'origine dei dati può richiedere da alcuni minuti a diverse ore, a seconda del numero e delle dimensioni dei documenti.
