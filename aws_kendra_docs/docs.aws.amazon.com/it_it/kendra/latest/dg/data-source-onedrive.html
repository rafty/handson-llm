Microsoft OneDrive

Microsoft OneDrive è un servizio di archiviazione basato in Cloud che puoi utilizzare per archiviare, condividere e ospitare i tuoi contenuti. Puoi usarloAmazon Kendra per indicizzare la tua fonte di OneDrive dati.

Puoi connettertiAmazon Kendra alla tua fonte di OneDrive dati utilizzando la Amazon Kendraconsole e l'OneDriveConfigurationAPI.

Amazon Kendraha due versioni del OneDrive connettore. Le funzionalità supportate di ogni versione includono:

OneDrive Connettore Microsoft OneDriveConfigurationv1.0/API

  • Mappature di campi

  • Filtri di inclusione/esclusione

OneDrive Connettore Microsoft TemplateConfigurationv2.0/API

  • Filtraggio del contesto utente

  • Filtri di inclusione/esclusione

  • Funzionalità di Identity Crawler

  • Sincronizza tutti i documenti/Sincronizza solo i documenti nuovi, modificati ed eliminati

Nota

Support per il OneDrive connettore OneDriveConfiguration v1.0/API dovrebbe terminare entro giugno 2023. Si consiglia di utilizzare il OneDrive connettore TemplateConfiguration v2.0/API.

Per la risoluzione dei problemi relativi al connettore della sorgenteAmazon Kendra OneDrive dati, consultaRisoluzione dei problemi delle origini dati.
