Origini dati

Un'origine dati è un archivio o una posizione di dati cheAmazon Kendra si connette e indicizza i documenti o i contenuti. Ad esempio, puoi configurare la connessioneAmazon Kendra a Microsoft SharePoint per scansionare e indicizzare i documenti archiviati in questa fonte. Puoi anche indicizzare le pagine Web fornendo gli URLAmazon Kendra per la scansione. È possibile sincronizzare automaticamente un'origine dati con unAmazon Kendra indice in modo che anche i documenti aggiunti, aggiornati o eliminati nell'origine dati vengano aggiunti, aggiornati o eliminati nell'indice.
