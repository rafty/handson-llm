CreateFaq

Crea una serie di domande frequenti (domande frequenti) utilizzando il file domande frequenti memorizzati in un bucket Amazon S3.

L'aggiunta delle domande frequenti a un indice è un'operazione asincrona.

Per un esempio di aggiunta di domande frequenti a un indice utilizzando gli SDK Python e Java, consulta Utilizzo del file delle domande frequenti.

Sintassi della richiesta

{
   "ClientToken": "string",
   "Description": "string",
   "FileFormat": "string",
   "IndexId": "string",
   "LanguageCode": "string",
   "Name": "string",
   "RoleArn": "string",
   "S3Path": { 
      "Bucket": "string",
      "Key": "string"
   },
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

Parametri della richiesta

Per informazioni sui parametri comuni a tutte le azioni, vedere Parametri comuni.

La richiesta accetta i seguenti dati in formato JSON.

ClientToken

Un token fornito per identificare la richiesta di creazione di una FAQ. Più chiamate all'CreateFaqRequestAPI con lo stesso token client creeranno una sola domanda frequente.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. Lunghezza massima di 100.

Required: No

Description

Una descrizione della domanda frequente (FAQ).

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 0. Lunghezza massima di 1000.

Modello: ^\P{C}*$

Required: No

FileFormat

Il formato del file di input delle domande frequenti. È possibile scegliere tra un formato CSV di base, un formato CSV che include attributi doganali in un'intestazione e un formato JSON che include attributi personalizzati.

Il formato deve corrispondere al formato del file memorizzato nel bucket S3 identificato nelS3Path parametro.

Per ulteriori informazioni, consulta Aggiungere domande e risposte.

Tipo: String

Valori validi: CSV | CSV_WITH_HEADER | JSON

Required: No

IndexId

L'identificatore dell'indice per le domande frequenti.

Tipo: String

Vincoli di lunghezza: lunghezza massima di 36.

Pattern: [a-zA-Z0-9][a-zA-Z0-9-]*

Campo obbligatorio: sì

LanguageCode

Il codice di una lingua. Ciò consente di supportare una lingua per il documento delle domande frequenti. L'inglese è supportato per impostazione predefinita. Per ulteriori informazioni sulle lingue supportate, inclusi i relativi codici, consulta Aggiungere documenti in lingue diverse dall'inglese.

Tipo: String

Vincoli di lunghezza: lunghezza massima 2. Lunghezza massima 10.

Modello: [a-zA-Z-]*

Required: No

Name

Un nome per le domande frequenti.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. Lunghezza massima di 100.

Pattern: [a-zA-Z0-9][a-zA-Z0-9_-]*

Campo obbligatorio: sì

RoleArn

L'Amazon Resource Name (ARN) di un ruolo IAM con l'autorizzazione ad accedere al bucket S3 che contiene le domande frequenti. Per ulteriori informazioni, consulta la sezione Ruoli di accesso IAM per Amazon Kendra.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 0. Lunghezza massima è 1284 caratteri.

Pattern: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Campo obbligatorio: sì

S3Path

Percorso del file delle domande frequenti in S3.

Tipo: S3Path oggetto

Campo obbligatorio: sì

Tags

Un elenco delle coppie chiave-valore che identificano le domande frequenti. È possibile utilizzare i tag per identificare e organizzare le risorse e controllare l'accesso alle risorse.

Tipo: matrice di oggetti Tag

Membri dell'array: numero minimo di 0 elementi. Numero massimo 200 elementi.

Required: No

Sintassi della risposta

{
   "Id": "string"
}

Elementi di risposta

Se l'operazione riesce, il servizio restituisce una risposta HTTP 200.

I dati seguenti vengono restituiti in formato JSON mediante il servizio.

Id

L'identificatore delle domande frequenti.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. Lunghezza massima di 100.

Pattern: [a-zA-Z0-9][a-zA-Z0-9_-]*

Errori

Per informazioni sugli errori comuni a tutte le operazioni, consultare Errori comuni.

AccessDeniedException

Non disponi di accesso sufficienti per eseguire questa azione massima per eseguire questa azione. Assicurati di disporre delle politiche di autorizzazione e degli account utente richiesti e riprova.

Codice di stato HTTP: 400

ConflictException

Si è verificato un conflitto con la richiesta. Correggi eventuali incongruenze con le tue risorse e riprova.

Codice di stato HTTP: 400

InternalServerException

Si è verificato un problema con il server interno utilizzato per il servizio Amazon Kendra. Attendi qualche minuto e riprova oppure contatta l'Support per ricevere assistenza.

Codice di stato HTTP: 500

ResourceNotFoundException

La risorsa che desideri utilizzare non esiste. Verifica di aver fornito la risorsa corretta e riprova.

Codice di stato HTTP: 400

ServiceQuotaExceededException

Hai superato i limiti stabiliti per il tuo servizio Amazon Kendra. Consulta la sezione Quote per ulteriori informazioni o contatta l'Support per richiedere informazioni su un aumento dei limiti.

Codice di stato HTTP: 400

ThrottlingException

La richiesta è stata negata a causa del throttling della richiesta. Ridurre il numero massimo e riprovare.

Codice di stato HTTP: 400

ValidationException

L'input non riesce a soddisfare i vincoli stabiliti dal servizio Amazon Kendra. Fornire i dati corretti e riprovare.

Codice di stato HTTP: 400
