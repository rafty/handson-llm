S3Path

Informazioni necessarie per trovare un file specifico in un bucket Amazon S3.

Indice

Bucket

Il nome del bucket S3 che contiene il file.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 3. La lunghezza massima è 63 caratteri.

Pattern: [a-z0-9][\.\-a-z0-9]{1,61}[a-z0-9]

Campo obbligatorio: sì

Key

Nome del file.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. La lunghezza massima è 1024 caratteri.

Campo obbligatorio: sì
