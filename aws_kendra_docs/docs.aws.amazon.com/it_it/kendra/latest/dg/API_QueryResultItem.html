QueryResultItem

Un singolo risultato dell'interrogazione.

Il risultato di una query contiene informazioni su un documento restituito dall'interrogazione. Ciò include la posizione originale del documento, un elenco di attributi assegnati al documento e il testo pertinente del documento che soddisfa la richiesta.

Indice

AdditionalAttributes

Uno o più attributi aggiuntivi associati ai risultati della query.

Tipo: matrice di oggetti AdditionalResultAttribute

Required: No

DocumentAttributes

Matrice di attributi del documento assegnati a un documento nei risultati della ricerca. Ad esempio, l'autore del documento (_author) o l'URI di origine (_source_uri) del documento.

Tipo: matrice di oggetti DocumentAttribute

Required: No

DocumentExcerpt

Un estratto del testo contenuto nel documento. Contiene informazioni su come evidenziare i termini pertinenti nell'estratto.

Tipo: TextWithHighlights oggetto

Required: No

DocumentId

L'identificatore del documento.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. La lunghezza massima è 2048 caratteri.

Required: No

DocumentTitle

Titolo del documento. Contiene il testo del titolo e le informazioni per evidenziare i termini pertinenti nel titolo.

Tipo: TextWithHighlights oggetto

Required: No

DocumentURI

L'URI della directory originale del documento.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. La lunghezza massima è 2048 caratteri.

Modello: ^(https?|ftp|file):\/\/([^\s]*)

Required: No

FeedbackToken

Un token che identifica un particolare risultato di una particolare query. Utilizza questo token per fornire un feedback immediato sul risultato. Per ulteriori informazioni, consulta Invio di feedback.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. La lunghezza massima è 2048 caratteri.

Modello: ^\P{C}*.\P{C}*$

Required: No

Format

Se ilType documento f all'interno della risposta èANSWER, allora si tratta di unaTABLE risposta o di unaTEXT risposta. Se si tratta di una risposta alla tabella, viene restituito un estratto della tabellaTableExcerpt. Se si tratta di una risposta testuale, viene restituito un estratto di testoDocumentExcerpt.

Tipo: String

Valori validi: TABLE | TEXT

Required: No

Id

L'identificatore dei risultati della query.

Tipo: String

Limitazioni di lunghezza: lunghezza minima di 1. Lunghezza massima di 73 caratteri.

Required: No

ScoreAttributes

Indica la sicurezza di Amazon Kendra che un risultato corrisponda alla query che hai fornito. Ogni risultato viene inserito in un contenitore che indica la confidenzaVERY_HIGH,HIGH,MEDIUM eLOW. Puoi utilizzare il punteggio per determinare se una risposta soddisfa la fiducia necessaria per la tua candidatura.

Il campo è impostato soloLOW quando ilType campo è impostato suDOCUMENT e Amazon Kendra non è sicuro che il risultato corrisponda alla query.

Tipo: ScoreAttributes oggetto

Required: No

TableExcerpt

Un estratto da una tabella all'interno di un documento.

Tipo: TableExcerpt oggetto

Required: No

Type

Il tipo di documento all'interno della risposta. Ad esempio, una risposta potrebbe includere una domanda-risposta pertinente alla domanda.

Tipo: String

Valori validi: DOCUMENT | QUESTION_ANSWER | ANSWER

Required: No
