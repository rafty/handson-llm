Strumento di ortografia di query

Amazon KendraIl correttore ortografico suggerisce correzioni ortografiche per un'interrogazione. Questo può aiutarti a ridurre al minimo le occorrenze di zero risultati di ricerca e a restituire risultati pertinenti. I tuoi utenti potrebbero ricevere zero risultati di ricerca da query con errori di ortografia senza risultati corrispondenti o senza documenti restituiti. In alternativa, i tuoi utenti potrebbero ricevere risultati di ricerca non pertinenti a causa di interrogazioni con errori di ortografia.

Il correttore ortografico è progettato per suggerire correzioni per le parole con errori ortografici in base alle parole che appaiono nei documenti indicizzati e alla corrispondenza tra una parola corretta e una parola errata. Ad esempio, se la parola «rendiconti» compare nei documenti indicizzati, potrebbe corrispondere perfettamente alla parola errata «rendiconti» nella query «rendiconti finanziari di fine anno».

Il controllo ortografico restituisce le parole desiderate o corrette che sostituiscono le parole con errori di ortografia nel testo della query originale. Ad esempio, 'depoying kendre search' potrebbe restituire 'deploying Kendra search'. Puoi anche utilizzare le posizioni di offset fornite nell'API per evidenziare o mettere in corsivo le parole corrette restituite in una query nell'applicazione front-end. Nella console, le parole corrette sono evidenziate o in corsivo per impostazione predefinita. Ad esempio, «implementazione di Kendra search».

Per quanto riguarda i termini aziendali o specializzati che compaiono nei documenti indicizzati, il Controllo ortografico non li interpreta erroneamente come errori ortografici nella query. Ad esempio, «amazon macie» non viene corretto in «amazon macie».

Per le parole sillabate, ad esempio «fine anno», il correttore ortografico le considera come singole parole per suggerire correzioni per queste parole. Ad esempio, la correzione suggerita per «fine anno» potrebbe essere «fine anno».

Per i tipi di risposta alleQUESTION_ANSWER interrogazioniDOCUMENT e ai tipi di risposta, il correttore ortografico suggerisce le correzioni alle parole con errori ortografici in base alle parole presenti nel corpo del documento. Il corpo del documento è più affidabile del titolo per suggerire correzioni che corrispondano perfettamente alle parole errate. Per i tipi di risposta alleANSWER interrogazioni, il correttore ortografico suggerisce le correzioni in base alle parole contenute nel documento di domanda e risposta predefinito dell'indice.

È possibile attivare il correttore ortografico utilizzando l'SpellCorrectionConfigurationoggetto. Hai impostatoIncludeQuerySpellCheckSuggestions perTRUE. Il controllo ortografico è attivato per impostazione predefinita nella console. È integrato nella console per impostazione predefinita.

Il correttore ortografico può anche suggerire correzioni ortografiche per le interrogazioni in più lingue, non solo in inglese. Per un elenco delle lingue supportate per il controllo ortografico, consulta le lingueAmazon Kendra supportate.

Utilizzo del correttore ortografico dell'interrogazione con limiti predefiniti

Il correttore ortografico è progettato con determinati valori o limiti predefiniti. Di seguito è riportato un elenco dei limiti attuali che si applicano quando si attivano i suggerimenti per la correzione ortografica.

  • Le correzioni ortografiche suggerite non possono essere restituite per parole di lunghezza inferiore a tre caratteri o superiore a 30 caratteri. Per consentire l'uso di più di 30 caratteri o meno di tre caratteri, contatta l'Support.

  • Le correzioni ortografiche suggerite non possono limitare i suggerimenti in base al controllo dell'accesso dell'utente o all'elenco di controllo degli accessi per il filtraggio del contesto utente. Le correzioni ortografiche si basano su tutte le parole nei documenti indicizzati, indipendentemente dal fatto che siano limitate a determinati utenti o meno. Se vuoi evitare che alcune parole compaiano nelle correzioni ortografiche suggerite per le interrogazioni, allora non attivarlaSpellCorrectionConfiguration.

  • Le correzioni ortografiche suggerite non possono essere restituite per le parole che includono numeri. Ad esempio, 'how 2 not br8k ubun2'.

  • Le correzioni ortografiche suggerite non possono utilizzare parole che non compaiono nei documenti indicizzati.

  • Le correzioni ortografiche suggerite non possono utilizzare parole con una frequenza inferiore allo 0,01% nei documenti indicizzati. Per modificare la soglia dello 0,01%, contatta l'Support.
