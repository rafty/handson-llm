Se si utilizza l'autenticazione di base, il segreto deve contenere una struttura JSON con le seguenti chiavi: 
{
"username": "Confluence account user name",
"password": "Confluence API token"
}
Se si utilizza l'autenticazione OAuth 2.0, il segreto deve contenere una struttura JSON con le seguenti chiavi: 
{
"confluenceAppKey": "app key for your Confluence account",
"confluenceAppSecret": "app secret from your Confluence token",
"confluenceAccessToken": "access token created in Confluence",
"confluenceRefreshToken": "refresh token created in Confluence"
}
Se si utilizza l'autenticazione del token di accesso personale, il segreto deve contenere una struttura JSON con le seguenti chiavi: 
{
"patToken": "Confluence token"
}
