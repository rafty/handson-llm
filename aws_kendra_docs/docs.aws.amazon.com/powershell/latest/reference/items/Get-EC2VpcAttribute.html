Get-EC2VpcAttribute Cmdlet

Amazon Elastic Compute Cloud (EC2)
Available in AWS.Tools.EC2, AWSPowerShell.NetCore and AWSPowerShell

Synopsis

Calls the Amazon Elastic Compute Cloud (EC2) DescribeVpcAttribute API operation.

Syntax

Get-EC2VpcAttribute
-VpcId <String>
-Attribute <VpcAttributeName>
-Select <String>
-PassThru <SwitchParameter>
-ClientConfig <AmazonEC2Config>

Description

Describes the specified attribute of the specified VPC. You can specify only one attribute at a time.

Outputs

Amazon.EC2.Model.DescribeVpcAttributeResponse
This cmdlet returns an Amazon.EC2.Model.DescribeVpcAttributeResponse object containing multiple properties. The object can also be referenced from properties attached to the cmdlet entry in the $AWSHistory stack.
