¿Qué es Amazon Lex V2?

Amazon Lex V2 es un servicio de AWS para crear interfaces conversacionales para aplicaciones que utilizan voz y texto. Amazon Lex V2 ofrece la amplia funcionalidad y flexibilidad de la comprensión del lenguaje natural (NLU) y el reconocimiento automático de voz (ASR) para que pueda crear experiencias de usuario altamente atractivas con interacciones conversacionales realistas y crear nuevas categorías de productos.

Amazon Lex V2 permite a cualquier desarrollador crear bots conversacionales rápidamente. Con Amazon Lex V2, no es necesario tener experiencia en aprendizaje profundo: para crear un bot, debe especificar el flujo de conversación básico en la consola de Amazon Lex V2. Amazon Lex V2 administra el cuadro de diálogo y ajusta de forma dinámica las respuestas de la conversación. Mediante la consola puede crear, probar y publicar su chatbot de texto o voz. A continuación, puede añadir las interfaces de conversación a los bots en dispositivos móviles, aplicaciones web y plataformas de chat (por ejemplo, Facebook Messenger).

Amazon Lex V2 ofrece integración y puede integrarse con muchos otros servicios de la plataforma AWS, como Amazon Connect, Amazon Comprehend y Amazon Kendra.AWS Lambda La integración con Lambda proporciona a los bots acceso a conectores empresariales sin servidor prediseñados para vincularlos a los datos de aplicaciones SaaS, como Salesforce.

En el caso de los bots creados después del 17 de agosto de 2022, puedes usar la ramificación condicional para controlar el flujo de conversación con tu bot. Con la ramificación condicional puedes crear conversaciones complejas sin necesidad de escribir código Lambda.

Amazon Lex V2 proporciona los siguientes beneficios:

  • Simplicidad: Amazon Lex V2 le guiará a través del uso de la consola para crear su propio bot en cuestión de minutos. Si proporciona algunas frases de ejemplo, Amazon Lex V2 crea un modelo completo de lenguaje natural mediante el cual el bot puede interactuar mediante voz y texto para hacer preguntas, obtener respuestas y completar tareas sofisticadas.

     

  • Tecnologías de aprendizaje profundo democratizadas: Amazon Lex V2 proporciona tecnologías ASR y NLU para crear un sistema de comprensión del lenguaje del habla (SLU). A través de SLU, Amazon Lex V2 utiliza la entrada de voz y texto en lenguaje natural, comprende la intención detrás de la entrada y cumple con la intención del usuario al invocar la función empresarial adecuada.

     

    El reconocimiento de voz y la comprensión del lenguaje natural son algunos de los problemas más difíciles de resolver en la informática, ya que requieren que se entrenen sofisticados algoritmos de aprendizaje profundo en enormes cantidades de datos e infraestructura. Amazon Lex V2 pone las tecnologías de aprendizaje profundo al alcance de todos los desarrolladores. Los bots de Amazon Lex V2 convierten la voz entrante en texto y comprenden la intención del usuario para generar una respuesta inteligente, de modo que pueda centrarse en crear sus bots con un valor añadido para sus clientes y definir categorías de productos completamente nuevas que son posibles gracias a las interfaces conversacionales.

     

  • Implementación y escalado sin problemas: con Amazon Lex V2, puede crear, probar e implementar sus bots directamente desde la consola Amazon Lex V2. Amazon Lex V2 le permite publicar sus bots de voz o texto para usarlos en dispositivos móviles, aplicaciones web y servicios de chat (por ejemplo, Facebook Messenger). Amazon Lex V2 se escala automáticamente. No tiene que preocuparse por aprovisionar el hardware y administrar la infraestructura para potenciar su experiencia con los bots.

     

  • Integración integrada con la plataforma AWS: Amazon Lex V2 funciona de forma nativa con otros servicios de AWS,AWS Lambda como Amazon CloudWatch. Puede aprovechar la eficacia de la plataforma de AWS para seguridad, monitorización, autenticación de usuarios, lógica de negocio, almacenamiento y desarrollo de aplicaciones móviles.

     

  • Rentabilidad: con Amazon Lex V2, no hay costes iniciales ni comisiones mínimas. Solo se le cobrará por las solicitudes de texto o voz que se realicen. El pay-as-you-go precio y el bajo costo por solicitud hacen que el servicio sea una forma rentable de crear interfaces conversacionales. Con la capa gratuita de Amazon Lex V2, puede probar Amazon Lex V2 fácilmente sin ninguna inversión inicial.
