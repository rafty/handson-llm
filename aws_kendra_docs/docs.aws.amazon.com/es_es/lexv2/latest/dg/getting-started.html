Introducción a Amazon Lex V2

Amazon Lex V2 proporciona operaciones de API que puede integrar con sus aplicaciones existentes. Para obtener una lista de operaciones admitidas, consulte la Referencia de la API. Puede utilizar cualquiera de las siguientes opciones:

  • AWS SDK: al utilizar los SDK, sus solicitudes a Amazon Lex V2 se firman y autentican automáticamente con las credenciales que usted proporciona. Le recomendamos que utilice un SDK para crear la aplicación.

  • AWS CLI— Puede utilizar laAWS CLI para obtener acceso a cualquier función de Amazon Lex V2 sin tener que escribir ningún código.

  • Consola de AWS: la consola es la forma más sencilla de comenzar a probar y usar Amazon Lex V2

Si es la primera vez que utiliza Amazon Lex V2, le recomendamos que leaCómo funciona primero.
