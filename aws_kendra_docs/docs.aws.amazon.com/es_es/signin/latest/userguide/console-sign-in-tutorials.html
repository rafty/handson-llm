Inicia sesión en AWS Management Console

Al iniciar sesión en la URL AWS Management Console de inicio de AWS sesión principal (https://signin.aws.amazon.com), debe elegir su tipo de usuario, usuario raíz o usuario de IAM.

El usuario root tiene acceso a la cuenta sin restricciones y está asociado a la persona que creó laCuenta de AWS. A continuación, el usuario raíz crea otros tipos de usuarios, como los usuarios de IAM yusuarios en AWS IAM Identity Center (sucesor de AWS Single Sign-On), y les asigna credenciales de acceso.

Un usuario de IAM es una identidad de su Cuenta de AWS que dispone de permisos personalizados específicos. Cuando un usuario de IAM inicia sesión, puede utilizar una URL de inicio de sesión que incluya su alias Cuenta de AWS o su alias, por ejemplo, en https://account_alias_or_id.signin.aws.amazon.com/console/ lugar de la URL de inicio de AWS sesión principal. https://signin.aws.amazon.com/

Si no estás seguro del tipo de usuario que eres, consultaTipos de usuarios.
