Inicie sesión en el portal de AWS acceso

Un usuario de IAM Identity Center es miembro deAWS Organizations. Un usuario de IAM Identity Center puede acceder a múltiples Cuentas de AWS aplicaciones empresariales iniciando sesión en el portal de AWS acceso con una URL de inicio de sesión específica. Para obtener más información acerca de la URL de inicio sesión específica, consulteAWSportal de acceso.

Antes de iniciar sesión en una cuenta de Cuenta de AWS como usuario en el Centro de Identidades IAM, recopile la siguiente información necesaria.

  • Nombre de usuario corporativo

  • Contraseña corporativa

  • URL de inicio sesión específica

nota

Tras iniciar sesión, la sesión del portal de AWS acceso será válida durante 8 horas. Deberá iniciar sesión de nuevo después de 8 horas.

Para iniciar sesión en el portal de AWS acceso

  1. En la ventana del navegador, pegue la URL de inicio de sesión que le han proporcionado por correo electrónico, por ejemplohttps://your_subdomain.awsapps.com/start. Luego pulse Intro.

  2. Inicie sesión con las credenciales de la empresa (como un nombre de usuario y una contraseña).

    nota

    Si el administrador te envió una contraseña de un solo uso (OTP) por correo electrónico y es la primera vez que inicias sesión, introduce esa contraseña. Después de iniciar sesión, debes crear una contraseña nueva para iniciar sesión en el future.

  3. Si se te pide un código de verificación, comprueba tu correo electrónico. A continuación, copie y pegue el código en la página de inicio de sesión.

    nota

    Los códigos de verificación suelen enviarse por correo electrónico, pero el método de entrega puede variar. Si no has recibido ninguno en tu correo electrónico, ponte en contacto con tu administrador para obtener más información sobre tu código de verificación.

  4. Si la MFA está habilitada para su usuario en IAM Identity Center, deberá autenticarse mediante ella.

  5. Tras la autenticación, puede acceder a cualquier Cuenta de AWS aplicación que aparezca en el portal.

    1. Para iniciar sesión, AWS Management Console seleccione el Cuenta de AWS icono y, a continuación, seleccione la cuenta individual que desee administrar.

      Los permisos de los que dispone el usuario se muestran junto con enlaces para abrir la consola de administración o para obtener credenciales de acceso mediante línea de comandos o mediante programación.

      Seleccione el enlace de la consola de administración.

    2. Para iniciar sesión en una aplicación, elija el icono de la aplicación a la que desee acceder.

Al iniciar sesión como usuario en IAM Identity Center, obtendrá credenciales para acceder a los recursos durante un período de tiempo determinado, denominado sesión. De forma predeterminada, un usuario puede iniciar sesión Cuenta de AWS durante una hora. El valor máximo que el administrador puede establecer para una sesión es de 12 horas. Una vez finalizada la sesión, podrás volver a iniciar sesión.
