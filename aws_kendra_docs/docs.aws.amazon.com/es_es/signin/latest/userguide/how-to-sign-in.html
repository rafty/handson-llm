Cómo iniciar sesión en AWS

La forma de iniciar sesión AWS depende del tipo de AWS usuario que sea. Hay diferentes tipos de AWS usuarios. Solo puede ser un usuario raíz de la cuenta, un usuario de IAM, un usuario en el Centro de Identidades IAM, una identidad federada o usar AWS el ID de Builder. Para obtener más información, consulte Tipos de usuarios.

Puede acceder AWS iniciando sesión con cualquiera de los siguientes métodos:

  • Inicia sesión en AWS Management Consolecomo usuario raíz o usuario de IAM

  • Inicie sesión en el portal de AWS accesocomo usuario en IAM Identity Center

  • Iniciar sesión como identidad federada

  • Inicia sesión a través del AWS Command Line Interfacey otros métodos programáticos como la API y el SDK (kit de desarrollo de software)

  • Iniciar sesión con AWS Builder ID

Si desea crear uno nuevoCuenta de AWS, consulte la parte 1: Configurar uno nuevo Cuenta de AWS en la Guía AWS de configuración.
