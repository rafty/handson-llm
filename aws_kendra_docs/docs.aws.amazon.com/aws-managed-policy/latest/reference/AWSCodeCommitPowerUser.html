AWSCodeCommitPowerUser

AWSCodeCommitPowerUser is an AWS managed policy that: Provides full access to AWS CodeCommit repositories, but does not allow repository deletion.

Using this policy

You can attach AWSCodeCommitPowerUser to your users, groups, and roles.

Policy details

  • Type: AWS managed policy

  • Creation time: July 09, 2015, 17:06 UTC

  • Edited time: July 30, 2020, 23:12 UTC

  • ARN: arn:aws:iam::aws:policy/AWSCodeCommitPowerUser

Policy version

Policy version: v14 (default)

The policy's default version is the version that defines the permissions for the policy. When a user or role with the policy makes a request to access an AWS resource, AWS checks the default version of the policy to determine whether to allow the request.
