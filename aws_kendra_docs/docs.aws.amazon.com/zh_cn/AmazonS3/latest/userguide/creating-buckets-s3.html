创建、配置和使用 Amazon S3 Bucket

要将数据存储在 Amazon S3 中，您需要使用称为桶和对象的资源。桶是对象的容器。对象指的是一个文件和描述该文件的任何元数据。

要将对象存储在 Amazon S3 中，您需要创建桶，然后将该对象上载到桶。当对象位于桶时，您可以将其打开、下载并移动它。当您不再需要对象或桶时，可以清理您的资源。

注意

使用 Amazon S3，您只需按实际用量付费。有关 Amazon S3 功能和定价的更多信息，请参阅 Amazon S3。如果您是 Amazon S3 的新客户，则可以免费开始使用 Amazon S3。有关更多信息，请参阅 AWS 免费套餐。

本节中的主题概述了在 Amazon S3 中使用桶。它们包括有关命名、创建、访问和删除桶的信息。有关查看或列出桶中对象的更多信息，请参阅 组织、列出和处理对象。
