如何登录 AWS

您登录的方式AWS取决于您是哪种类型的AWS用户。有不同类型的AWS用户。您可以是账户根用户、IAM 用户、IAM Identity Center 中的用户、联合身份身份或使用 AWS Builder ID。有关更多信息，请参阅用户类型：

您可以使用以下任一方法登录进行访问AWS：

  • 登录到 AWS Management Console以根用户或 IAM 用户身份使用

  • 登录AWS访问门户作为 IAM Identer 中的用户

  • 以联邦身份登录

  • 通过登录 AWS Command Line Interface以及其他编程方法，例如 API 和 SDK（软件开发套件）

  • 使用AWS生成器 ID 登录

如果要创建新的AWS 账户，请参阅AWS安装指南AWS 账户中的第 1 部分：设置新的。
