登录到 AWS Management Console

当您AWS Management Console从主AWS登录网址 (https://signin.aws.amazon.com) 登录时，必须选择您的用户类型，即 root 用户或 IAM 用户。

root 用户拥有不受限制的帐户访问权限，并且与创建者AWS 账户相关联。然后，根用户创建其他类型的用户，例如 IAM 用户和AWS IAM Identity Center（AWS Single Sign-On 的后继者）中的用户，并为他们分配访问证书。

IAM 用户是您的中AWS 账户具有特定自定义权限的实体。当 IAM 用户登录时，他们可以使用包含其AWS 账户或别名的登录 URL，例如 https://account_alias_or_id.signin.aws.amazon.com/console/代替主AWS登录 URL https://signin.aws.amazon.com/。

如果您不确定自己是哪种类型的用户用户类型。
