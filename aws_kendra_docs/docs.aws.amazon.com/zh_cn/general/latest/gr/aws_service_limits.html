AWS 服务配额

您的 AWS 账户对于每项 AWS 服务都具有默认配额（以前称为限制）。除非另有说明，否则，每个配额是区域特定的。您可以请求增加某些配额，但其他一些配额无法增加。

Service Quotas 是一项 AWS 服务，可帮助您从一个位置管理多个 AWS 服务的配额。除了查找配额值，您也可以从 Service Quotas 控制台请求提高配额。

AWS Support 可能会批准、拒绝或部分批准您的请求。

查看服务配额

您可以使用以下选项查看服务配额：

  • 在文档中打开 服务终端节点和配额页面，搜索服务名称，然后单击链接以转到该服务的页面。要在不切换页面的情况下查看文档中所有 AWS 服务的服务配额，请改为查看 PDF 中服务终端节点和配额页面的信息。

  • 打开 Service Quotas 控制台。在导航窗格中，选择 AWS 服务，然后选择一项服务。

  • 使用list-service-quotas和 list-aws-default-service-quotaAWS CLI s 命令。

请求提高配额

通过以下任意一个选项，您可使用服务限额请求增加限额。不会立即批准提高配额。增加限额可能需要几天才能生效。

  • 打开 Service Quotas 控制台。在导航窗格中，选择 AWS services (亚马逊云科技服务)。选择服务，选择配额，然后按照说明请求提高配额。有关更多信息，请参阅《服务限额用户指南》中的请求增加限额。

  • 使用 request-service-quota-increaseAWS CLI 命令。

如果某项服务在服务限额中不可用，请通过 AWS Support Center Console 创建提高服务限制工单。如果该服务在服务限额中可用，我们强烈建议您使用服务限额而非创建支持工单。
