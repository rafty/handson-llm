需要根用户凭证的任务

重要

以下 AWS Identity and Access Management (IAM) 操作将于 2023 年 7 月结束标准支持：aws-portal:ModifyAccount和aws-portal:ViewAccount。请参阅使用精细AWS Billing操作将这些操作替换为精细操作，以便您可以访问AWS成本管理和AWS Billing账户控制台。AWS

如果您已在 AWS 2023 3 3 3 3 3 年 6 日之前创建或AWS Organizations管理账户，则精细操作将从 2023 3 年 6 日之前创建。我们建议您添加精细操作，但不要删除带有 aws-portal 或 purchase-orders 前缀的现有权限。

如果您已在 AWS 2023 3 3 3 3 3 年 6 日当天或之后创建或AWS Organizations管理账户，则精细操作将立即生效。

我们建议您在中配置管理用户AWS IAM Identity Center (successor to AWS Single Sign-On)以执行日常任务和访问AWS资源。不过，您只能在以账户的根用户身份登录时才能执行下列任务。

任务

  • 更改您的账户设置。这包括账户名称、电子邮件地址、根用户密码和根用户访问密钥。其他账户设置，例如联系人信息、付款货币偏好等AWS 区域，不需要根用户凭证。

  • 恢复 IAM 用户权限。如果唯一的 IAM 管理员意外撤消了自己的权限，您可以使用根用户身份登录来编辑策略并还原这些权限。

  • 激活 IAM 对账单和成本管理控制台的访问权限。

  • 查看特定税务发票。拥有 aws-portal: ViewBilling 权限的 IAM 用户可以查看和下载来自AWS欧洲的增值税发票，但不能查看和下载来自 AWS Inc.或亚马逊互联网服务私人有限公司 (AISPL) 的增值税发票。

  • 关闭 AWS 账户。

  • 已在预留实例 Marketplace 中注册为卖家。

  • 配置 Amazon S3 存储桶以启用 MFA（多重身份验证）。

  • 编辑或删除一个包含无效 VPC ID 或 VPC 终端节点 ID 的Simple Storage Service (Amazon S3) 存储桶策略。

  • 注册 AWS GovCloud (US)。

  • 从中请求AWS GovCloud (US)账户根用户访问密钥AWS Support。
