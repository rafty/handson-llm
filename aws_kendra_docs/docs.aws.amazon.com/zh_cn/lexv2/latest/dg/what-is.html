Amazon Lex V2 是什么？

Amazon Lex V2 是一项 AWS 服务，用于使用语音和文本为应用程序构建对话界面。Amazon Lex V2 提供自然语言理解 (NLU) 和自动语音识别 (ASR) 的深层功能和灵活性，因此您可以通过逼真的对话互动打造极具吸引力的用户体验，并创建新的产品类别。

Amazon Lex V2 使任何开发人员都能快速构建对话机器人。使用 Amazon Lex V2，无需深度学习专业知识，要创建机器人，您需要在 Amazon Lex V2 控制台中指定基本对话流程。Amazon Lex V2 管理对话框并动态调整对话中的响应。借助此控制台，您可构建、测试和发布您的文本或语音聊天自动程序。随后，您可将对话接口添加到移动设备、Web 应用程序和聊天平台 (例如，Facebook Messenger) 上的自动程序。

Amazon Lex V2 可与AWS Lambda AWS 平台上的许多其他服务（包括 Amazon Connect、Amazon Comprehend 和 Amazon Kendra）集成，您可以与这些服务集成。与 Lambda 的集成使机器人可以访问预先构建的无服务器企业连接器，从而链接到 Salesforce 等 SaaS 应用程序中的数据。

对于 2022 年 8 月 17 日之后创建的机器人，您可以使用条件分支来控制与机器人的对话流程。使用条件分支，您无需编写 Lambda 代码即可创建复杂的对话。

Amazon Lex V2 提供以下优势：

  • 简单 — Amazon Lex V2 指导您使用控制台在几分钟内创建自己的机器人。您提供几个示例短语，然后 Amazon Lex V2 构建了一个完整的自然语言模型，机器人可以通过该模型使用语音和文本进行交互，以提问、获得答案和完成复杂的任务。

     

  • 民主化的深度学习技术 — Amazon Lex V2 提供 ASR 和 NLU 技术，用于创建语音语言理解 (SLU) 系统。通过 SLU，Amazon Lex V2 接受自然语言语音和文本输入，理解输入背后的意图，并通过调用相应的业务功能来实现用户意图。

     

    语音识别和自然语言理解是计算机科学中最具挑战性的问题之一，需要在海量数据和基础设施上训练复杂的深度学习算法。Amazon Lex V2 让所有开发人员都能使用深度学习技术。Amazon Lex V2 机器人将传入的语音转换为文本，并了解用户意图以生成智能响应，这样您就可以专注于构建为客户带来附加值的机器人，并定义通过对话界面实现的全新产品类别。

     

  • 无缝部署和扩展 — 借助 Amazon Lex V2，您可以直接从 Amazon Lex V2 控制台构建、测试和部署机器人。Amazon Lex V2 使您能够发布语音或文本机器人，以便在移动设备、网络应用程序和聊天服务（例如 Facebook Messenger）上使用。Amazon Lex V2 可自动扩展。您无需担心配置硬件和管理基础设施来增强您的机器人体验。

     

  • 与 AWS 平台的内置集成 — Amazon Lex V2 可与其他 AWS 服务（例如AWS Lambda和亚马逊）原生运行 CloudWatch。您可以利用 AWS 平台的强大功能进行安全、监控、用户身份验证、业务逻辑、存储和移动应用程序开发。

     

  • 成本效益 — 使用 Amazon Lex V2，无需预付费用或最低费用。您只需为发出的文本或语音请求付费。定 pay-as-you-go 价和每个请求的低成本使该服务成为构建对话界面的经济实惠的方式。借助 Amazon Lex V2 免费套餐，您无需任何初始投资即可轻松试用 Amazon Lex V2。
