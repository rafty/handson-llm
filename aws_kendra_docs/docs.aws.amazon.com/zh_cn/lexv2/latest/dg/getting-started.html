Amazon Lex 入门

Amazon Lex V2 提供 API 操作，您可以将其与现有应用程序集成。有关支持的操作的列表，请参阅 API 参考。您可以使用以下任意选项：

  • AWS SDK — 使用软件开发工具包时，您对 Amazon Lex V2 的请求将使用您提供的证书自动签名和身份验证。我们建议您使用 SDK 构建应用程序。

  • AWS CLI— 您可以使用AWS CLI访问任何 Amazon Lex V2 功能，而无需编写任何代码。

  • AWS 控制台 — 控制台是开始测试和使用 Amazon Lex 2 的最简单方式

如果您是 Amazon Lex 的新手，我们建议您工作原理先阅读。
