JwtTokenTypeConfiguration

提供 JWT 令牌类型的配置信息。

目录

ClaimRegex

标识声明的正则表达式。

类型：字符串

长度限制：最小长度为 1。长度上限为 100。

模式：^\P{C}*$

必需：否

GroupAttributeField

组属性名称。

类型：字符串

长度限制：最小长度为 1。长度上限为 100。

模式：^\P{C}*$

必需：否

Issuer

令牌的发布者。

类型：字符串

长度限制：最小长度为 1。长度为 7 7。

模式：^\P{C}*$

必需：否

KeyLocation

密钥的位置。

类型：字符串

有效值: URL | SECRET_MANAGER

必需：是

SecretManagerArn

密钥的 Amazon 资源名称（ARN）。

类型：字符串

长度约束：最小长度为 0。长度上限为 127。

模式：arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

必需：否

URL

签名密钥 URL。

类型：字符串

长度限制：最小长度为 1。最大长度为 2048。

模式：^(https?|ftp|file):\/\/([^\s]*)

必需：否

UserNameAttributeField

用户名称属性字段。

类型：字符串

长度限制：最小长度为 1。长度上限为 100。

模式：^\P{C}*$

必需：否
