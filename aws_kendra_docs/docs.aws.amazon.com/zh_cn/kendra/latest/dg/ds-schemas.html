Amazon S3模板架构

作为模板配置的一部分，您可以添加包含数据源架构的 JSON。您提供 S3 存储桶的名称作为连接配置或存储库终端节点详细信息的一部分。还要将数据源的类型指定为S3，以及其他必要的配置。然后在呼叫Type时指定TEMPLATE为CreateDataSource。

您可以使用本开发者指南中提供的模板。请参阅S3 JSON 模式。

以下提供了有关要配置的重要 JSON 密钥的信息。

配置 描述
连接配置 有关数据来源端点的配置信息。
repositoryEndpointMetadata 数据源的端点信息。
BucketName Amazon S3 存储桶的名称。
存储库配置 数据源内容的配置信息。例如，配置特定类型的内容和字段映射。
其他属性 数据源中内容的其他配置选项

  • 包容模式

  • 排除模式

  • 排除前缀
