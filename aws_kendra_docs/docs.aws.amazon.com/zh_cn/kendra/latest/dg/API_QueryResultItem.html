QueryResultItem

单个查询结果。

查询结果包含有关查询返回的文档的信息。这包括文档的原始位置、分配给文档的属性列表以及满足查询的文档中的相关文本。

目录

AdditionalAttributes

与查询结果关联的一个或多个附加属性。

类型：AdditionalResultAttribute 对象数组

必需：否

DocumentAttributes

在搜索结果中分配给文档的文档属性数组。例如，文档的作者 (_author) 或文档的源 URI (_source_uri)。

类型：DocumentAttribute 对象数组

必需：否

DocumentExcerpt

文档中文本的摘录。包含有关突出显示摘录中相关术语的信息。

类型：TextWithHighlights 对象

必需：否

DocumentId

文档的标识符。

类型：字符串

长度限制：最小长度为 1。最大长度为 2048。

必需：否

DocumentTitle

文档的标题。包含标题文本和用于突出显示标题中相关术语的信息。

类型：TextWithHighlights 对象

必需：否

DocumentURI

文档的 URI。

类型：字符串

长度限制：最小长度为 1。最大长度为 2048。

模式：^(https?|ftp|file):\/\/([^\s]*)

必需：否

FeedbackToken

一种标识特定查询的特定结果的标记。使用此令牌为结果提供点击反馈。有关更多信息，请参阅提交反馈。

类型：字符串

长度限制：最小长度为 1。最大长度为 2048。

模式：^\P{C}*.\P{C}*$

必需：否

Format

如果响应Type中的文档是，则它要么是ANSWERTABLE答案，要么是TEXT答案。如果是表格答案，则返回表格摘录TableExcerpt。如果是文本答案，则返回文本摘录DocumentExcerpt。

类型：字符串

有效值: TABLE | TEXT

必需：否

Id

查询结果的标识符。

类型：字符串

长度限制：最小长度为 1。长度上限为 73。

必需：否

ScoreAttributes

表示 Amazon Kendra 对结果与您提供的查询相匹配的信心。每个结果都放入一个表示置信度、VERY_HIGHHIGH、MEDIUM和的方格中LOW。您可以使用分数来确定回复是否符合申请所需的可信度。

仅LOW当该Type字段设置为DOCUMENT且 Amazon Kendra 不确定结果与查询匹配时，该字段才设置为。

类型：ScoreAttributes 对象

必需：否

TableExcerpt

文档中表格的摘录。

类型：TableExcerpt 对象

必需：否

Type

响应中的文档类型。例如，响应可能包含与查询相关的问题答案。

类型：字符串

有效值: DOCUMENT | QUESTION_ANSWER | ANSWER

必需：否
