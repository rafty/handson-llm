InlineCustomDocumentEnrichmentConfiguration

提供当提取文档到 Amazon Kendra 时，用于应用基本逻辑以更改文档元数据和内容的配置信息。要应用高级逻辑，以及了解除基本逻辑功能以外的信息，请参阅HookConfiguration。

有关更多信息，请参阅在提取过程中自定义文档元数据。

目录

Condition

在提取文档到 Amazon Kendra 时，用于目标文档属性或元数据字段的条件的配置。

类型：DocumentAttributeCondition 对象

必需：否

DocumentContentDeletion

如果满足用于目标属性的条件，TRUE 会删除内容。

类型：布尔值

必需：否

Target

在提取文档到 Amazon Kendra 时，目标文档属性或元数据字段的配置。您还可以包含一个值。

类型：DocumentAttributeTarget 对象

必需：否
