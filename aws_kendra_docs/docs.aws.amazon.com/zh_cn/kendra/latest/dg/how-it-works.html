Amazon Kendra 的工作原理

Amazon Kendra为您的应用程序提供搜索功能。它直接或从第三方文档存储库中索引您的文档，并智能地向您的用户提供相关信息。您可以Amazon Kendra使用创建各种类型的文档的可更新索引。有关支持的文档类型的列表，Amazon Kendra请参阅文档类型。

Amazon Kendra将与其他服务集成。例如，您可以通过Amazon Kendra搜索为Amazon Lex聊天机器人提供有用的答案，为用户的问题提供有用的答案。您可以使用Amazon Simple Storage Service存储段作为数据源Amazon Kendra来连接和索引您的文档。而且，您可以使用设置访问策略或资源权限AWS Identity and Access Management。

Amazon Kendra 包含下列组件：

  • 一个索引，用于保存您的文档并使其可供搜索。

  • 存储您的文档并Amazon Kendra连接到的数据源。您可以自动将数据源与Amazon Kendra索引同步，以便您的索引与源存储库保持同步。

  • 一种文档添加 API，可将文档直接添加到索引。

您可以Amazon Kendra通过控制台或 API 使用。您可以创建、更新和删除索引。删除索引会删除其所有数据源连接器，并从中永久删除所有文档信息Amazon Kendra。
