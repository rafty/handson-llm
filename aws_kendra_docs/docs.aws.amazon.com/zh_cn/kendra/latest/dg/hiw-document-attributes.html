文档属性或字段

文档具有与之关联的属性或字段。文档的字段是文档的属性或文档结构中包含的内容。例如，您的每个文档可能包含标题、正文和作者。您还可以添加自己的文档自定义字段。自定义字段是您根据自己的需求指定的属性。例如，如果您的索引搜索税务文档，则可以为税务文件类型指定自定义字段，例如 W-2、1099 等。

在查询中使用文档字段之前，必须将其映射到索引字段。例如，标题字段可以映射到该字段_document_title。有关更多信息，请参阅映射字段。要添加新字段，必须创建一个将该字段映射到的索引字段。您可以使用控制台或 UpdateIndexAPI 创建索引字段。

您可以使用文档字段筛选响应并提出分面搜索建议。例如，您可以筛选回复以仅返回文档的特定版本，也可以筛选搜索以仅返回与搜索词匹配的 1099 种税务文件。有关更多信息，请参阅筛选查询。

您也可以使用文档字段手动调整查询响应。例如，在确定回复中返回哪些文档时，您可以选择增加标题字段的重要性以增加Amazon Kendra分配给该字段的权重。有关更多信息，请参阅调整搜索相关性。

如果您将文档直接添加到索引，则需要在BatchPutDocument API 的文档输入参数中指定字段。您在DocumentAttribute对象数组中指定自定义字段值。如果您使用的是数据源，则用于添加文档字段的方法取决于数据源。有关更多信息，请参阅创建自定义文档字段数：
