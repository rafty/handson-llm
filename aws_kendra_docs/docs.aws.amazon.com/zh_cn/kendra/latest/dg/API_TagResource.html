TagResource

将指定标签添加到指定的索引、常见问题解答或数据源资源。如果标签已经存在，则现有值将替换为新值。

请求语法

{
   "ResourceARN": "string",
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

请求参数

有关所有操作的通用参数的信息，请参阅常用参数。

请求接受采用 JSON 格式的以下数据。

ResourceARN

要将索引的 Amazon 资源名称（ARN）。

类型：字符串

长度限制：最小长度为 1。最大长度上限为 1011。

必需：是

Tags

要添加到索引、常见问题解答或数据源的标签键列表。如果标签已经存在，则现有值将替换为新值。

类型：Tag 对象数组

数组成员：最少 0 项。最多 A200 200 200 200 200

必需：是

响应元素

如果此操作成功，则该服务会发送回带有空 HTTP 正文的 HTTP 200 响应。

错误

有关所有操作常见错误的信息，请参阅常见错误。

AccessDeniedException

您没有足够的权限来执行此操作。请确保您拥有所需的权限策略和用户帐户，然后重试。

HTTP 状态代码：400

InternalServerException

用于您的Amazon Kendra 服务的内部服务器出现问题。请稍等几分钟，然后重试，或联系Su pport 部门寻求帮助。

HTTP 状态代码：500

ResourceUnavailableException

您要使用的资源不可用。请检查您提供的资源是否正确，然后重试。

HTTP 状态代码：400

ThrottlingException

由于请求限制而导致请求被拒绝。请减少请求数量，然后重试。

HTTP 状态代码：400

ValidationException

输入未能满足 Amazon Kendra 请提供正确的输入并重试。

HTTP 状态代码：400
