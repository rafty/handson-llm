文档类型

一个索引可以包含多个文档和多种类型的文档。索引还可以包括结构化和非结构化文本。

结构化文本

  • 常见问题和解答

常见问答格式文档用于回答诸如 “太空针塔有多高？” 之类的问题 您可以指定多个返回相同答案的问题。您可以在存储Amazon S3段中存储的逗号分隔值（CSV）文件中指定问题和答案。

如果您想使用 JSON 文件来解答您的常见问题和答案，JSON 文件被视为纯文本，是一种非结构化文本。

有关示例，请参阅将问题和答案直接添加到索引。

非结构化文本

Amazon Kendra支持以下类型的非结构化文本：

  • HTML 文件

  • 微软 PowerPoint （PPT）演示文稿

  • MS WORD 文档

  • 纯文本文档

  • PDF文件

  • 逗号分隔值 (CSV) 文件

  • 微软 Excel (MS Excel) 文件

  • XML 文件

  • JSON 文件

  • Markdown 文档 (MD) 文件

  • 富文本格式 (RTF) 文件

  • 可扩展样式表语言转换 (XSLT) 文件

您可以通过三种方式将非结构化文档添加到索引中：

  • 使用 BatchPutDocumentAPI。

  • 作为数据来源。有关更多信息，请参阅从数据源添加文档。

  • 作为二进制数据。有关更多信息，请参阅使用 API 添加文档。
