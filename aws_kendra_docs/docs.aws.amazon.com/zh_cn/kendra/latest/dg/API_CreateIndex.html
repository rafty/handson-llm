CreateIndex

创建 Amazon Kendra 索引。索引创建是一个异步 API。要确定索引创建是否已完成，请检查调用后返回的Status字段DescribeIndex。当索引准备就绪ACTIVE时，该Status字段设置为。

一旦索引处于活动状态，就可以使用BatchPutDocument API 或使用受支持的数据源之一对文档进行索引。

有关使用 Python SDK 创建索引和数据源的示例，请参阅 Pyt hon SDK 入门。有关使用 Java SDK 创建索引和数据源的示例，请参阅 Jav a SDK 入门。

请求参数

有关所有操作的通用参数的信息，请参阅常用参数。

请求接受采用 JSON 格式的以下数据。

ClientToken

您提供的用于识别创建索引的请求的令牌。使用相同的客户端令牌多次调用CreateIndex API 只会创建一个索引。

类型：字符串

长度限制：最小长度为 1。长度上限为 100。

必需：否

Description

索引的描述。

类型：字符串

长度限制：最小长度为 0。最大长度为 1000。

模式：^\P{C}*$

必需：否

Edition

用于索引的Amazon Kendra 版本。选择DEVELOPER_EDITION用于开发、测试或概念验证的索引。ENTERPRISE_EDITION用于生产。一旦为索引设置了版本，就无法对其进行更改。

Edition 参数是可选的。如果您未提供值，则默认值为ENTERPRISE_EDITION。

有关企业版和开发者版配额限制的更多信息，请参阅配额。

类型：字符串

有效值: DEVELOPER_EDITION | ENTERPRISE_EDITION

必需：否

Name

索引的名称。

类型：字符串

长度限制：最小长度为 1。长度上限为 1000。

模式：[a-zA-Z0-9][a-zA-Z0-9_-]*

必需：是

RoleArn

IAmazon 角色的 Amazon 资源名称（ARN），有权访问您的 Amazon CloudWatch 日志和指标的权限。有关更多信息，请参阅 Amazon Kendra 的 IIAmazon KM 访问角色。

类型：字符串

长度约束：最小长度为 0。最大长度。

模式：arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

必需：是

ServerSideEncryptionConfiguration

AWS KMS客户托管的标识符（CMK）的标识符，用于加密由 Amazon Kendra 编制索引的数据。Amazon Kendra 不支持非对称 CMK。

类型：ServerSideEncryptionConfiguration 对象

必需：否

Tags

用于标识或分类索引的键值对列表。您还可以使用标签来帮助控制对索引的访问。标签键和值可以由 Unicode 字母、数字、空格和以下任何符号：_.:/= +-@

类型：Tag 对象数组

数组成员：最少 0 项。最大。。

必需：否

UserContextPolicy

用户上下文策略。

ATTRIBUTE_FILTER

所有索引内容都可以搜索，并显示给所有用户。如果您要根据用户上下文筛选搜索结果，则可以使用 _user_id 和 _group_ids 的属性筛选条件，也可以在 UserContext 中提供用户和组信息。

USER_TOKEN

启用基于令牌的用户访问控制以根据用户上下文筛选搜索结果。所有没有访问控制的文档以及用户可访问的所有文档都将是可搜索和显示的。

类型：字符串

有效值: ATTRIBUTE_FILTER | USER_TOKEN

必需：否

UserGroupResolutionConfiguration

从AWS IAM Identity Center (successor to AWS Single Sign-On)身份源获取用户和群组。要对此进行配置，请参阅UserGroupResolutionConfiguration。

类型：UserGroupResolutionConfiguration 对象

必需：否

UserTokenConfigurations

用户令牌配置。

类型：UserTokenConfiguration 对象数组

数组成员：最多 1 项。

必需：否

响应语法

{
   "Id": "string"
}

响应元素

如果此操作成功，则该服务将会发送回 HTTP 200 响应。

服务以 JSON 格式返回的以下数据。

Id

索引的标识符。在查询索引、设置数据源或为文档编制索引时使用此标识符。

类型：字符串

长度限制：固定长度为 36。

模式：[a-zA-Z0-9][a-zA-Z0-9-]*

错误

ConflictException

请求发生了冲突。请修复与您的资源存在的任何不一致之处，然后重试。

HTTP 状态代码：400

HTTP 状态代码：400

HTTP 状态代码：400
