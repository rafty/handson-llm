开始使用 (AWS CLI)

以下过程显示如何使用创建Amazon Kendra索引AWS CLI。该过程创建数据源、索引并在索引上运行查询。

创建Amazon Kendra索引 (CLI)

  1. 做先决条件.

  2. 要创建索引，请输入以下命令，请输入以下命令。

    aws kendra create-index \
 --name cli-getting-started-index \
 --description "Index for CLI getting started guide." \
 --role-arn arn:aws:iam::account id:role/KendraRoleForGettingStartedIndex

  3. 等待Amazon Kendra创建索引。使用以下命令检查进度。当状态字段为时ACTIVE，请继续执行步。

    aws kendra describe-index \
 --id index id

  4. 在命令提示符处，输入以下命令以创建数据源，请输入以下命令。

    aws kendra create-data-source \
 --index-id index id \
 --name data source name \
 --role-arn arn:aws:iam::account id:role/KendraRoleForGettingStartedDataSource \
 --type S3 \
 --configuration '{"S3Configuration":{"BucketName":"S3 bucket name"}}'

    如果您使用模板架构连接到数据源，请配置模板架构。

    aws kendra create-data-source \
 --index-id index id \
 --name data source name \
 --role-arn arn:aws:iam::account id:role/KendraRoleForGettingStartedDataSource \
 --type TEMPLATE \
 --configuration '{"TemplateConfiguration":{"Template":{JSON schema}}}'

  5. 创建数据源需要Amazon Kendra一段时间。要检查进度，请输入以下命令，请输入以下命令。当状态为时ACTIVE，请继续执行步。

    aws kendra describe-data-source \
 --id data source ID \
 --index-id index ID

  6. 要同步执行数据源，请输入以下命令，请输入以下命令。

    aws kendra start-data-source-sync-job \
 --id data source ID \
 --index-id index ID

  7. Amazon Kendra将索引您的数据源。所花费的时间取决于文档的数量。您可以使用以下命令检查同步任务的状态。当状态为时ACTIVE，请继续执行步。

    aws kendra describe-data-source \
 --id data source ID \
 --index-id index ID

  8. 要进行查询，请输入以下命令，请输入以下命令。

    aws kendra query \
 --index-id index ID \
 --query-text "search term"

    搜索结果以 JSON 格式显示。
