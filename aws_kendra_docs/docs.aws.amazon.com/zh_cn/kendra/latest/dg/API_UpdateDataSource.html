请求参数

有关所有操作的通用参数的信息，请参阅通用参数。

请求接受采用 JSON 格式的以下数据。

Configuration

您要更新数据来源连接器的配置信息。

类型：DataSourceConfiguration 对象

必需：否

CustomDocumentEnrichmentConfiguration

您要更新用于在文档提取过程中更改文档元数据和内容的配置信息。

有关如何在将文档载入 Amazon Kendra 时创建、修改和删除文档元数据或进行其他内容更改的更多信息，请参阅在摄取过程中自定义文档元数据。

类型：CustomDocumentEnrichmentConfiguration 对象

必需：否

Description

数据来源连接器的新描述。

类型：字符串

长度限制：最小长度为 0。最大长度为 1000。

模式：^\P{C}*$

必需：否

Id

您要更新的数据来源连接器的标识符。

类型：字符串

长度限制：最小长度为 1。长度上限为 100。

模式：[a-zA-Z0-9][a-zA-Z0-9_-]*

必需：是

IndexId

数据来源连接器结合使用的索引的标识符。

类型：字符串

度度度度度度度度度度度度度度度度度

模式：[a-zA-Z0-9][a-zA-Z0-9-]*

必需：是

LanguageCode

您要为数据来源连接器更新的语言的代码。这允许您在更新数据源时支持所有文档的语言。默认情况下支持英语。有关支持的语言（包括其代码）的更多信息，请参阅添加英语以外语言的文档。

类型：字符串

最大度度度度度度度度度度度度度度度度 最大度度度度度度度度度度

模式：[a-zA-Z-]*

必需：否

Name

数据来源连接器的新名称。

类型：字符串

长度限制：最小长度为 1。长度上限为 1000。

模式：[a-zA-Z0-9][a-zA-Z0-9_-]*

必需：否

RoleArn

具有访问数据源权限的角色的 Amazon 资源名称（ARN）。有关更多信息，请参阅 Amazon Kendra 的 IAM 角色。

类型：字符串

长度约束：最小长度为 0。最大度度度度度度度度度度度度

模式：arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

必需：否

Schedule

您要为数据来源连接器更新的同步计划。

类型：字符串

必需：否

VpcConfiguration

用于将 AAmazon Virtual Private Cloud 连接到数据源的配置信息。有关更多信息，请参阅配置 VPC。

类型：DataSourceVpcConfiguration 对象

必需：否
