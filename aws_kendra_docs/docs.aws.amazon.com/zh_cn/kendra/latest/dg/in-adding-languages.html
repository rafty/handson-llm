添加英语以外其他语言的文档

您可以索引多种语言的文档。如果您未指定语言，默认情况下将英语文档Amazon Kendra索引。您可以将文档的语言代码作为字段包含在文档元数据中。有关文档字段的更多信息，请参阅_language_code字段映射和自定义属性。

调用时，可以为数据源中的所有文档指定语言代码CreateDataSource。如果文档未在元数据字段中指定语言代码，则使用在数据源级别为所有文档指定的语言代码为该文档编制索引。在控制台中，您只能在数据源级别使用支持的语言对文档进行索引。转到数据源，然后转到指定数据源详细信息页面，然后从语言下拉列表中选择一种语言。

您也可以使用支持的语言搜索或查询文档。有关更多信息，请参阅使用语言搜索。

对于支持语义搜索的语言，支持以下功能。

  • 文档相关性不仅仅是简单的关键字匹配。

  • 常见问题解答不仅仅是简单的关键字匹配。

  • 根据Amazon Kendra阅读理解从文档中提取答案。

  • 搜索结果的置信度等级（非常高、高、中和低）。

对于不支持语义搜索的语言，文档相关性和常见问题解答支持简单的关键字匹配。

仅英语（默认语言）支持同义词（包括自定义同义词）、增量学习和反馈以及查询建议。
