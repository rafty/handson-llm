Confluence

Confluence 是一种协作式工作管理工具，专为共享、存储和处理项目规划、软件开发和产品管理而设计。您可以使用Amazon Kendra索引 Confluence 空间、页面（包括嵌套页面）、博客以及索引页面和博客的评论和附件。

Amazon Kendra同时支持 Confluence 服务器和 Confluence Cloud。

注意

默认情况下，Amazon Kendra不索引 Confluence 档案和个人空间。在创建数据源时，您可以选择对它们进行索引。如果您不想索引空间Amazon Kendra，请在 Confluence 中将其标记为私有。

您可以使用Amazon Kendra控制台、API 或 TemplateConfigurationAPIAmazon Kendra 连接到您的 Confluence 数据源。ConfluenceConfiguration

Amazon Kendra有两个版本的 Confluence 连接器。每个版本支持的功能包括：

Confluence 连接器 V1.0/ConfluenceConfigurationAPI

  • 字段映射

  • 用户上下文过滤

  • 包含/排除过滤器

  • （仅适用于 Confluence）Clud（VPC）

Confluence 连接器 V2.0/TemplateConfigurationAPI

  • 字段映射

  • 用户上下文过滤

  • Virtual Private Cloud (VPC)

  • 同步所有文档/仅同步新的、修改过的或已删除的文档

  • 纳入/排除模式

注意

对 Confluence 连接器 V1.0/ ConfluenceConfiguration API 的Support 计划于 2023 年 6 月结束。我们建议使用 Confluence 连接器 V2.0/ TemplateConfiguration API。

要对Amazon Kendra Confluence 数据源连接器进行故障排除，请参阅解决数据源问题。
