CreateDataSource

创建您要与 Amazon Kendra 索引结合使用的数据来源连接器。

您可以为数据来源指定名称、数据来源连接器类型和描述。您还可以为数据来源连接器指定配置信息。

CreateDataSource是一个同步操作。如果成功创建了数据源，则该操作返回 200。否则，会引发异常。

有关使用 Python SDK 创建索引和数据源的示例，请参阅 Pyt hon SDK 入门。有关使用 Java SDK 创建索引和数据源的示例，请参阅 Jav a SDK 入门。

请求参数

有关所有操作的通用参数的信息，请参阅常用参数。

请求接受采用 JSON 格式的以下数据。

ClientToken

您提供的令牌，用于识别识别创建数据源连接器创建数据源连接器创建数据源连接器请求的标记。使用相同的客户端令牌CreateDataSource对 API 进行多次调用将仅创建一个数据源连接器。

类型：字符串

长度限制：最小长度为 1。长度上限为 100。

必需：否

Configuration

连接到您的数据源存储库的配置信息。

Type 参数设为 CUSTOM 时，无法指定 Configuration 参数。如果你这样做，你会收到一个ValidationException异常。

对于所有其他数据源，Configuration 参数是必需的。

类型：DataSourceConfiguration 对象

必需：否

CustomDocumentEnrichmentConfiguration

用于在文档提取过程中更改文档元数据和内容的配置信息。

有关如何在将文档载入 Amazon Kendra 时创建、修改和删除文档元数据或进行其他内容更改的更多信息，请参阅在摄取过程中自定义文档元数据。

类型：CustomDocumentEnrichmentConfiguration 对象

必需：否

Description

数据来源连接器的描述。

类型：字符串

长度限制：最小长度为 0。最大长度为 1000。

模式：^\P{C}*$

必需：否

IndexId

您要与数据来源连接器结合使用的索引的标识符。

类型：字符串

长度限制：固定长度为 36

模式：[a-zA-Z0-9][a-zA-Z0-9-]*

必需：是

LanguageCode

一种语言的代码。这允许您在创建数据源连接器时支持所有文档的语言。默认情况下支持英语。有关支持的语言（包括其代码）的更多信息，请参阅添加英语以外语言的文档。

类型：字符串

长度限制：最最最最最最最最最最最最最 最大长度为 10

模式：[a-zA-Z-]*

必需：否

Name

数据源连接器的名称。

类型：字符串

长度限制：最小长度为 1。长度上限为 1000。

模式：[a-zA-Z0-9][a-zA-Z0-9_-]*

必需：是

RoleArn

Type 参数设为 CUSTOM 时，无法指定 RoleArn 参数。如果你这样做，你会收到一个ValidationException异常。

对于所有其他数据源，RoleArn 参数是必需的。

类型：字符串

长度约束：最小长度为 0。最大长度为 1284

模式：arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Schedule

为 Amazon Kendra 设置检查数据源存储库中的文档并更新索引的频率，如果您未设置计划 Amazon Kendra 将不会定期更新索引 Amazon 您可以调用StartDataSourceSyncJob API 来更新索引。

指定cron-格式化时间表字符串或空字符串以表示索引是按需更新的。

Type 参数设为 CUSTOM 时，无法指定 Schedule 参数。如果你这样做，你会收到一个ValidationException异常。

类型：字符串

必需：否

Tags

用于标识或分类数据源连接器的一组键值对列表，您还可以使用标签来帮助控制对数据源连接器访问，标签键和值可以包含Unicodecodept 字母、数字、空格和以下任何符号组成：_.:/= +-@

Type
