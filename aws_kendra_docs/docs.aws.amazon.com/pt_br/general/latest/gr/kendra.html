Endpoints e cotas do Amazon Kendra

A seguir estão os endpoints de serviço e as cotas de serviço desse serviço. Para se conectar a um serviço da AWS de forma programática, use um endpoint. Além dosAWS endpoints padrão, algunsAWS serviços oferecem endpoints FIPS em regiões selecionadas. Para obter mais informações, consulte Endpoints de serviço da AWS. As cotas de serviço, também conhecidas como limites, são o número máximo de recursos ou operações de serviço para suaAWS conta. Para obter mais informações, consulte Cotas de serviço do AWS.
