Autenticação multifator

Quando você habilita a autenticação multifator (MFA), os usuários devem fazer login no portal de AWS acesso com seu nome de usuário e senha. Esse é o primeiro fator, algo que eles sabem. Os usuários também devem fazer login com um código ou uma chave de segurança. Esse é o segundo fator, algo que eles têm ou são. O segundo fator pode ser um código de autenticação gerado a partir do dispositivo móvel ou, alternativamente, ao tocar em uma chave de segurança conectada ao computador. Juntos, esses vários fatores oferecem maior segurança ao impedir o acesso não autorizado aos seus AWS recursos, a menos que um desafio válido de MFA tenha sido concluído com êxito. Cada usuário pode registrar até oito dispositivos MFA.

Importante

Como uma prática recomendada de segurança, é altamente recomendável que você habilite a autenticação multifator. O MFA fornece uma maneira simples e segura de adicionar uma camada extra de proteção ao mecanismo de autenticação padrão de nome de usuário e senha.
