O que é Amazon Lex V2?

O Amazon Lex V2 é um serviço da AWS para criar interfaces conversacionais para aplicativos usando voz e texto. O Amazon Lex V2 fornece a ampla funcionalidade e flexibilidade do entendimento de linguagem natural (NLU) e do reconhecimento automático de fala (ASR) para que você possa criar experiências de usuário altamente envolventes com interações conversacionais e realistas e criar novas categorias de produtos.

O Amazon Lex V2 permite que qualquer desenvolvedor crie bots conversacionais rapidamente. Com o Amazon Lex V2, não é necessário nenhum conhecimento em aprendizado profundo — para criar um bot, você especifica o fluxo básico de conversação no console do Amazon Lex V2. O Amazon Lex V2 gerencia a caixa de diálogo e ajusta dinamicamente as respostas na conversa. Usando o console, você pode criar, testar e publicar o chatbot de texto ou voz. Em seguida, você pode adicionar as interfaces de conversa aos bots em dispositivos móveis, aplicativos Web e plataformas de bate-papo (por exemplo, Facebook Messenger).

O Amazon Lex V2 fornece integração comAWS Lambda, e você pode integrá-lo com muitos outros serviços na plataforma da AWS, incluindo Amazon Connect, Amazon Comprehend e Amazon Kendra. A integração com o Lambda fornece aos bots acesso a conectores corporativos sem servidor pré-construídos para se vincularem aos dados em aplicativos SaaS, como o Salesforce.

Para bots criados após 17 de agosto de 2022, você pode usar a ramificação condicional para controlar o fluxo de conversas com seu bot. Com a ramificação condicional, você pode criar conversas complexas sem precisar escrever código Lambda.

O Amazon Lex V2 oferece os seguintes benefícios:

  • Simplicidade — O Amazon Lex V2 orienta você a usar o console para criar seu próprio bot em minutos. Você fornece algumas frases de exemplo, e o Amazon Lex V2 cria um modelo completo de linguagem natural por meio do qual o bot pode interagir usando voz e texto para fazer perguntas, obter respostas e concluir tarefas sofisticadas.

     

  • Tecnologias democratizadas de aprendizado profundo — o Amazon Lex V2 fornece tecnologias ASR e NLU para criar um sistema de compreensão de fala e linguagem (SLU). Por meio do SLU, o Amazon Lex V2 usa voz e texto em linguagem natural, compreende a intenção por trás da entrada e cumpre a intenção do usuário invocando a função comercial apropriada.

     

    O reconhecimento de fala e a compreensão da linguagem natural são alguns dos problemas mais difíceis de resolver na ciência da computação, exigindo que algoritmos sofisticados de aprendizado profundo sejam treinados em grandes quantidades de dados e infraestrutura. O Amazon Lex V2 coloca as tecnologias de aprendizado profundo ao alcance de todos os desenvolvedores. Os bots Amazon Lex V2 convertem a fala recebida em texto e entendem a intenção do usuário de gerar uma resposta inteligente para que você possa se concentrar em criar seus bots com valor agregado para seus clientes e definir categorias totalmente novas de produtos possibilitadas por meio de interfaces de conversação.

     

  • Implantação e escalabilidade perfeitas — Com o Amazon Lex V2, você pode criar, testar e implantar seus bots diretamente do console Amazon Lex V2. O Amazon Lex V2 permite que você publique seus bots de voz ou texto para uso em dispositivos móveis, aplicativos da web e serviços de bate-papo (por exemplo, o Facebook Messenger). O Amazon Lex V2 é escalável automaticamente. Você não precisa se preocupar em provisionar hardware e gerenciar a infraestrutura para potencializar sua experiência com bots.

     

  • Integração integrada com a plataforma da AWS — o Amazon Lex V2 opera nativamente com outros serviços da AWS, comoAWS Lambda a Amazon CloudWatch. Você pode aproveitar o poder da plataforma da AWS para segurança, monitoramento, autenticação do usuário, lógica de negócios, armazenamento e desenvolvimento de aplicativos móveis.

     

  • Custo-benefício — Com o Amazon Lex V2, não há custos iniciais ou taxas mínimas. Você será cobrado apenas pelas solicitações de texto ou fala feitas. O pay-as-you-go preço e o baixo custo por solicitação tornam o serviço uma forma econômica de criar interfaces de conversação. Com o nível gratuito do Amazon Lex V2, você pode experimentar facilmente o Amazon Lex V2 sem nenhum investimento inicial.
