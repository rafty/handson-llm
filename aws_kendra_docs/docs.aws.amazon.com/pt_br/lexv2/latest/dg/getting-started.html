V2

O Amazon Lex V2 fornece operações de API que você pode integrar com seus aplicativos existentes. Para obter uma lista das operações com suporte, consulte a Referência de APIs. Você pode usar qualquer uma das opções a seguir:

  • SDK da AWS — Ao usar os SDKs, suas solicitações para o Amazon Lex V2 são automaticamente assinadas e autenticadas usando as credenciais que você fornece. Recomendamos que você use um SDK para criar seu aplicativo.

  • AWS CLI— Você pode usar oAWS CLI para acessar qualquer recurso do Amazon Lex V2 sem ter que escrever nenhum código.

  • Console AWS — O console é a maneira mais fácil de começar a testar e usar o Amazon Lex V2

Se você for novidade na Amazon Lex V2, recomendamos lerComo funciona primeiro.
