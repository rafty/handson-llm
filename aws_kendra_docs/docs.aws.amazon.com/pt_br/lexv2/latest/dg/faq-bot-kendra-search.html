Exemplo: Criação de um bot de perguntas frequentes para um índice Amazon Kendra

Este exemplo cria um bot Amazon Lex V2 que usa um índice Amazon Kendra para fornecer respostas às perguntas dos usuários. O bot de perguntas frequentes gerencia a caixa de diálogo para o usuário. Ele usa a intenção AMAZON.KendraSearchIntent para consultar o índice e apresentar a resposta ao usuário. Aqui está um resumo de como você criará seu bot de perguntas frequentes usando um índice Amazon Kendra:

  1. Criar um bot com o qual seus clientes vão interagir para obter respostas do bot.

  2. Criar uma intenção personalizada. Como osAMAZON.KendraSearchIntent eAMAZON.FallbackIntent são intenções de backup, seu bot exige pelo menos uma outra intenção que deve conter pelo menos uma expressão. Essa intenção permite que o bot crie, mas não seja usado de outra forma. Portanto, seu bot de perguntas frequentes conterá pelo menos três intenções, como na imagem abaixo:

  3. Adicione aAMAZON.KendraSearchIntent intenção ao seu bot e configure-o para funcionar com seu índice Amazon Kendra.

  4. Teste o bot fazendo uma consulta e verificando se os resultados do seu índice Amazon Kendra são documentos que respondem à consulta.

Pré-requisitos

Antes que você possa usar esse exemplo, é necessário criar um índice do Amazon Kendra. Para obter mais informações, consulte Introdução ao console Amazon Kendra no Guia do desenvolvedor do Amazon Kendra. Neste exemplo, escolha o conjunto de dados de amostra (amostra de documentação da AWS) como sua fonte de dados.

Para criar um bot de perguntas frequentes:

  1. Faça login noAWS Management Console e abra o console do Lex do Amazon Lex em https://console.aws.amazon.com/lex/.

  2. No painel de navegação, selecione Bots.

  3. Escolha Criar bot.

    1. Para o método de criação, escolha Criar um bot em branco.

    2. Na seção Configuração do bot, dê ao bot um nome que indique sua finalidadeKendraTestBot, como uma descrição opcional. O nome deve ser exclusivo na sua conta.

    3. Na seção Permissões do IAM, escolha Criar uma função com permissões básicas do Amazon Lex. Isso criará uma função AWS Identity and Access Management(IAM) com as permissões que o Amazon Lex V2 precisa para executar seu bot.

    4. Na seção Lei de Proteção à Privacidade Online para Crianças (COPPA), escolha Não.

    5. Nas seções Tempo limite da sessão inativa e Configurações avançadas, deixe as configurações padrão e escolha Avançar.

    6. Agora você está na seção Adicionar idioma ao bot. No menu em Interação por voz, selecione Nenhuma. Este é apenas um aplicativo baseado em texto. Mantenha as configurações padrão

    7. Escolha Done (Concluído). O Amazon Lex V2 cria seu bot e uma intenção padrão chamada NewIntent, e leva você à página para configurar essa intenção.

Para criar um bot com sucesso, você deve criar pelo menos uma intenção separada doAMAZON.FallbackIntent e doAMAZON.KendraSearchIntent. Essa intenção é necessária para criar seu bot Amazon Lex V2, mas não é usada para responder às perguntas frequentes. Essa intenção deve conter pelo menos um exemplo de enunciado e o enunciado não deve se aplicar a nenhuma das perguntas feitas pelo cliente.

Para criar a intenção necessária:

  1. Na seção Detalhes da intenção, dê um nome à intenção, comoRequiredIntent.

  2. Na seção Exemplos de enunciados, digite um enunciado na caixa ao lado de Adicionar enunciado, comoRequired utterance. Em seguida, escolha Adicionar enunciado.

  3. Selecione Salvar intenção.

Crie a intenção de pesquisar um índice do Amazon Kendra e a mensagem de resposta que ele deve retornar.

Para criar uma AMAZON. KendraSearchIntent mensagem de intenção e resposta:

  1. Selecione Voltar à lista de intenções no painel de navegação para retornar à página de intenções do seu bot. Escolha Adicionar intenção e selecione Usar intenção embutida no menu suspenso.

  2. Na caixa exibida, selecione o menu em Intenção embutida. EntreAMAZON.KendraSearchIntent na barra de pesquisa e, em seguida, escolha-a na lista.

  3. Dê um nome à intenção, comoKendraSearchIntent.

  4. No menu suspenso do índice do Amazon Kendra, escolha o índice no qual você deseja que a intenção pesquise. O índice que você criou na seção Pré-requisitos deve estar disponível.

  5. Select Add (Adicionar).

  6. No editor de intenção, role para baixo até a seção Cumprimento, selecione a seta para a direita para expandir a seção e adicione a seguinte mensagem na caixa em Sobre o cumprimento bem-sucedido: 

    I found a link to a document that could help you: ((x-amz-lex:kendra-search-response-document-link-1)).

    Para obter mais informações sobre o Amazon Kendra Search Response, consulte Usando a resposta de pesquisa.

  7. Escolha Salvar intenção e selecione Criar para criar o bot. Quando o bot estiver pronto, o banner na parte superior da tela ficará verde e exibirá uma mensagem de sucesso.

Por fim, use a janela de teste do console para testar as respostas do bot.

Para testar seu bot de perguntas frequentes:

  1. Depois que o bot for criado com sucesso, escolha Testar.

  2. EntreWhat is Amazon Kendra? na janela de teste do console. Verifique se o bot responde com um link.

  3. Para obter mais informações sobre configuraçãoAMAZON.KendraSearchIntent, consulte AMAZON. KendraSearchIntente KendraConfiguration.
