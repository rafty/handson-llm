Respostas de consulta e tipos de resposta

Amazon Kendraoferece suporte a diferentes respostas de consulta e tipos de resposta.

Respostas da consulta

Uma chamada para a API Query retorna informações sobre os resultados de uma pesquisa. Os resultados estão em uma matriz deQueryResultItem objetos (ResultItems). Cada umQueryResultItem inclui um resumo do resultado. Os atributos do documento associados ao resultado da consulta estão incluídos.

Informações resumidas

As informações resumidas variam de acordo com o tipo de resultado. Em cada caso, ele inclui texto de documento que corresponde ao termo de pesquisa. Ele também inclui informações de destaque que você pode usar para destacar o texto de pesquisa na saída do seu aplicativo. Por exemplo, se o termo de pesquisa for qual é a altura do Space Needle? , as informações resumidas incluem a localização do texto para as palavras altura e espaço da agulha. Para obter mais informações sobre os tipos de respostas, consulteRespostas de consulta e tipos de resposta.

Atributos de documentos

Cada resultado contém atributos do documento que correspondem a uma consulta. Alguns dos atributos são predefinidosDocumentId, comoDocumentTitle,DocumentUri e. Outros são atributos personalizados que você define. Você pode usar atributos do documento para filtrar a resposta daQuery API. Por exemplo, talvez você queira somente os documentos escritos por um autor específico ou uma versão específica de um documento. Para obter mais informações, consulte Filtragem e pesquisa de facetas. Você especifica atributos do documento ao adicionar documentos a um índice. Para obter mais informações, consulte Campos ou atributos personalizados.

Tipos de respostas

Amazon Kendraretorna três tipos de resposta à consulta.

  • Resposta (inclui a resposta da tabela)

  • Documento

  • Pergunta e resposta

O tipo da resposta é retornado no campo deType resposta do QueryResultItemobjeto.

Resposta

Amazon Kendradetectou uma ou mais respostas de perguntas na resposta. Um factoide é a resposta a uma pergunta sobre quem, o quê, quando ou onde, como Onde fica o centro de serviços mais próximo de mim? Amazon Kendraretorna o texto no índice que melhor corresponde à consulta. O texto está noAnswerText campo e contém informações de destaque para o termo de pesquisa no texto da resposta. AnswerTextinclui o trecho completo do documento com texto destacado, enquantoDocumentExcerpt inclui o trecho do documento truncado (290 caracteres) com texto destacado.

Amazon Kendraretorna apenas uma resposta por documento, e essa é a resposta com a maior confiança. Para retornar várias respostas de um documento, você deve dividir o documento em vários documentos.

Documento

Amazon Kendraretorna documentos classificados para aqueles que correspondem ao termo de pesquisa. A classificação é baseada na confiança queAmazon Kendra tem na precisão do resultado da pesquisa. As informações sobre o documento correspondente são retornadas no QueryResultItem. Inclui o título do documento. O trecho inclui informações de destaque para o texto de pesquisa e a seção do texto correspondente no documento. O URI para documentos correspondentes está no atributo doSourceURI documento. Pergunta e resposta

Uma resposta de pergunta e resposta é retornada quandoAmazon Kendra combina uma pergunta com uma das perguntas mais frequentes em seu índice. A resposta inclui a pergunta e a resposta correspondentes no QueryResultItemcampo. Também inclui informações de destaque para termos de consulta detectados na sequência de caracteres de consulta. O JSON a seguir mostra uma resposta de pergunta e resposta. Observe que a resposta inclui o texto da pergunta.
