CreateIndex

Cria um índice do Amazon Kendra. A criação de índices é uma API assíncrona. Para determinar se a criação do índice foi concluída, verifique oStatus campo retornado de uma chamada paraDescribeIndex. OStatus campo é definido paraACTIVE quando o índice estiver pronto para uso.

Quando o índice estiver ativo, será possível indexar os documentos com aBatchPutDocument API ou usando uma das fontes de dados compatíveis.

Para ver um exemplo de como criar um índice e uma fonte de dados usando o SDK do Python, consulte Introdução ao SDK do Python. Para ver um exemplo de como criar um índice e uma fonte de dados usando o Java SDK, consulte Introdução ao Java SDK.

Parâmetros de solicitação

Para obter informações sobre os parâmetros que são comuns a todas as ações, consulte Parâmetros comuns.

A solicitação aceita os dados a seguir no formato JSON.

ClientToken

Um token que você fornece para identificar a solicitação de criação de um índice. Várias chamadas para aCreateIndex API com o mesmo token de cliente criarão somente um índice.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 100.

Obrigatório: não

Description

Uma descrição para o índice.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1000.

Padrão: ^\P{C}*$

Obrigatório: não

Edition

A edição Amazon Kendra a ser usada no índice. Escolha índicesDEVELOPER_EDITION destinados ao desenvolvimento, teste ou prova de conceito. UseENTERPRISE_EDITION para produção. Depois de definir a edição de um índice, ela não pode ser alterada.

O parâmetro Edition é opcional. Se você não fornecer um valor, o padrão seráENTERPRISE_EDITION.

Para obter mais informações sobre limites de cota para as edições Enterprise e Developer, consulte Cotas.

Tipo: String

Valores válidos: DEVELOPER_EDITION | ENTERPRISE_EDITION

Obrigatório: não

Name

Um nome para o índice.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 1000.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

Obrigatório: Sim

RoleArn

O nome de recurso da Amazon (ARN) de um perfil do IAM com permissão para acessar seus CloudWatch registros e as métricas da Amazon. Para obter mais informações, consulte Funções de acesso do IAM para o Amazon Kendra.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1284.

Padrão: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Obrigatório: Sim

ServerSideEncryptionConfiguration

O identificador da chave gerenciada doAWS KMS cliente (CMK) que é usada para criptografar dados indexados pelo Amazon Kendra. O Amazon Kendra não oferece suporte a CMKs assimétricas.

Tipo: objeto ServerSideEncryptionConfiguration

Obrigatório: não

Tags

Uma lista de pares de chave e que identificam ou categorizam o índice. Também é possível usar etiquetas para ajudar a controlar o acesso ao índice. As chaves e os valores de etiqueta podem consistir em letras, dígitos, espaços em branco e os seguintes símbolos: _.:/= + - @.: @.:

Tipo: matriz de objetos Tag

Membros da matriz: número mínimo de 0 itens. Número máximo de 200 itens.

Obrigatório: não

UserContextPolicy

A política de contexto do usuário.

ATTRIBUTE_FILTER

Todo o conteúdo indexado é pesquisável e pode ser exibido para todos os usuários. Se quiser filtrar os resultados da pesquisa por contexto do usuário, você poderá usar os filtros de atributos de _user_id e _group_ids, ou poderá fornecer as informações de usuário e grupo no UserContext.

USER_TOKEN

Habilita o controle de acesso de usuário baseado em token para filtrar os resultados de pesquisa por contexto do usuário. Todos os documentos sem controle de acesso e todos os documentos acessíveis ao usuário serão pesquisáveis e exibidos.

Tipo: String

Valores válidos: ATTRIBUTE_FILTER | USER_TOKEN

Obrigatório: não

UserGroupResolutionConfiguration

Obtém usuários e grupos da fonte deAWS IAM Identity Center (successor to AWS Single Sign-On) identidade. Para configurar isso, consulte UserGroupResolutionConfiguration.

Tipo: objeto UserGroupResolutionConfiguration

Obrigatório: não

UserTokenConfigurations

A configuração do token do usuário.

Tipo: matriz de objetos UserTokenConfiguration

Membros da matriz: número máximo de 1 item.

Obrigatório: não

Elementos de resposta

Se a ação for bem-sucedida, o serviço reenviará uma resposta HTTP 200.

Os dados a seguir são retornados no formato JSON pelo serviço.

Id

O identificador do índice. Use esse identificador ao consultar um índice, configurar uma fonte de dados ou indexar um documento.

Tipo: String

Restrições de comprimento: comprimento fixo de 36.

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*
