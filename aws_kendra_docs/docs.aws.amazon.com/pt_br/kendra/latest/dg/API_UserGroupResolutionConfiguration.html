UserGroupResolutionConfiguration

Fornece as informações de configuração para obter usuários e grupos de uma fonte deAWS IAM Identity Center (successor to AWS Single Sign-On) identidade. Isso é útil para filtragem de contexto do usuário, em que os resultados da pesquisa são filtrados com base no acesso do usuário ou do grupo aos documentos. Você também pode usar a PutPrincipalMappingAPI para mapear usuários para seus grupos, de forma que você só precise fornecer o ID do usuário ao emitir a consulta.

Para configurar uma fonte de identidade do IAM Identity Center no console para uso com o Amazon Kendra, consulte Introdução a uma fonte de identidade do IAM Identity Center. Você também deve conceder as permissões necessárias para usar o IAM Identity Center com o Amazon Kendra. Para obter mais informações, consulte Funções do IAM para IAM Identity Center.

Atualmente, o Amazon Kendra não oferece suporte ao usoUserGroupResolutionConfiguration com uma conta de membro daAWS organização para sua fonte de identificação do IAM Identity Center. É necessário criar seu índice na conta de gerenciamento da organização para usarUserGroupResolutionConfiguration.

UserGroupResolutionMode

O provedor de armazenamento de identidades (modo) que você deseja usar para obter usuários e grupos. AWS IAM Identity Center (successor to AWS Single Sign-On)atualmente é o único modo disponível. Seus usuários e grupos devem existir em uma fonte de identidade do IAM Identity Center para usar esse modo.

Tipo: String

Valores válidos: AWS_SSO | NONE

Obrigatório: Sim
