Document

Um documento em um índice.

Índice

AccessControlConfigurationId

O identificador da configuração de controle de acesso que você deseja aplicar ao documento.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 36.

Padrão: [a-zA-Z0-9-]+

Obrigatório: não

AccessControlList

Informações sobre diretores (usuários e/ou grupos) e a quais documentos eles devem ter acesso. Isso é útil para filtragem de contexto do usuário, em que os resultados da pesquisa são filtrados com base no acesso do usuário ou do grupo aos documentos.

Tipo: matriz de objetos Principal

Obrigatório: não

Attributes

Atributos personalizados a serem aplicados ao documento. Use os atributos personalizados para fornecer informações adicionais para pesquisa, fornecer facetas para refinar pesquisas e fornecer informações adicionais na resposta da consulta.

Por exemplo, 'DataSourceId' e 'DataSourceSyncJobId' são atributos personalizados que fornecem informações sobre a sincronização de documentos executados em uma fonte de dados. Observe que 'DataSourceSyncJobId' pode ser um atributo personalizado opcional, pois o Amazon Kendra usará o ID de um trabalho de sincronização em execução.

Tipo: matriz de objetos DocumentAttribute

Obrigatório: não

Blob

O conteúdo do documento.

Os documentos que oBlob parâmetro é codificado em base64. Talvez seu código não precise codificar os bytes do arquivo do documento se você estiver usando umAWS SDK para chamar as APIs do Amazon Kendra. Se você estiver chamando o endpoint Amazon Kendra diretamente usando REST, deverá codificar o conteúdo em base64 antes de enviar.

Tipo: Objeto de dados binários codificado pelo Base64

Obrigatório: não

ContentType

O tipo de arquivo do documento noBlob campo.

Tipo: String

Valores válidos: PDF | HTML | MS_WORD | PLAIN_TEXT | PPT | RTF | XML | XSLT | MS_EXCEL | CSV | JSON | MD

Obrigatório: não

HierarchicalAccessControlList

A lista das principais listas que definem a hierarquia à qual os documentos os usuários devem ter acesso.

Tipo: matriz de objetos HierarchicalPrincipal

Membros da matriz: número mínimo de 1 item. Número máximo de 30 itens.

Obrigatório: não

Id

Um identificador do documento no índice.

Observe que cada ID de documento deve ser exclusiva por índice. Você não pode criar uma fonte de dados para indexar seus documentos com suas IDs exclusivas e usar aBatchPutDocument API para indexar os mesmos documentos, ou vice-versa. Você pode excluir uma fonte de dados e usar aBatchPutDocument API para indexar os mesmos documentos ou vice-versa.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 2048.

Obrigatório: Sim

S3Path

Informações necessárias para encontrar um arquivo específico em um bucket do Amazon S3.

Tipo: objeto S3Path

Obrigatório: não

Title

O título do documento.

Tipo: String

Obrigatório: não
