QueryResultItem

Um único resultado da consulta.

O resultado de uma consulta contém informações sobre um documento retornado pela consulta. Isso inclui a localização original do documento, uma lista de atributos atribuídos ao documento e o texto relevante do documento que satisfaz a consulta.

Índice

AdditionalAttributes

Um ou mais atributos adicionais associados ao resultado da consulta.

Tipo: matriz de objetos AdditionalResultAttribute

Obrigatório: não

DocumentAttributes

Uma matriz de atributos do documento atribuídos a um documento nos resultados da pesquisa. Por exemplo, o autor do documento (_author) ou o URI de origem (_source_uri) do documento.

Tipo: matriz de objetos DocumentAttribute

Obrigatório: não

DocumentExcerpt

Um extrato do texto no documento. Contém informações sobre como destacar os termos relevantes no trecho.

Tipo: objeto TextWithHighlights

Obrigatório: não

DocumentId

O identificador do documento.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 2048.

Obrigatório: não

DocumentTitle

O título do documento. Contém o texto do título e informações para destacar os termos relevantes no título.

Tipo: objeto TextWithHighlights

Obrigatório: não

DocumentURI

O URI do local do documento do documento.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 2048.

Padrão: ^(https?|ftp|file):\/\/([^\s]*)

Obrigatório: não

FeedbackToken

Um token que identifica um resultado específico de uma consulta específica. Use esse token para fornecer feedback sobre o resultado com um clique. Para obter mais informações, consulte Envio de feedback.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 2048.

Padrão: ^\P{C}*.\P{C}*$

Obrigatório: não

Format

Se o documentoType do dentro da resposta forANSWER, então é umaTABLE resposta ou umaTEXT resposta. Se for uma resposta de tabela, um trecho da tabela será retornado emTableExcerpt. Se for uma resposta de texto, um trecho de texto será retornado emDocumentExcerpt.

Tipo: String

Valores válidos: TABLE | TEXT

Obrigatório: não

Id

O identificador do resultado da consulta.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 73.

Obrigatório: não

ScoreAttributes

Indica a confiança que o Amazon Kendra tem de que um resultado corresponde à consulta que você forneceu. Cada resultado é colocado em um compartimento que indica a confiançaVERY_HIGHHIGH,,MEDIUMLOW e. Você pode usar a pontuação para determinar se uma resposta atende à confiança necessária para sua inscrição.

O campo só é definido comoLOW quando oType campo está definido comoDOCUMENT e o Amazon Kendra não tem certeza de que o resultado corresponda à consulta.

Tipo: objeto ScoreAttributes

Obrigatório: não

TableExcerpt

Um trecho de uma tabela dentro de um documento.

Tipo: objeto TableExcerpt

Obrigatório: não

Type

O tipo de documento na resposta. Por exemplo, uma resposta pode incluir uma pergunta-resposta que seja relevante para a consulta.

Tipo: String

Valores válidos: DOCUMENT | QUESTION_ANSWER | ANSWER

Obrigatório: não
