JwtTokenTypeConfiguration

Fornece as informações de configuração para o tipo de token JWT.

ClaimRegex

A expressão regular que identifica a reivindicação.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 100.

Padrão: ^\P{C}*$

Obrigatório: não

GroupAttributeField

O campo de atributo de grupo.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 100.

Padrão: ^\P{C}*$

Obrigatório: não

Issuer

O emissor do token.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 65.

Padrão: ^\P{C}*$

Obrigatório: não

KeyLocation

A localização da chave.

Tipo: String

Valores válidos: URL | SECRET_MANAGER

Obrigatório: Sim

SecretManagerArn

O nome de recurso da Amazon (ARN) do segredo.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1284.

Padrão: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Obrigatório: não

URL

A URL da chave de assinatura.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 2048.

Padrão: ^(https?|ftp|file):\/\/([^\s]*)

Obrigatório: não
