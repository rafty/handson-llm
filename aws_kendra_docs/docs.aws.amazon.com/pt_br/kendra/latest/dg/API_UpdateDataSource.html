Parâmetros de solicitação

Para obter informações sobre os parâmetros que são comuns a todas as ações, consulte Parâmetros comuns.

A solicitação aceita os dados a seguir no formato JSON.

Configuration

Informações de configuração para o conector da fonte de dados.

Tipo: objeto DataSourceConfiguration

Obrigatório: não

CustomDocumentEnrichmentConfiguration

Informações de configuração para alterar os metadados e o conteúdo do documento durante o processo de ingestão de documentos.

Para obter mais informações sobre como criar, modificar e excluir metadados de documentos ou fazer outras alterações de conteúdo ao ingerir documentos no Amazon Kendra, consulte Personalização de metadados de documentos durante o processo de ingestão.

Tipo: objeto CustomDocumentEnrichmentConfiguration

Obrigatório: não

Description

Uma nova descrição para o conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1000.

Padrão: ^\P{C}*$

Obrigatório: não

Id

O identificador do conector da fonte de dados que você deseja atualizar.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 100.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

Obrigatório: Sim

IndexId

O identificador do índice utilizado com o conector da fonte de dados.

Tipo: String

Restride de de de de de de de de de de de de de de de de de de

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*

Obrigatório: Sim

LanguageCode

O código para o conector da fonte de dados. Isso permite que você ofereça suporte a um idioma para todos os documentos ao atualizar a fonte de dados. O inglês é compatível por padrão. Para obter mais informações sobre os idiomas suportados, incluindo seus códigos, consulte Adição de documentos em outros idiomas além do inglês.

Tipo: String

Restride de de de de de de de de de de de de de de de de de de Maximum length of 10.

Padrão: [a-zA-Z-]*

Obrigatório: não

Name

Um novo conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 1000.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

Obrigatório: não

RoleArn

O nome de recurso da Amazon (ARN) de uma função com permissão para acessar a fonte de dados e os recursos necessários. Para obter mais informações, consulte IAM roles for Amazon Kendra (Perfis do IAM para o Amazon Kendra).

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Maximum length of 1284.

Padrão: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Obrigatório: não

Schedule

A programação de sincronização para o conector da fonte de dados.

Tipo: String

Obrigatório: não

VpcConfiguration

Informações de configuração para que uma Amazon Virtual Private Cloud se conecte à sua fonte de dados. Para obter mais informações, consulte Configuring a VPC (Configuração de uma VPC).

Tipo: objeto DataSourceVpcConfiguration

Obrigatório: não
