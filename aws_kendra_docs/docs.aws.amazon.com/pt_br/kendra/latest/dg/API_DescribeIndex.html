DescribeIndex

Obtém informações sobre um índice do Amazon Kendra.

Sintaxe da solicitação

{
   "Id": "string"
}

Parâmetros de solicitação

Para obter informações sobre os parâmetros que são comuns a todas as ações, consulte Parâmetros comuns.

A solicitação aceita os dados a seguir no formato JSON.

Id

O identificador do índice sobre o qual você deseja obter informações.

Tipo: String

Restrições de comprimento: comprimento fixo de 36.

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*

Obrigatório: Sim

Elementos de resposta

Se a ação for bem-sucedida, o serviço reenviará uma resposta HTTP 200.

Os dados a seguir são retornados no formato JSON pelo serviço.

CapacityUnits

Para índices da Enterprise Edition, você pode optar por usar capacidade adicional para atender às necessidades do seu aplicativo. Ele contém as unidades de capacidade usadas para o índice. Uma capacidade zero de armazenamento de consultas ou documentos indica que o índice está usando a capacidade padrão. Para obter mais informações sobre a capacidade padrão de um índice e como ajustá-la, consulte Ajuste da capacidade.

Tipo: objeto CapacityUnitsConfiguration

CreatedAt

O timestamp de criação do índice.

Tipo: Timestamp

Description

A descrição do índice.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1000.

Padrão: ^\P{C}*$

DocumentMetadataConfigurations

Informações de configuração para metadados ou campos do documento. Os metadados do documento são campos ou atributos associados aos seus documentos. Por exemplo, o nome do departamento da empresa associado a cada documento.

Tipo: matriz de objetos DocumentMetadataConfiguration

Membros da matriz: número mínimo de 0 itens. Número máximo de 500 itens.

Edition

A edição Amazon Kendra usada para o índice. Você decide a edição ao criar o índice.

Tipo: String

Valores válidos: DEVELOPER_EDITION | ENTERPRISE_EDITION

ErrorMessage

Quando o valor doStatus campo éFAILED, oErrorMessage campo contém uma mensagem que explica o porquê.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 2048.

Padrão: ^\P{C}*$

Id

O identificador do índice.

Tipo: String

Restrições de comprimento: comprimento fixo de 36.

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*

IndexStatistics

Fornece informações sobre o número de perguntas e respostas do FAQ e o número de documentos de texto indexados.

Tipo: objeto IndexStatistics

Name

O nome do índice.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 1000.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

RoleArn

O nome de recurso da Amazon (ARN) da função do IAM que dá permissão para Amazon (ARN) da função do IAM que dá permissão para Amazon Kendra função do IAM que dá ao Amazon (ARN) da função do IAM que dá ao Amazon (ARN

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1284.

Padrão: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

ServerSideEncryptionConfiguration

O identificador da chave mestra mestra mestra doAWS KMS cliente (CMK) que é usada para criptografar seus dados. O Amazon Kendra não oferece suporte a CMKs assimétricas.

Tipo: objeto ServerSideEncryptionConfiguration

Status

O status atual do índice. Quando o valor éACTIVE, o índice está pronto para uso. Se o valor doStatus campo forFAILED, oErrorMessage campo conterá uma mensagem que explica o porquê.

Tipo: String

Valores válidos: CREATING | ACTIVE | DELETING | FAILED | UPDATING | SYSTEM_UPDATING

UpdatedAt

O Unix quando o índice foi atualizado pela última vez.

Tipo: Timestamp

UserContextPolicy

A política de contexto do usuário para o índice Amazon Kendra.

Tipo: String

Valores válidos: ATTRIBUTE_FILTER | USER_TOKEN

UserGroupResolutionConfiguration

Se você ativou a configuração para buscar níveis de acesso de grupos e usuários de uma fonte deAWS IAM Identity Center (successor to AWS Single Sign-On) identidade.

Tipo: objeto UserGroupResolutionConfiguration

UserTokenConfigurations

A configuração do token do usuário para o índice Amazon Kendra.

Tipo: matriz de objetos UserTokenConfiguration

Membros da matriz: número máximo de 1 item.

Erros

Para obter informações sobre os erros comuns a todas as ações, consulteErros comuns.

AccessDeniedException

Você não tem acesso suficiente para realizar esta ação. Certifique-se de ter as políticas de permissão e contas de usuário necessárias e tente novamente.

Código de status HTTP: 400

InternalServerException

Ocorreu um problema com o servidor interno usado para seu serviço Amazon Kendra. Aguarde alguns minutos e tente novamente ou entre em contato com o Support para obter ajuda.

Código de status HTTP: 500

ResourceNotFoundException

O recurso que você deseja usar não existe. Verifique se você forneceu o recurso correto e tente novamente.

Código de status HTTP: 400

ThrottlingException

A solicitação foi negada devido à limitação da solicitação. Reduza o número de solicitações e tente novamente.

Código de status HTTP: 400

ValidationException

A entrada não atende às restrições definidas pelo serviço do Amazon Kendra. Forneça a entrada correta e tente novamente.

Código de status HTTP: 400
