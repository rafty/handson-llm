CreateDataSource

Cria um conector de fonte de dados para ser usado com um índice do Amazon Kendra.

Você especifica um nome, um tipo de conector de fonte de dados e uma descrição para a fonte de dados. Você também especifica informações de configuração para o conector da fonte de dados.

CreateDataSourceé uma operação síncrona. A operação retornará 200 se a fonte de dados tiver sido criada com êxito. Caso contrário, uma exceção será gerada.

Para ver um exemplo de como criar um índice e uma fonte de dados usando o SDK do Python, consulte Introdução ao SDK do Python. Para ver um exemplo de como criar um índice e uma fonte de dados usando o Java SDK, consulte Introdução ao Java SDK.

Parâmetros de solicitação

Para obter informações sobre os parâmetros que são comuns a todas as ações, consulte Parâmetros comuns.

A solicitação aceita os dados a seguir no formato JSON.

ClientToken

Um token que você fornece para identificar a solicitação para criar um conector da fonte de dados. Várias chamadas para aCreateDataSource API com o mesmo token de cliente criarão somente um conector de fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 100.

Obrigatório: não

Configuration

Informações de configuração para se conectar ao seu repositório de fontes de dados.

Não é possível especificar o parâmetro Configuration quando o parâmetro Type estiver definido como CUSTOM. Se você fizer isso, você receberá umaValidationException exceção.

O parâmetro Configuration é necessário para todas as outras fontes de dados.

Tipo: objeto DataSourceConfiguration

Obrigatório: não

CustomDocumentEnrichmentConfiguration

Informações de configuração para alterar os metadados e o conteúdo do documento durante o processo de ingestão de documentos.

Para obter mais informações sobre como criar, modificar e excluir metadados de documentos ou fazer outras alterações de conteúdo ao ingerir documentos no Amazon Kendra, consulte Personalização de metadados de documentos durante o processo de ingestão.

Tipo: objeto CustomDocumentEnrichmentConfiguration

Obrigatório: não

Description

Uma descrição para o conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1000.

Padrão: ^\P{C}*$

Obrigatório: não

IndexId

O identificador do índice que você deseja usar com o conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho fixo 36.

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*

Obrigatório: Sim

LanguageCode

O código de um idioma. Isso permite que você ofereça suporte a um idioma para todos os documentos ao criar o conector da fonte de dados. O inglês é compatível por padrão. Para obter mais informações sobre os idiomas suportados, incluindo seus códigos, consulte Adição de documentos em outros idiomas além do inglês.

Tipo: String

Restrições de tamanho: tamanho mínimo 2. Tamanho máximo 10.

Padrão: [a-zA-Z-]*

Obrigatório: não

Name

Um nome para o conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 1000.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

Obrigatório: Sim

RoleArn

O nome de recurso da Amazon (ARN) de uma função do IAM com permissão para acessar a fonte de dados e os recursos necessários. Para obter mais informações, consulte Funções de acesso do IAM para Amazon Kendra. .

Não é possível especificar o parâmetro RoleArn quando o parâmetro Type estiver definido como CUSTOM. Se você fizer isso, você receberá umaValidationException exceção.

O parâmetro RoleArn é necessário para todas as outras fontes de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo 1284.

Padrão: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Obrigatório: não

Schedule

Define a frequência com que o Amazon Kendra verifica os documentos em seu repositório da fonte de dados e atualiza o índice. Se você não definir uma programação, o Amazon Kendra não atualizará periodicamente o índice. Você pode chamar aStartDataSourceSyncJob API para atualizar o índice.

Especifique uma sequência de caracteres de programação decron- formato ou uma string vazia para indicar que o índice é atualizado sob demanda.

Não é possível especificar o parâmetro Schedule quando o parâmetro Type estiver definido como CUSTOM. Se você fizer isso, você receberá umaValidationException exceção.

Tipo: String

Obrigatório: não

Tags

Uma lista de pares chave-valor que identificam ou categorizam o conector da fonte de dados. Você também pode usar tags para ajudar a controlar o acesso ao conector da fonte de dados. As chaves e os valores de etiqueta podem consistir em letras, dígitos, espaços em branco e qualquer um dos seguintes símbolos: _.:/= - @.

Tipo: matriz de objetos Tag

Membros da matriz: número mínimo de 0 itens. Número máximo 200 itens.

Obrigatório: não

Type

O tipo do repositório da fonte de dados. Por exemplo, SHAREPOINT.

Tipo: String

Valores válidos: S3 | SHAREPOINT | DATABASE | SALESFORCE | ONEDRIVE | SERVICENOW | CUSTOM | CONFLUENCE | GOOGLEDRIVE | WEBCRAWLER | WORKDOCS | FSX | SLACK | BOX | QUIP | JIRA | GITHUB | ALFRESCO | TEMPLATE

Obrigatório: Sim

VpcConfiguration

Informações de configuração para que uma Amazon Virtual Private Cloud se conecte à sua fonte de dados. Para obter mais informações, consulte Configuring a VPC (Configuração de uma VPC).

Tipo: objeto DataSourceVpcConfiguration

Obrigatório: não

{

Elementos de resposta
