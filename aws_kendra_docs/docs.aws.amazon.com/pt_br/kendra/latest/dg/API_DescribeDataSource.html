DescribeDataSource

Obtém informações sobre um conector de fonte de dados do Amazon Kendra.

Sintaxe da solicitação

{
   "Id": "string",
   "IndexId": "string"
}

Parâmetros de solicitação

Para obter informações sobre os parâmetros que são comuns a todas as ações, consulte Parâmetros comuns.

A solicitação aceita os dados a seguir no formato JSON.

Id

O identificador do conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 100.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

Obrigatório: Sim

IndexId

O identificador do índice usado com o conector da fonte de dados.

Tipo: String

Restrições de comprimento: comprimento fixo de 36.

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*

Obrigatório: Sim

Elementos de resposta

Se a ação for bem-sucedida, o serviço reenviará uma resposta HTTP 200.

Os dados a seguir são retornados no formato JSON pelo serviço.

Configuration

Detalhes de configuração para o conector da fonte de dados. Isso mostra como a fonte de dados está configurada. As opções de configuração de uma fonte de dados dependem do provedor da fonte de dados.

Tipo: objeto DataSourceConfiguration

CreatedAt

A data e hora do Unix quando o conector da fonte de dados foi criado.

Tipo: Timestamp

CustomDocumentEnrichmentConfiguration

Informações de configuração para alterar os metadados e o conteúdo do documento durante o processo de ingestão de dados.

Para obter mais informações sobre como criar, modificar e excluir metadados de documentos ou fazer outras alterações de conteúdo ao ingerir documentos no Amazon Kendra, consulte Personalização de metadados de documentos durante o processo de ingestão.

Tipo: objeto CustomDocumentEnrichmentConfiguration

Description

A descrição para o conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1000.

Padrão: ^\P{C}*$

ErrorMessage

Quando o valor doStatus campo éFAILED, oErrorMessage campo contém uma descrição do erro que causou a falha na fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 2048.

Padrão: ^\P{C}*$

Id

O identificador do conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 100.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

IndexId

O identificador do índice usado com o conector da fonte de dados.

Tipo: String

Restrições de comprimento: comprimento fixo de 36.

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*

LanguageCode

O código de um idioma. Isso mostra um idioma compatível com todos os documentos na fonte de dados. O inglês é compatível por padrão. Para obter mais informações sobre os idiomas suportados, incluindo seus códigos, consulte Adição de documentos em outros idiomas além do inglês.

Tipo: String

Restrições de comprimento: comprimento mínimo de 2. Tamanho máximo de 10.

Padrão: [a-zA-Z-]*

Name

O nome para o conector da fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 1000.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

RoleArn

O nome do recurso da Amazon (ARN) da função com permissão para acessar a fonte de dados e os recursos necessários.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1284.

Padrão: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Schedule

O cronograma para o Amazon Kendra atualizar o índice.

Tipo: String

Status

O status atual do conector da fonte de dados. Quando o status for,ACTIVE a fonte de dados estará pronta para ser usada. Quando o status éFAILED, oErrorMessage campo contém o motivo pelo qual a fonte de dados falhou.

Tipo: String

Valores válidos: CREATING | DELETING | FAILED | UPDATING | ACTIVE

Type

O tipo da fonte de dados. Por exemplo, SHAREPOINT.

Tipo: String

Valores válidos: S3 | SHAREPOINT | DATABASE | SALESFORCE | ONEDRIVE | SERVICENOW | CUSTOM | CONFLUENCE | GOOGLEDRIVE | WEBCRAWLER | WORKDOCS | FSX | SLACK | BOX | QUIP | JIRA | GITHUB | ALFRESCO | TEMPLATE

UpdatedAt

O carimbo de data/hora do Unix de última atualização do conector da fonte de dados.

Tipo: Timestamp

VpcConfiguration

Informações de configuração para que uma Amazon Virtual Private Cloud se conecte à sua fonte de dados. Para obter mais informações, consulte Configuring a VPC (Configuração de uma VPC).

Tipo: objeto DataSourceVpcConfiguration

Erros

Para obter informações sobre os erros comuns a todas as ações, consulteErros comuns.

AccessDeniedException

Você não tem acesso suficiente para realizar esta ação. Certifique-se de ter as políticas de permissão e contas de usuário necessárias e tente novamente.

Código de status HTTP: 400

InternalServerException

Ocorreu um problema com o servidor interno usado para seu serviço Amazon Kendra. Aguarde alguns minutos e tente novamente ou entre em contato com o Support para obter ajuda.
