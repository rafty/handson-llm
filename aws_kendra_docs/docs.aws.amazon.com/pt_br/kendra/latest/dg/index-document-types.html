Tipos de documento do

Um índice pode conter vários documentos e vários tipos de documentos. Um índice também pode incluir texto estruturado e não estruturado.

Texto estruturado

  • Perguntas e respostas frequentes

Documentos em formato de perguntas e respostas frequentes são usados para responder perguntas como: Qual é a altura do Space Needle? Você pode especificar várias perguntas que retornem a mesma resposta. Você especifica as perguntas e respostas em um arquivo de valores separados por vírgula (CSV) armazenado em umAmazon S3 bucket.

Se você quiser usar um arquivo JSON para seu conjunto de perguntas e respostas frequentes, os arquivos JSON são tratados como texto simples e são um tipo de texto não estruturado.

Para ver um exemplo, consulte Adicionar perguntas e respostas diretamente a um índice.

Texto não estruturado

Amazon Kendrasuporta os seguintes tipos de texto não estruturado:

  • Arquivos HTML

  • Apresentações da Microsoft PowerPoint (PPT)

  • Documentos do MS WORD

  • Documentos de texto simples

  • PDFs

  • Arquivos de valores separados por vírgula (CSV)

  • Arquivos do Microsoft Excel (MS EXCEL)

  • Arquivos XML

  • Arquivos JSON

  • Arquivos de documentação Markdown (MD)

  • Arquivos RTF (Rich Text Format)

  • Arquivos Extensible Stylesheet Language Transformation (XSLT)

Você pode adicionar documentos não estruturados ao seu índice de três maneiras:

  • Usando a BatchPutDocumentAPI.

  • Como fonte de dados. Para obter mais informações, consulte Adicionar documentos de uma fonte de dados.

  • Como dados binários. Para obter mais informações, consulte Adicionar documentos com a API.
