Tag

Uma lista de pares de chave/valor que identificam um índice, perguntas frequentes ou fonte de dados. As chaves e valores de etiqueta podem consistir em letras, os valores de etiqueta podem incluir letras, espaços em branco e qualquer um dos seguintes símbolos: _.:/= + - @.

Índice

Key

A chave. As chaves não diferenciam maiúsculas de minúsculas e devem ser exclusivas para o índice, as perguntas frequentes ou a fonte de dados.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 128.

Obrigatório: Sim

Value

O valor da. O valor pode ser uma string vazia, mas não pode ser nulo.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 256.

Obrigatório: Sim
