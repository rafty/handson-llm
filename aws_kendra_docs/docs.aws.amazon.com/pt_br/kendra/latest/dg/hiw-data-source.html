Fontes de dados

Uma fonte de dados é um repositório ou local de dados queAmazon Kendra se conecta e indexa seus documentos ou conteúdo. Por exemplo, você pode configurarAmazon Kendra a conexão com a Microsoft SharePoint para rastrear e indexar seus documentos armazenados nessa fonte. Você também pode indexar páginas da Web fornecendo os URLsAmazon Kendra para rastreamento. Você pode sincronizar automaticamente uma fonte de dados com umAmazon Kendra índice para que documentos adicionados, atualizados ou excluídos na fonte de dados também sejam adicionados, atualizados ou excluídos no índice.

nota

Para criar um índice, você não precisa de uma fonte de dados. Você pode adicionar documentos diretamente a um índice. Para obter mais informações, consulte Adicionar documentos diretamente a um índice.

Para indexar documentos usando uma fonte de dados.

  1. Crie um índice.

  2. Crie uma fonte de dados.
