Salesforce

O Salesforce é uma ferramenta de gerenciamento de relacionamento com o cliente (CRM) para gerenciar equipes de suporte, vendas e marketing. Você pode usarAmazon Kendra para indexar seus objetos padrão do Salesforce e até mesmo objetos personalizados.

Você pode se conectarAmazon Kendra à sua fonte de dados do Salesforce usando o Amazon Kendraconsole, a TemplateConfigurationAPI ou a SalesforceConfigurationAPI.

Amazon Kendratem duas versões do conector Salesforce. Os recursos compatíveis de cada versão incluem:

Conector Salesforce V1.0/SalesforceConfigurationAPI

  • Mapeamentos de campos

  • Filtragem de contexto do usuário

  • Filtros de inclusão/exclusão

Conector Salesforce V2.0/TemplateConfigurationAPI

  • Mapeamentos de campos

  • Filtragem de contexto do usuário

  • Filtros de inclusão/exclusão

  • Suporte para anexação de entidades

  • Rastreamento de identidade

  • Suporte à VPC

  • Sincronizar todos os documentos/Sincronizar somente documentos novos, modificados ou excluídos

nota

O Support para o conector Salesforce SalesforceConfiguration V1.0/API está programado para terminar em junho de 2023. Recomendamos usar o conector Salesforce V2.0/ TemplateConfiguration API.

Para solucionar problemas com seu conector de fonte de dados doAmazon Kendra Salesforce, consulteSolução de problemas de origens de dados.
