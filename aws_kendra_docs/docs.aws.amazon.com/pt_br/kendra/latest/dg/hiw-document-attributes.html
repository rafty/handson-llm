Atributos ou campos do documento

Um documento tem atributos ou campos associados a ele. Os campos de um documento são as propriedades de um documento ou o que está contido na estrutura de um documento. Por exemplo, cada um dos seus documentos pode conter título, corpo do texto e autor. Você também pode adicionar seus próprios campos personalizados aos seus documentos. Campos personalizados são atributos que você especifica para suas próprias necessidades. Por exemplo, se seu índice pesquisar documentos fiscais, você poderá especificar um campo personalizado para o tipo de documento fiscal, como W-2, 1099 e assim por diante.

Antes de usar um campo de documento em uma consulta, ele deve ser mapeado para um campo de índice. Por exemplo, o campo do título pode ser mapeado para o campo_document_title. Para obter mais informações, consulte Mapear campos. Para adicionar um novo campo, você deve criar um campo de índice para o qual mapear o campo. Você cria campos de índice usando o console ou usando a UpdateIndexAPI do.

Você pode usar campos de documentos para filtrar respostas e fazer sugestões de pesquisa facetadas. Por exemplo, você pode filtrar uma resposta para retornar somente uma versão específica de um documento ou filtrar pesquisas para retornar somente documentos fiscais do tipo 1099 que correspondam ao termo de pesquisa. Para obter mais informações, consulte Filtrar consultas.

Você também pode usar campos de documentos para ajustar manualmente a resposta da consulta. Por exemplo, você pode optar por aumentar a importância do campo de título para aumentar o pesoAmazon Kendra atribuído ao campo ao determinar quais documentos retornarão na resposta. Para obter mais informações, consulte Ajustar a relevância da pesquisa.

Se você estiver adicionando um documento diretamente a um índice, especifique os campos no parâmetro de entrada do documento para aBatchPutDocument API. Você especifica os valores do campo personalizado em uma matriz deDocumentAttribute objetos. Se você estiver usando uma fonte de dados, o método usado para adicionar os campos do documento depende da fonte de dados. Para obter mais informações, consulte Criação de campos de documentos personalizados.
