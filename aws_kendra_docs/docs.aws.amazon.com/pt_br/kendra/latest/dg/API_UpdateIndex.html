Parâmetros de solicitação

Para obter informações sobre os parâmetros que são comuns a todas as ações, consulte Parâmetros comuns.

A solicitação aceita os dados a seguir no formato JSON.

CapacityUnits

Define o número de unidades adicionais de armazenamento de documentos e capacidade de consulta que devem ser usadas pelo índice. Você pode alterar a capacidade do índice até 5 vezes por dia ou fazer 5 chamadas de API.

Se você estiver usando unidades de armazenamento extras, não poderá reduzir a capacidade de armazenamento abaixo do necessário para atender às necessidades de armazenamento do seu índice.

Tipo: objeto CapacityUnitsConfiguration

Obrigatório: não

Description

Uma nova descrição para o índice.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1000.

Padrão: ^\P{C}*$

Obrigatório: não

DocumentMetadataConfigurationUpdates

A configuração de metadados do documento que você deseja atualizar para o índice. Os metadados do documento são campos ou atributos associados aos seus documentos. Por exemplo, o nome do departamento da empresa associado a cada documento.

Tipo: matriz de objetos DocumentMetadataConfiguration

Membros da matriz: número mínimo de 0 itens. Número máximo de 500 itens.

Obrigatório: não

Id

O identificador do índice que você deseja atualizar.

Tipo: String

Restrições de comprimento: comprimento fixo de 36.

Padrão: [a-zA-Z0-9][a-zA-Z0-9-]*

Obrigatório: Sim

Name

O nome do índice que você deseja atualizar.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo de 1000.

Padrão: [a-zA-Z0-9][a-zA-Z0-9_-]*

Obrigatório: não

RoleArn

Uma funçãoAWS Identity and Access Management (IAM) que concede ao Amazon Kendra permissões para acessar CloudWatch logs e métricas da Amazon.

Tipo: String

Restrições de tamanho: tamanho mínimo 0. Tamanho máximo de 1284.

Padrão: arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Obrigatório: não

UserContextPolicy

A política de contexto do usuário.

Tipo: String

Valores válidos: ATTRIBUTE_FILTER | USER_TOKEN

Obrigatório: não

UserGroupResolutionConfiguration

Permite obter níveis de acesso de grupos e usuários de uma fonte deAWS IAM Identity Center (successor to AWS Single Sign-On) identidade. Para configurar isso, consulte UserGroupResolutionConfiguration.

Tipo: objeto UserGroupResolutionConfiguration

Obrigatório: não

UserTokenConfigurations

A configuração do token do usuário.

Tipo: matriz de objetos UserTokenConfiguration

Membros da matriz: número máximo de 1 item.

Obrigatório: não

Elementos de resposta

Se a ação for bem-sucedida, o serviço reenviará uma resposta 200 HTTP com um corpo HTTP vazio.

Erros

Para obter informações sobre os erros comuns a todas as ações, consulteErros comuns.

AccessDeniedException

Você não tem acesso suficiente para realizar esta ação. Verifique se você tem as políticas de permissão e contas de usuário necessárias e tente novamente.

Código de status HTTP: 400

ConflictException

Ocorreu um conflito com a solicitação. Corrija quaisquer inconsistências com seus recursos e tente novamente.

Código de status HTTP: 400

InternalServerException

Ocorreu um problema com o servidor interno usado para seu serviço Amazon Kendra. Aguarde alguns minutos e tente novamente ou entre em contato com o Support para obter ajuda.

Código de status HTTP: 500

ResourceNotFoundException

O recurso que você deseja usar não existe. Verifique se você forneceu o recurso correto e tente novamente.

Código de status HTTP: 400

ServiceQuotaExceededException

Você excedeu os limites estabelecidos para seu serviço Amazon Kendra. Consulte Cotas para obter mais informações ou entre em contato com o Support para obter informações sobre um aumento de limites.

Código de status HTTP: 400

ThrottlingException

A solicitação foi negada devido à limitação da solicitação. Reduza o número de solicitações e tente novamente.

Código de status HTTP: 400

ValidationException

A entrada não atende às restrições definidas pelo serviço Amazon Kendra. Forneça a entrada correta e tente novamente.

Código de status HTTP: 400
