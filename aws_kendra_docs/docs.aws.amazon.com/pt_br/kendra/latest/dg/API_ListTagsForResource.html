ListTagsForResource

Obtém uma lista lista das tags tags associadas a um recurso específico. Índices, perguntas frequentes e fontes de dados podem ter tags associadas a eles.

Sintaxe da solicitação

{
   "ResourceARN": "string"
}

Parâmetros de solicitação

Para obter informações sobre os parâmetros que são comuns a todas as ações, consulte Parâmetros comuns.

A solicitação aceita os dados a seguir no formato JSON.

ResourceARN

O nome do recurso da Amazon (ARN) do índice, perguntas frequentes ou fonte de dados no qual você deseja que a Amazon Resource Name (ARN) do índice, perguntas frequentes ou fonte de tags tags tags tags tags tags tags tags tags tags.

Tipo: String

Restrições de tamanho: tamanho mínimo 1. Tamanho máximo 1011.

Obrigatório: Sim

Sintaxe da resposta

{
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

Elementos de resposta

Se a ação for bem-sucedida, o serviço reenviará uma resposta HTTP 200.

Os dados a seguir são retornados no formato JSON pelo serviço.

Tags

Uma lista lista tags tags tags tags associadas ao índice, perguntas frequentes ou fonte dados.

Tipo: matriz de objetos Tag

Membros da matriz: número mínimo de 0 itens. Número máximo 200 itens.
