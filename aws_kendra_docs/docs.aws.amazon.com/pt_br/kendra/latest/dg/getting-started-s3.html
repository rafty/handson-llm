Introdução a uma fonteAmazon S3 de dados (console)

Você pode usar oAmazon Kendra console para começar a usar umAmazon S3 bucket como armazenamento de dados. Ao usar o console, você especifica todas as informações de conexão necessárias para indexar o conteúdo do bucket. Para obter mais informações, consulteAmazon S3

Use o procedimento a seguir para criar uma fonte de dados básica do bucket do S3 usando a configuração padrão. O procedimento pressupõe que você criou um índice seguindo as etapas na etapa 1 doConceitosAmazon Kendra.

Para criar uma fonte de dados do bucket do S3 usando oAmazon Kendra console

  1. Faça loginAWS Management Console e abra oAmazon Kendra console em https://console.aws.amazon.com/kendra/home.

  2. Na lista de índices, escolha o índice ao qual você deseja adicionar a fonte de dados.

  3. Escolha Adicionar fontes de dados.

  4. Na lista de conectores de fonte de dados, escolha Amazon S3.

  5. Na página Definir atributos, dê um nome à fonte de dados e, opcionalmente, uma descrição. Deixe o campo Tags em branco. Escolha Próximo para continuar.

  6. No campo Inserir o local da fonte de dados, digite o nome do bucket do S3 que contém seus documentos. Você pode inserir o nome diretamente ou procurar o nome escolhendo Procurar. O bucket deve estar na mesma região da que o índice.

  7. Na IAMfunção, escolha Criar uma nova função e, em seguida, digite o nome da função. Para obter mais informações, consulte IAMfunções para fontesAmazon S3 de dados.

  8. Na seção Definir cronograma de execução de sincronização, escolha Executar sob demanda.

  9. Escolha Próximo para continuar.

  10. Na página Revisar e criar, revise os detalhes da sua fonte de dados do S3. Se você quiser fazer alterações, escolha o botão Editar ao lado do item que você deseja alterar. Quando estiver satisfeito com suas escolhas, escolha Criar para criar sua fonte de dados do S3.

Depois de escolher Criar,Amazon Kendra começa a criar a fonte de dados. Pode levar vários minutos para o criar a fonte de dados. Ao terminar, o status da fonte de dados muda de Criando para Ativa.

Depois de criar a fonte de dados, você precisa sincronizar oAmazon Kendra índice com a fonte de dados. Escolha Sincronizar agora para iniciar o processo de sincronização. A sincronização da fonte de dados pode levar de alguns minutos a várias horas, dependendo do número e do tamanho dos documentos.
