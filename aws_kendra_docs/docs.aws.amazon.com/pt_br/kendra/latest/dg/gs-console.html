ConceitosAmazon Kendra

Os procedimentos a seguir mostram como criar e testar umAmazon Kendra índice usando oAWS console. Nos procedimentos, você cria um índice e uma fonte de dados para um índice. Por fim, você testa seu índice fazendo uma solicitação de pesquisa.

Etapa 1: Para criar um índice (console)

  1. Faça login no ConsoleAWS de Gerenciamento e abra oAmazon Kendra console em https://console.aws.amazon.com/kendra/.

  2. Selecione Criar índice na seção Índices.

  3. Na página Especificar detalhes do índice, dê um nome e uma descrição ao índice.

  4. Em IAMfunção, escolha Criar uma nova função e, em seguida, dê um nome à função. AIAM função terá o prefixo "AmazonKendra-”.

  5. Deixe todos os outros campos em seus padrões. Escolha Próximo.

  6. Na página Configurar controle de acesso do usuário, escolha Avançar.

  7. Na página de detalhes do provisionamento, escolha Edição para desenvolvedores.

  8. Escolha Criar para criar seu índice.

  9. Aguarde até que seu índice seja criado. Amazon Kendraprovisiona o hardware para seu índice. Essa operação pode levar algum tempo.

Etapa 2: Para adicionar uma fonte de dados a um índice (console)

  1. Veja as fontes de dados disponíveis para seAmazon Kendra conectar e indexar seus documentos.

  2. No painel de navegação, selecione Fontes de dados e selecione Adicionar fonte de dados para a fonte de dados escolhida.

  3. Siga as etapas para configurar a fonte de dados.

Etapa 3: Para pesquisar um índice (console)

  1. No painel de navegação, escolha a opção para pesquisar em seu índice.

  2. Insira um termo de pesquisa que seja apropriado para seu índice. Os principais resultados e os principais resultados do documento são exibidos.
