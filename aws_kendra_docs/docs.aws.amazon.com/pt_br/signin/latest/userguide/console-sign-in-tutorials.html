Faça login no AWS Management Console

Ao fazer login a AWS Management Console partir do URL de AWS login principal (https://signin.aws.amazon.com), você deve escolher seu tipo de usuário, usuário raiz ou usuário do IAM.

O usuário root tem acesso irrestrito à conta e está associado à pessoa que criou o. Conta da AWS Em seguida, o usuário raiz cria outros tipos de usuários, como usuários do IAM e usuários no Centro de Identidade do AWS IAM (sucessor do AWS Single Sign-On) atribui a eles credenciais de acesso.

Um usuário do IAM é uma identidade dentro da sua Conta da AWS com permissões personalizadas específicas. Quando um usuário do IAM faz login, ele pode usar um URL de login que inclui seu alias Conta da AWS ou seu alias, como, por exemplo, https://account_alias_or_id.signin.aws.amazon.com/console/ em vez do URL de AWS login principal. https://signin.aws.amazon.com/

Se você não tiver certeza de que tipo de usuário você é, consulteTipos de usuário.
