Como fazer login em AWS

A maneira como você fizer login AWS depende do seu tipo de AWS usuário. Há tipos diferentes de AWS usuários. Você pode ser um usuário raiz da conta, um usuário do IAM, um usuário no Centro de Identidade do IAM, uma identidade federada ou usar o AWS Builder ID. Para obter mais informações, consulte Tipos de usuário.

Você pode acessar AWS fazendo login com qualquer um dos seguintes métodos:

  • Faça login no AWS Management Consolecomo usuário raiz ou usuário do IAM

  • Faça login no portal AWS de acessocomo usuário do IAM Identity Center

  • Faça login como uma identidade federada

  • Faça login através do AWS Command Line Interfacee outros métodos programáticos como API e SDK (Software Development Kit)

  • Faça login com o AWS Builder ID

Se você quiser criar um novoConta da AWS, consulte a Parte 1: Configurar um novo Conta da AWS no Guia AWS de configuração.
