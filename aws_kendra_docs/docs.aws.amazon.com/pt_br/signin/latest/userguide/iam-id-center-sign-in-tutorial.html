Faça login no portal AWS de acesso

Um usuário no IAM Identity Center é membro doAWS Organizations. Um usuário no IAM Identity Center pode acessar vários Contas da AWS aplicativos de negócios fazendo login no portal de AWS acesso com um URL de login específico. Para obter mais informações sobre o URL de login específico, consulte. AWSportal de acesso

Antes de fazer login em uma Conta da AWS como usuário no IAM Identity Center, reúna as informações necessárias a seguir.

  • Nome de usuário corporativo

  • Senha corporativa

  • URL de login específico

nota

Depois de fazer login, sua sessão do portal de AWS acesso é válida por 8 horas. É necessário fazer login novamente após 8 horas.

Para entrar no portal de AWS acesso

  1. Na janela do navegador, cole a URL de login que você recebeu por e-mail, como. https://your_subdomain.awsapps.com/start Depois, pressione Enter.

  2. Faça login usando suas credenciais corporativas (como nome de usuário e senha).

    nota

    Se o administrador lhe enviou uma senha de uso único (OTP) por e-mail e essa é a primeira vez que você faz login, insira essa senha. Depois de fazer login, você deve criar uma nova senha para future logins.

  3. Se for solicitado um código de verificação, verifique seu e-mail. Em seguida, copie e cole o código na página de login.

    nota

    Normalmente, os códigos de verificação são enviados por e-mail, mas o método de entrega pode variar. Se não tiver recebido um em seu e-mail, consulte seu administrador para obter detalhes sobre seu código de verificação.

  4. Se o MFA estiver habilitado para seu usuário no IAM Identity Center, você se autentica usando ele.

  5. Após a autenticação, você pode acessar qualquer Conta da AWS aplicativo que apareça no portal.

    1. Para entrar no, AWS Management Console selecione o Conta da AWS bloco e, em seguida, selecione a conta individual a ser gerenciada.

      As permissões que seu usuário tem são exibidas junto com links para abrir o console de gerenciamento ou para obter credenciais para acesso programático ou por linha de comando.

      Selecione o link do console de gerenciamento.

    2. Para entrar em um aplicativo, escolha o ícone do aplicativo que você deseja acessar.

O login como usuário no IAM Identity Center fornece credenciais para acessar recursos por um período de tempo definido, chamado de sessão. Por padrão, um usuário pode se conectar a um Conta da AWS por uma hora. O valor máximo que o administrador pode definir para uma sessão é 12 horas. Depois que sua sessão terminar, você poderá entrar novamente.
