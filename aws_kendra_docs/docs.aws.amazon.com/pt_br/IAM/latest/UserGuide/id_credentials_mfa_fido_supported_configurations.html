Configurações compatíveis com o uso de chaves de segurança FIDO

Você pode usar o padrão WebAuthn como um método de autenticação multifator (MFA) no IAM usando as configurações compatíveis. Isso inclui dispositivos certificados pela FIDO compatíveis com o IAM e navegadores compatíveis com WebAuthn.

Dispositivos de FIDO compatíveis com a AWS

Atualmente, o IAM oferece suporte a dispositivos de segurança certificados pela FIDO que se conectam a seus dispositivos por USB, Bluetooth ou NFC. Atualmente, não há compatibilidade com os autenticadores de plataforma, como TouchID, FaceID nem Windows Hello.

nota

A AWS requer acesso à porta USB física em seu computador para verificar seu dispositivo certificado pela FIDO. A MFA WebAuthn não funcionará com uma máquina virtual, uma conexão remota ou o modo incogno de um navegador.

A FIDO Alliance conta com uma lista de todos os produtos certificados pela FIDO compatíveis com as especificações da FIDO.

Navegadores compatíveis com WebAuthn

Os seguintes navegadores são compatíveis com o uso de chaves de segurança certificados pela FIDO:

macOS 10.15+ Windows 10 Linux
Chrome Sim Sim Sim
Safari Sim Não Não
Borda Sim Sim Sim
Firefox Sim Sim Sim
nota

Atualmente, a maioria das versões do Firefox compatíveis com WebAuthn não habilita a compatibilidade por padrão. Para obter instruções sobre como habilitar a compatibilidade com WebAuthn no Firefox, consulte Solução de problemas de chaves de segurança FIDO.

Plug-ins de navegador

No momento, a AWS só é compatível com navegadores que têm compatibilidade nativa com o padrão FIDO2/WebAuthn. A AWS não é compatível com o uso de plug-ins para adicionar compatibilidade do navegador com FIDO2/WebAuthn. Observe também que alguns plug-ins de navegador são incompatíveis com o padrão U2F e podem causar resultados inesperados com chaves de segurança FIDO2.

Para obter informações sobre como desabilitar plugins do navegador e outras dicas de solução de problemas, consulte Não consigo habilitar minha chave de segurança FIDO.

Ambientes móveis

Os seguintes navegadores são compatíveis com o uso de chaves de segurança certificados pela FIDO:

iOS 14.5+ Android 7+
Chrome Sim Sim
Safari Sim Não
Borda Não Não
Firefox Sim Não
nota

No momento, o AWS Console Mobile App não é compatível com o uso de chaves de segurança com certificação FIDO para MFA.

AWS CLI e API da AWS

No momento, a AWS só é compatível com o uso de chaves de segurança com certificação FIDO no AWS Management Console. Não há compatibilidade para o uso de chaves de segurança certificadas pela FIDO na AWS CLI e na API da AWS ou para acesso a operações de API protegidas por MFA.

Recursos adicionais

  • Para obter ajuda com a solução de problemas de FIDO na AWS, consulte Solução de problemas de chaves de segurança FIDO.

  • Para obter informações gerais do setor sobre compatibilidade com FIDO, consulte Projeto FIDO2.
