에 로그인 AWS Management Console

기본 AWS 로그인 URL (https://signin.aws.amazon.com) AWS Management Console 에서 로그인할 때는 사용자 유형 (루트 사용자 또는 IAM 사용자) 을 선택해야 합니다.

루트 사용자는 무제한 계정 액세스 권한을 가지며 계정을 만든 사람과 연결됩니다. AWS 계정 그런 다음 루트 사용자는 IAM 사용자 및 AWS IAM Identity Center(AWS Single Sign-On의 후속 제품) 사용자 같은 다른 유형의 사용자를 생성하고 액세스 자격 증명을 할당합니다.

IAM 사용자는 특정 사용자 지정 권한을 가지고 AWS 계정 있는 내의 ID입니다. IAM 사용자는 로그인할 때 기본 로그인 URL https://account_alias_or_id.signin.aws.amazon.com/console/ 대신 AWS 계정 또는 별칭이 포함된 AWS 로그인 URL을 사용할 수 있습니다. https://signin.aws.amazon.com/

어떤 사용자인지 잘 모르겠으면 을 참조하십시오사용자 유형.
