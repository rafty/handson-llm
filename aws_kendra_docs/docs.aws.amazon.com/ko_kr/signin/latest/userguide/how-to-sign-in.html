로그인하는 방법 AWS

로그인하는 방법은 AWS 사용자 유형에 AWS 따라 다릅니다. AWS사용자 유형에는 여러 가지가 있습니다. 계정 루트 사용자, IAM 사용자, IAM Identity Center의 사용자, 페더레이션 아이덴티 또는 사용자 Builder ID가 될 수 있습니다. AWS 자세한 정보는 사용자 유형을 참조하세요.

다음 방법 중 하나로 AWS 로그인하여 액세스할 수 있습니다.

  • 에 로그인 AWS Management Console루트 사용자 또는 IAM 사용자로

  • AWS액세스 포털에 로그인IAM Identity Center의 사용자로

  • 페더레이션 ID로 로그인

  • 를 통해 로그인 AWS Command Line InterfaceAPI 및 SDK (소프트웨어 개발 키트) 와 같은 기타 프로그래밍 방식

  • AWS빌더 ID로 로그인

새로 AWS 계정 만들려면 AWS설치 가이드의 1부: 새 항목 AWS 계정 설정을 참조하십시오.
