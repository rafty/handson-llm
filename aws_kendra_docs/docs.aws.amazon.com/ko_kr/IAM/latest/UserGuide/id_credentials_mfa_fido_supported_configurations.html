FIDO 보안 키 사용에 지원되는 구성

현재 지원되는 구성을 사용하여 IAM에서 WebAuthn을 다중 인증(MFA) 방법으로 사용할 수 있습니다. 여기에는 IAM에서 지원하는 FIDO 인증 디바이스와 WebAuthn을 지원하는 브라우저가 포함됩니다.

AWS에서 지원하는 FIDO 디바이스

IAM은 현재 USB, Bluetooth 또는 NFC를 통해 디바이스에 연결하는 FIDO 인증 보안 디바이스를 지원합니다. 현재 플랫폼 인증(예: TouchID, FaceID 또는 Windows Hello)은 지원하지 않습니다.

참고

AWS에는 FIDO 인증 디바이스를 확인할 컴퓨터의 물리적 USB 포트에 대한 액세스 권한이 필요합니다. WebAuthn MFA는 가상 머신, 원격 연결 또는 브라우저의 익명 모드에서 작동하지 않습니다.

FIDO Alliance는 FIDO 사양과 호환되는 모든 FIDO 인증 제품 목록을 유지 관리합니다.

브라우저 플러그 인

현재 AWS는 FIDO2/WebAuthn 표준을 기본적으로 지원하는 브라우저만 지원합니다. AWS는 FIDO2/WebAuthn 브라우저 지원을 추가하기 위한 플러그인 사용을 지원하지 않습니다. 또한 일부 브라우저 플러그인은 U2F 표준과 호환되지 않으며 FIDO2 보안 키와 연결할 때 예기치 않은 결과를 초래할 수 있습니다.

브라우저 플러그인 비활성화 및 기타 문제 해결을 위한 자세한 내용은 FIDO 보안 키를 활성화할 수 없습니다. 섹션을 참조하세요.

AWS CLI 및 AWS API

AWS에서는 현재 AWS Management Console에서만 FIDO 인증 보안 키 사용을 지원합니다. MFA에 대한 FIDO 인증 보안 키 사용은 현재 AWS CLI 및 AWS API 또는 MFA 보호 API 작업에 대한 액세스에는 지원되지 않습니다.

추가 리소스

  • AWS에서 FIDO 인증 보안 키 사용에 대한 자세한 내용을 알아보려면 FIDO 보안 키 활성화(콘솔) 섹션을 참조하세요.

  • AWS에서 FIDO 문제 해결에 대한 도움말은 FIDO 보안 키 문제 해결 섹션을 참조하세요.

  • FIDO 지원에 대한 일반적인 업계 정보는 FIDO2 Project(FIDO2 프로젝트)를 참조하세요.
