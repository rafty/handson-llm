루트 사용자 자격 증명이 필요한 작업

중요

다음 AWS Identity and Access Management (IAM) 조치는 2023년 7월에 표준 지원이 종료됩니다. aws-portal:ModifyAccount 및. aws-portal:ViewAccount 세분화된 작업을 사용하여 이러한 AWS Billing 작업을 세분화된 작업으로 바꾸어 AWS 비용 관리 및 계정 콘솔에 AWS Billing 액세스할 수 있게 하기를 참조하세요. AWS

2023년 3월 6일 이전에 AWS 계정 또는 AWS Organizations 관리 계정을 생성한 경우 세분화된 작업이 2023년 7월부터 적용됩니다. 세분화된 작업을 추가하는 것이 좋지만, 접두사가 aws-portal 또는purchase-orders인 기존 권한은 제거하지 않는 것이 좋습니다.

2023년 3월 6일 당일 또는 이후에 계정 또는 AWS Organizations 관리 계정을 AWS 생성한 경우 세분화된 작업이 2023년 즉시 적용됩니다.

에서 관리 사용자가 일상적인 작업을 수행하고 AWS 리소스에 AWS IAM Identity Center (successor to AWS Single Sign-On) 액세스하도록 구성하는 것이 좋습니다. 하지만 아래 나열된 작업은 계정의 루트 사용자로 로그인한 경우에만 수행할 수 있습니다.

작업

  • 계정 설정을 변경합니다. 여기에는 계정 이름, 이메일 주소, 루트 사용자 암호 및 루트 사용자 액세스 키가 포함됩니다. 연락처 정보, 결제 통화 기본 설정 등과 같은 기타 계정 설정에는 루트 사용자 자격 증명이 필요하지 않습니다. AWS 리전

  • IAM 사용자 권한을 복원합니다. 유일한 IAM 관리자가 실수로 권한을 취소한 경우 루트 사용자로 로그인하여 정책을 편집하고 해당 권한을 복원할 수 있습니다.

  • Billing and Cost Management 콘솔에 대한 IAM 액세스를 활성화합니다.

  • 특정 세금 계산서를 조회합니다. aws-portal: ViewBilling 권한을 가진 IAM 사용자는 AWS 유럽의 VAT 세금계산서를 보고 다운로드할 수 있지만 AWS Inc. 또는 아마존 인터넷 서비스 프라이빗 리미티드 (AISPL) 는 볼 수 없습니다.

  • 당신의 문을 닫으세요 AWS 계정.

  • 예약 인스턴스 마켓플레이스에 판매자로 등록합니다.

  • MFA (멀티 팩터 인증) 를 활성화하도록 Amazon S3 버킷을 구성합니다.

  • 잘못된 가상 사설 클라우드 (VPC) ID 또는 VPC 엔드포인트 ID가 포함된 Amazon 심플 스토리지 서비스 (Amazon S3) 버킷 정책을 편집하거나 삭제합니다.

  • AWS GovCloud (US)에 가입합니다.

  • 에서 AWS GovCloud (US) 계정 루트 사용자 액세스 키를 AWS Support 요청합니다.
