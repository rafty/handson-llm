Amazon Lex R2

Amazon Lex V2는 음성 및 텍스트를 사용하는 애플리케이션을 위한 대화형 인터페이스를 구축하기 위한 AWS 서비스입니다. Amazon Lex V2는 자연어 이해 (NLU) 및 자동 음성 인식 (ASR) 의 심층적인 기능과 유연성을 제공하므로 실제와 같은 대화형 상호 작용으로 참여도가 높은 사용자 경험을 구축하고 새로운 범주의 제품을 만들 수 있습니다.

Amazon Lex V2를 사용하면 모든 개발자가 대화형 봇을 신속하게 구축할 수 있습니다. Amazon Lex V2를 사용하면 딥 러닝 전문 지식이 필요하지 않습니다. 봇을 만들려면 Amazon Lex V2 콘솔에서 기본 대화 흐름을 지정해야 합니다. Amazon Lex V2는 대화 상자를 관리하고 대화의 응답을 동적으로 조정합니다. 콘솔을 사용하여 텍스트 또는 음성 챗봇을 구축, 테스트 및 게시할 수 있습니다. 그런 다음 모바일 장치, 웹 애플리케이션 및 채팅 플랫폼(예: Facebook Messenger)에서 봇에 대화형 인터페이스를 추가할 수 있습니다.

Amazon Lex V2는 Amazon ConnectAWS Lambda, Amazon Comprehend 및 Amazon Kendra Kendra를 비롯한 AWS 플랫폼의 다른 많은 서비스와의 통합을 제공하며, 이를 통해 AWS 플랫폼의 다른 여러 서비스와 통합할 수 있습니다. Lambda와의 통합을 통해 봇은 사전 구축된 서버리스 엔터프라이즈 커넥터에 액세스하여 Salesforce와 같은 SaaS 애플리케이션의 데이터에 연결할 수 있습니다.

2022년 8월 17일 이후에 생성된 봇의 경우 조건부 분기를 사용하여 봇과의 대화 흐름을 제어할 수 있습니다. 조건부 분기를 사용하면 Lambda 코드를 작성할 필요 없이 복잡한 대화를 생성할 수 있습니다.

Amazon Lex R2에 사용되는 이점을

  • 단순성 — Amazon Lex V2는 콘솔을 사용하여 몇 분 만에 자체 봇을 만들 수 있도록 안내합니다. 몇 가지 예제 문구를 제공하면 Amazon Lex V2는 봇이 음성과 텍스트를 사용하여 상호 작용하여 질문하고 답변을 얻고 정교한 작업을 완료할 수 있는 완전한 자연어 모델을 구축합니다.

     

  • 대중화된 딥 러닝 기술 — Amazon Lex V2는 음성 언어 이해 (SLU) 시스템을 만들기 위한 ASR 및 NLU 기술을 제공합니다. Amazon Lex V2는 SLU를 통해 자연어 음성 및 텍스트 입력을 받아 입력의 의도를 이해하고 적절한 비즈니스 함수를 호출하여 사용자 의도를 충족합니다.

     

    음성 인식과 자연어 이해는 컴퓨터 과학에서 해결하기가 가장 어려운 문제 중 하나로, 엄청난 양의 데이터와 인프라에 대해 정교한 딥 러닝 알고리즘을 훈련해야 합니다. Amazon Lex V2를 사용하면 모든 개발자가 딥 러닝 기술을 사용할 수 있습니다. Amazon Lex V2 봇은 수신 음성을 텍스트로 변환하고 사용자 의도를 이해하여 지능적인 응답을 생성하므로 고객에게 부가 가치를 제공하는 봇을 구축하는 데 집중하고 대화형 인터페이스를 통해 가능해진 완전히 새로운 범주의 제품을 정의할 수 있습니다.

     

  • 원활한 배포 및 확장 — Amazon Lex V2를 사용하면 Amazon Lex V2 콘솔에서 직접 봇을 구축, 테스트 및 배포할 수 있습니다. Amazon Lex V2를 사용하면 모바일 디바이스, 웹 앱 및 채팅 서비스 (예: Facebook Messenger) 에서 사용할 음성 또는 텍스트 봇을 게시할 수 있습니다. Amazon Lex R2에 사용되는 Amazon R2에 사용되는 봇 경험을 강화하기 위해 하드웨어를 프로비저닝하고 인프라를 관리하는 것에 대해 걱정할 필요가 없습니다.

     

  • AWS 플랫폼과의 통합 기능 내장 — Amazon Lex V2는 기본적으로AWS Lambda Amazon과 같은 다른 AWS 서비스와 함께 작동합니다 CloudWatch. 보안, 모니터링, 사용자 인증, 비즈니스 로직, 스토리지 및 모바일 앱 개발에 대해 AWS 플랫폼이 제공하는 기능을 이용할 수 있습니다.

     

  • 비용 효율성 — Amazon Lex V2를 사용하면 선결제 비용이나 최소 요금이 없습니다. 사용한 텍스트 또는 음성 요청에 대해서만 청구됩니다. pay-as-you-go 가격 및 요청당 비용이 저렴하기 때문에 이 서비스는 대화형 인터페이스를 구축하는 비용 효율적인 방법입니다. Amazon Lex V2 프리 티어를 사용하면 초기 투자 없이 Amazon Lex V2를 쉽게 사용해 볼 수 있습니다.
