멀티 팩터 인증

멀티 팩터 인증 (MFA) 을 활성화하면 사용자는 사용자 이름 및 암호로 AWS 액세스 포털에 로그인해야 합니다. 이것이 그들이 알고 있는 첫 번째 요인입니다. 또한 사용자는 코드나 보안 키를 사용하여 로그인해야 합니다. 이것이 두 번째 요소입니다. 그들이 가지고 있거나 있는 것입니다. 두 번째 요인은 모바일 장치에서 생성된 인증 코드일 수도 있고 컴퓨터에 연결된 보안 키를 탭하여 생성한 것일 수도 있습니다. 이러한 여러 요소를 종합하면 유효한 MFA 챌린지가 성공적으로 완료되지 않는 한 AWS 리소스에 대한 무단 액세스를 방지하여 보안을 강화할 수 있습니다. 각 사용자는 최대 8개의 MFA 디바이스를 등록할 수 있습니다.

중요

최상의 보안을 위해 멀티 팩터 인증을 활성화하는 것이 좋습니다. MFA는 사용자 이름 및 암호의 기본 인증 메커니즘에 추가 보호 계층을 추가할 수 있는 간단하고 안전한 방법을 제공합니다.
