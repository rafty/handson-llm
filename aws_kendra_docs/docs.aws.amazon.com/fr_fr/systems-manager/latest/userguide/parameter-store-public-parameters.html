Utilisation de paramètres publics

Certains Services AWS publient des informations sur des artefacts courants sous forme de paramètres AWS Systems Manager publics. Par exemple, le service Amazon Elastic Compute Cloud (Amazon EC2) publie des informations sur des Amazon Machine Images (AMIs) sous forme de paramètres publics.

Vous pouvez appeler ces informations à partir de vos scripts et de votre code à l'aide des opérations d'API DescribeParameters, GetParametersByPath, GetParameter, et GetParameters.

Plus d'informations

  • Requête pour Régions AWS, les points de terminaison etc. à l'aide de AWS Systems ManagerParameter Store

  • Requête pour les derniers ID Amazon Linux AMI à l'aide de AWS Systems ManagerParameter Store

  • Requête pour la dernière version de Windows AMI à l'aide de AWS Systems ManagerParameter Store
