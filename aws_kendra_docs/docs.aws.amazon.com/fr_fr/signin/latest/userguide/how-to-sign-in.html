Comment se connecter à AWS

La manière dont vous vous connectez AWS dépend du type d'AWSutilisateur auquel vous appartenez. Il existe des types d'AWSutilisateurs différents. Vous pouvez être un utilisateur fédérées fédérées fédérées fédérées fédérées fédérées ou utiliser AWS Builder ID ID. Pour plus d'informations, veuillez consulter Types d'utilisateurs.

Vous pouvez y accéder AWS en vous connectant à l'aide de l'une des méthodes suivantes :

  • Connectez-vous au AWS Management Consoleen tant qu'utilisateur root ou utilisateur IAM

  • Connectez-vous au portail d'AWSaccèsen tant qu'utilisateur dans IAM Identity Center

  • Connectez-vous en tant qu'identité fédérée

  • Connectez-vous via le AWS Command Line Interfaceet d'autres méthodes de programmation telles que l'API et le SDK (kit de développement logiciel)

  • Connectez-vous avec AWS Builder ID

Si vous souhaitez en créer un nouveauCompte AWS, consultez la partie 1 : Configurer un nouveau Compte AWS dans le Guide de AWS configuration.
