Connectez-vous au AWS Management Console

Lorsque vous vous connectez à AWS Management Console partir de l'URL de AWS connexion principale (https://signin.aws.amazon.com), vous devez choisir votre type d'utilisateur, utilisateur root ou utilisateur IAM.

L'utilisateur root dispose d'un accès illimité au compte et est associé à la personne qui a créé leCompte AWS. L'utilisateur root crée ensuite d'autres types d'utilisateurs, tels que les utilisateurs IAM etutilisateurs dans AWS IAM Identity Center (successeur d'AWS Single Sign-On), et leur attribue des informations d'accès.

Un utilisateur IAM est une identité au sein de votre Compte AWS qui dispose d'autorisations personnalisées spécifiques. Lorsqu'un utilisateur IAM se connecte, il peut utiliser une URL de connexion incluant son alias Compte AWS ou, par exemple, https://account_alias_or_id.signin.aws.amazon.com/console/ au lieu de l'URL de AWS connexion principale. https://signin.aws.amazon.com/

Si vous ne savez pas quel type d'utilisateur vous appartenez, veuillez consulterTypes d'utilisateurs.
