Connectez-vous au portail d'AWSaccès

Un utilisateur d'IAM Identity Center est membre deAWS Organizations. Un utilisateur d'IAM Identity Center peut accéder à plusieurs Comptes AWS applications métiers en se connectant au portail d'AWSaccès à l'aide d'une URL de connexion spécifique. Pour plus d'informations sur l'URL de connexion spécifique, veuillez consulterAWSportail d'accès.

Avant de vous connecter à un en Compte AWS tant qu'utilisateur IAM Identity Center, collectez les informations requises suivantes.

  • Nom d'utilisateur d'entreprise

  • Mot de passe d'entreprise

  • URL de connexion spécifique

Note

Une fois connecté, votre session du portail AWS d'accès est valide pendant 8 heures. Vous devez vous reconnecter après 8 heures.

Pour vous connecter au portail AWS d'accès

  1. Dans la fenêtre de votre navigateur, collez l'URL de connexion qui vous a été fournie par e-mail, par exemplehttps://your_subdomain.awsapps.com/start. Ensuite, appuyez sur Entrée.

  2. Connectez-vous à l'aide de vos informations d'identification d'entreprise (comme un nom d'utilisateur et un mot de passe).

    Note

    Si votre administrateur vous a envoyé un mot de passe à usage unique (OTP) par e-mail et que c'est la première fois que vous vous connectez, saisissez ce mot de passe. Une fois connecté, vous devez créer un nouveau mot de passe pour vos future connexions.

  3. Si un code de vérification vous est demandé, consultez votre e-mail. Copiez et collez ensuite le code dans la page de connexion.

    Note

    Les codes de vérification sont généralement envoyés par e-mail, mais le mode de livraison peut varier. Si vous n'en avez pas reçu par e-mail, contactez votre administrateur pour plus de détails concernant votre code de vérification.

  4. Si l'authentification MFA est activée pour votre utilisateur dans IAM Identity Center, vous vous authentifiez ensuite à l'aide de cette fonctionnalité.

  5. Après l'authentification, vous pouvez accéder à n'importe quelle application Compte AWS et qui apparaît sur le portail.

    1. Pour vous connecter, AWS Management Console sélectionnez la Compte AWS vignette, puis sélectionnez le compte individuel à gérer.

      Les autorisations dont dispose votre utilisateur sont affichées avec des liens permettant d'ouvrir la console de gestion ou d'obtenir des informations d'identification pour un accès par ligne de commande ou par programmation.

      Cliquez sur le lien de la console de gestion.

    2. Pour vous connecter à une application, choisissez l'icône de l'application à laquelle vous souhaitez accéder.

La connexion en tant qu'utilisateur dans IAM Identity Center vous fournit les informations d'identification nécessaires pour accéder aux ressources pendant une durée définie, appelée session. Par défaut, un utilisateur peut être connecté à un compte Compte AWS pendant une heure. La valeur maximale que l'administrateur peut définir pour une session est de 12 heures. Une fois votre session terminée, vous pouvez vous reconnecter.
