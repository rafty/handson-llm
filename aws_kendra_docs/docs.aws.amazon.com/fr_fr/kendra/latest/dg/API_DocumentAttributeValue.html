DocumentAttributeValue

La valeur d'un attribut de document. Vous ne pouvez fournir qu'une seule valeur pour un attribut de document.

Table des matières

DateValue

Une date exprimée sous la forme d'une chaîne ISO 8601.

Il est important que le fuseau horaire soit inclus dans le format de date et d'heure ISO 8601. Par exemple, 2012-03-25T12:30:10+01:00 est le format de date et d'heure ISO 8601 pour le 25 mars 2012 à 12 h 30 (plus 10 secondes) à l'heure d'Europe centrale.

Type : horodatage

Obligatoire : non

LongValue

Une valeur de type entier long.

Type : long

Obligatoire : non

StringListValue

Une liste de chaînes. La longueur ou le nombre maximum de chaînes par défaut est de 10.

Type : tableau de chaînes

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 2048.

Obligatoire : non

StringValue

Une chaîne, telle que « département ».

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 2048.

Obligatoire : non
