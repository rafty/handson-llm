Sources de données

Une source de données est un référentiel de données ou un emplacement quiAmazon Kendra se connecte à vos documents ou à votre contenu et les indexe. Par exemple, vous pouvez configurer la connexionAmazon Kendra à Microsoft SharePoint afin d'explorer et d'indexer vos documents stockés dans cette source. Vous pouvez également indexer des pages Web en fournissant les URLAmazon Kendra à explorer. Vous pouvez synchroniser automatiquement une source de données avec unAmazon Kendra index afin que les documents ajoutés, mis à jour ou supprimés dans la source de données soient également ajoutés, mis à jour ou supprimés dans l'index.

Pour une liste des types de documents pris en charge par,Amazon Kendra voir Types de documents.

Note

Pour créer un index, il n'est pas nécessaire de disposer d'une source de données. Vous pouvez ajouter des documents directement à un index. Pour plus d'informations, veuillez consulter Ajout de documents directement à un index.

Pour indexer des documents à l'aide d'une source de données.

  1. Créez un index.

  2. Créez une source de données.

Pour une procédure pas à pas avec laAmazon Kendra console ou avec l'AWSinterface de ligne de commande, voirDémarrer.
