CustomDocumentEnrichmentConfiguration

Fournit les informations de configuration pour modifier les métadonnées et le contenu des documents pendant le processus d'intégration de documents.

Pour plus d'informations, veuillez consulter la rubrique Personnalisation des métadonnées de documents pendant le processus d'intégration.

Table des matières

InlineConfigurations

Informations de configuration permettant de modifier les attributs des documents ou les champs de métadonnées et le contenu lors de l'intégration de documents dans Amazon Kendra.

Type : tableau d’objets InlineCustomDocumentEnrichmentConfiguration

Membres du tableau : nombre minimum de 0 élément. Nombre maximal de 100 éléments.

Obligatoire : non

PostExtractionHookConfiguration

Informations de configuration pour l'appel d'une fonction Lambda dans AWS Lambda sur les documents structurés avec leurs métadonnées et leur texte extraits. Vous pouvez utiliser une fonction Lambda pour appliquer une logique avancée en vue de créer, modifier ou supprimer les métadonnées et le contenu des documents. Pour plus d'informations, veuillez consulter la rubrique Manipulation avancée des données.

Type : objet HookConfiguration

Obligatoire : non

PreExtractionHookConfiguration

Informations de configuration pour appeler une fonction Lambda dans AWS Lambda sur les documents originaux ou bruts avant l'extraction de leurs métadonnées et de leur texte. Vous pouvez utiliser une fonction Lambda pour appliquer une logique avancée en vue de créer, modifier ou supprimer les métadonnées et le contenu des documents. Pour plus d'informations, veuillez consulter la rubrique Manipulation avancée des données.

Type : objet HookConfiguration

Obligatoire : non

RoleArn

L'Amazon Resource Name (ARN) d'un rôle avec l'autorisation d'exécuter PreExtractionHookConfiguration et PostExtractionHookConfiguration en vue de modifier les métadonnées et le contenu des documents pendant le processus d'intégration de documents. Pour plus d'informations, veuillez consulter la rubrique Rôles IAM pour Amazon Kendra.

Type : String

Contraintes de longueur : longueur minimum de 0. Longueur maximale de 1284.

Modèle : arn:[a-z0-9-\.]{1,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[a-z0-9-\.]{0,63}:[^/].{0,1023}

Obligatoire : non
