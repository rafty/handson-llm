StartDataSourceSyncJob

Lance une tâche de synchronisation pour un connecteur de source de données. Si une tâche de synchronisation est déjà en cours, Amazon Kendra renvoie uneResourceInUseException exception.

Syntaxe de la requête

{
   "Id": "string",
   "IndexId": "string"
}

Paramètres de demande

Pour plus d'informations sur les paramètres communs à toutes les actions, voir Paramètres communs.

Cette demande accepte les données suivantes au format JSON.

Id

Identifiant du connecteur de source de données à synchroniser.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximum de 100.

Modèle : [a-zA-Z0-9][a-zA-Z0-9_-]*

Obligatoire : oui

IndexId

Identifiant de l'index utilisé avec le connecteur de source de données.

Type : String

Contraintes de longueur : longueur fixe de 36.

Modèle : [a-zA-Z0-9][a-zA-Z0-9-]*

Obligatoire : oui

Syntaxe de la réponse

{
   "ExecutionId": "string"
}

Eléments de réponse

Si l'action aboutit, le service renvoie une réponse HTTP 200.

Les données suivantes sont renvoyées au format JSON par le service.

ExecutionId

Identifie une tâche de synchronisation particulière.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 2048.

Erreurs

Pour plus d'informations sur les erreurs communes à toutes les actions, consultezErreurs courantes.

AccessDeniedException

L'accès dont vous disposez ne satisfait pas les conditions nécessaires pour effectuer cette action. Assurez-vous de disposer des politiques d'autorisation et des comptes utilisateur requis, puis réessayez.

Code d'état HTTP : 400

ConflictException

Un conflit est survenu avec la demande. Corrigez toute incohérence avec vos ressources et réessayez.

Code d'état HTTP : 400

InternalServerException

Un problème s'est produit avec le serveur interne utilisé pour votre service Amazon Kendra. Patientez quelques minutes et réessayez, ou contactez le support pour obtenir de l'aide.

Code d'état HTTP : 500

ResourceInUseException

La ressource que vous souhaitez utiliser est actuellement utilisée. Vérifiez que vous avez fourni la bonne ressource et réessayez.

Code d'état HTTP : 400

ResourceNotFoundException

La ressource que vous souhaitez utiliser n'existe pas. Vérifiez que vous avez fourni la bonne ressource et réessayez.

Code d'état HTTP : 400

ThrottlingException

La demande a été refusée suite à une limitation des demandes. Veuillez réduire le nombre de demandes et réessayer.

Code d'état HTTP : 400

ValidationException

L'entrée ne satisfait pas les contraintes définies par le service Amazon Kendra. Veuillez saisir la bonne entrée et réessayer.

Code d'état HTTP : 400
