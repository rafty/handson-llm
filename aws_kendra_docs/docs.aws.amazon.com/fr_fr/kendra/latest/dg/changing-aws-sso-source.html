Modification de la source d'identité de votre IAM Identity Center

Avertissement

La modification de votre source d'identité dans les paramètres d'IAM Identity Center peut affecter la préservation des informations relatives aux utilisateurs et aux groupes. Pour le faire en toute sécurité, il est recommandé de consulter les Considérations relatives à la modification de la source d'identité. Lorsque vous modifiez votre source d'identité, un nouvel identifiant de source d'identité est généré. Vérifiez que vous utilisez le bon identifiant avant de régler le mode surAWS_SSO In UserGroupResolutionConfiguration.

Pour modifier la source d'identité de votre IAM Identity Center

  1. Ouvrez la console IAM Identity Center>.

  2. Sélectionnez Settings (Paramètres).

  3. Sur la page Paramètres, sous Source d'identité, choisissez Modifier.

  4. Sur la page Modifier la source d'identité, sélectionnez votre source d'identité préférée, puis choisissez Suivant.
