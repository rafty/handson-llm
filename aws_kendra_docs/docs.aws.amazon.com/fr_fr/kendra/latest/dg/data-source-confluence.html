Confluence

Confluence est un outil de gestion du travail collaboratif conçu pour partager, stocker et travailler sur la planification de projets, le développement de logiciels et la gestion des produits. Vous pouvez l'utiliserAmazon Kendra pour indexer vos espaces Confluence, vos pages (y compris les pages imbriquées), vos blogs, vos commentaires et pièces jointes vers des pages et des blogs indexés.

Amazon Kendraprend en charge à la fois Confluence Server et Confluence Cloud.

Note

Par défaut,Amazon Kendra n'indexe pas les archives et les espaces personnels de Confluence. Vous pouvez choisir de les indexer lorsque vous créez la source de données. Si vous ne souhaitez pasAmazon Kendra indexer un espace, marquez-le comme privé dans Confluence.

Vous pouvez vous connecterAmazon Kendra à votre source de données Confluence à l'aide de la Amazon Kendraconsole, de l'TemplateConfigurationAPI ou de l'ConfluenceConfigurationAPI.

Amazon Kendrapossède deux versions du connecteur Confluence. Les fonctionnalités prises en charge par chaque version incluent :

Connecteur Confluence V1.0/ConfluenceConfigurationAPI

  • Mappages de champs

  • Filtrage du contexte utilisateur

  • Filtres d'inclusion/d'exclusion

  • (Pour Confluence Server uniquement) Cloud privé virtuel (VPC)

Connecteur Confluence V2.0/ TemplateConfigurationAPI

  • Mappages de champs

  • Filtrage du contexte utilisateur

  • Virtual privatecCloud (VPC)

  • Synchroniser tous les documents/Synchroniser uniquement les documents nouveaux, modifiés ou supprimés

  • Schémas d'inclusion/exclusion

Note

La prise en Support du connecteur Confluence ConfluenceConfiguration V1.0/API devrait prendre fin en juin 2023. Nous vous recommandons d'utiliser le connecteur Confluence V2.0/ TemplateConfiguration API.

Pour résoudre les problèmes liésAmazon Kendra à votre connecteur de source de données Confluence, consultezRésolution des problèmes de sources de données.
