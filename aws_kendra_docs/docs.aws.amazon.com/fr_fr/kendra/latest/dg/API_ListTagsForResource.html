ListTagsForResource

Obtient une liste des balises associés à une ressource spécifié. Les index, les FAQ et les sources de données peuvent être associés à des balises.

Syntaxe de la requête

{
   "ResourceARN": "string"
}

Paramètres de demande

Pour plus d'informations sur les paramètres communs à toutes les actions, voir Paramètres communs.

Cette demande accepte les données suivantes au format JSON.

ResourceARN

Amazon Resource Name (ARN) de l'index, de la FAQ ou de la source de données pour lequel vous souhaitez que l'intention Amazon Resource Name (ARN) de l'index, de la FAQ ou de la source de données.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximum de 1011.

Obligatoire : oui

Syntaxe de la réponse

{
   "Tags": [ 
      { 
         "Key": "string",
         "Value": "string"
      }
   ]
}

Eléments de réponse

Si l'action aboutit, le service renvoie une réponse HTTP 200.

Les données suivantes sont renvoyées au format JSON par le service.

Tags

Une liste de balises associés à l'index, à la FAQ ou à la source de données.

Type : tableau d’objets Tag

Membres du tableau : nombre minimum de 0 élément. maximum maximum de 200 éléments.

Erreurs

Pour plus d'informations sur les erreurs communes à toutes les actions, consultezErreurs courantes.

AccessDeniedException

Vous ne disposez pas de suffisamment d'accès pour effectuer cette action. Assurez-vous de disposer des politiques d'autorisation et des comptes utilisateur requis, puis réessayez.

Code d'état HTTP : 400

InternalServerException

Un problème s'est produit avec le serveur interne utilisé pour votre service Amazon Kendra. Patientez quelques minutes et réessayez, ou contactez le support pour obtenir de l'aide.

Code d'état HTTP : 500

ResourceUnavailableException

La ressource que vous souhaitez utiliser n'est pas disponible. Vérifiez que vous avez fourni la bonne ressource et réessayez.

Code d'état HTTP : 400

ThrottlingException

La demande a été refusée suite à une limitation des demandes. Veuillez réduire le nombre de demandes et réessayez.

Code d'état HTTP : 400

ValidationException

L'entrée ne satisfait pas les contraintes définis par le service Amazon Kendra. Veuillez fournir l'entrée correct et réessayez.

Code d'état HTTP : 400
