Démarrer avec une sourceAmazon S3 de données (console)

Vous pouvez utiliser laAmazon Kendra console pour commencer à utiliser unAmazon S3 bucket comme magasin de données. Lorsque vous utilisez la console, vous spécifiez toutes les informations de connexion dont vous avez besoin pour indexer le contenu du bucket. Pour plus d'informations, consultez Amazon S3.

Utilisez la procédure suivante pour créer une source de données de compartiment S3 de base en utilisant la configuration par défaut. La procédure suppose que vous avez créé un index en suivant les étapes de l'étape 1 deDémarrage avec laAmazon Kendra console.

Pour créer une source de données de compartiment S3 vide vide vide vide à l'aide de laAmazon Kendra console avec la

  1. Connectez-vousAWS Management Console et ouvrez laAmazon Kendra console à l'adresse https://console.aws.amazon.com/kendra/home.

  2. Dans la liste des index, choisissez l'index auquel vous souhaitez ajouter la source de données.

  3. Choisissez Ajouter des sources de données.

  4. Dans la liste des connecteurs de source de données, choisissez Amazon S3.

  5. Sur la page Définir les attributs, donnez un nom et éventuellement une description à votre source de données. Laissez le champ Tags vide. Choisissez Next (Suivant) pour continuer.

  6. Dans le champ Enter l'emplacement de la source de données, saisissez le nom du compartiment S3 qui contient vos documents. Vous pouvez saisir le nom directement ou rechercher le nom en choisissant Parcourir. Le compartiment doit se situer dans la même région que l'index.

  7. Dans IAMRôle, choisissez Créer un nouveau rôle, puis tapez un nom de rôle. Pour plus d'informations, consultez la section IAMRôles relatifs aux sources deAmazon S3 données.

  8. Dans la section Définir le calendrier d'exécution de la synchronisation, choisissez Exécuter à la demande.

  9. Choisissez Next (Suivant) pour continuer.

  10. Sur la page Réviser et créer, vérifiez les détails de votre source de données S3. Si vous souhaitez apporter des modifications, cliquez sur le bouton Modifier à côté de l'élément que vous souhaitez modifier. Lorsque vous êtes satisfait de vos choix, choisissez Create pour créer votre source de données S3.

Après avoir choisi Create,Amazon Kendra commence à créer la source de données. La création de la source de données par peut prendre plusieurs minutes. Une fois l'opération terminée, le statut de la source de données passe de Création à Active.

Après avoir créé la source de données, vous devez synchroniser l'Amazon Kendraindex avec la source de données. Choisissez Synchroniser maintenant pour démarrer le processus de synchronisation. La synchronisation de la source de données peut prendre de quelques minutes à plusieurs heures, selon le nombre et la taille des documents.
