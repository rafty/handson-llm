DocumentInfo

Identifie un document pour lequel récupérer des informations d'état

Table des matières

Attributes

Attributs qui identifient une version spécifique d'un document à vérifier.

Les seuls attributs valides sont :

  • version ;

  • ID de source de données

  • jobExecutionId

Les attributs suivent ces règles :

  • dataSourceIdetjobExecutionId doivent être utilisés ensemble.

  • versionest ignoré sidataSourceId et nejobExecutionId sont pas fournis.

  • SidataSourceId etjobExecutionId sont fournis, mais ne leversion sont pas, la version par défaut est « 0 ».

Type : tableau d’objets DocumentAttribute

Obligatoire : non

DocumentId

Identifiant du document.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 2048.

Obligatoire : oui
