Attributs ou champs du document

Des attributs ou des champs sont associés à un document. Les champs d'un document sont les propriétés d'un document ou le contenu de la structure d'un document. Par exemple, chacun de vos documents peut contenir un titre, un corps de texte et un auteur. Vous pouvez également ajouter vos propres champs personnalisés à vos documents. Les champs personnalisés sont des attributs que vous spécifiez pour vos propres besoins. Par exemple, si votre index recherche des documents fiscaux, vous pouvez spécifier un champ personnalisé pour le type de document fiscal, tel que W-2, 1099, etc.

Avant de pouvoir utiliser un champ de document dans une requête, il doit être mappé à un champ d'index. Par exemple, le champ de titre peut être mappé au champ_document_title. Pour plus d'informations, consultez Mappage des champs. Pour ajouter un nouveau champ, vous devez créer un champ d'index auquel mapper le champ. Vous créez des champs d'index à l'aide de la console ou de l'UpdateIndexAPI.

Vous pouvez utiliser les champs du document pour filtrer les réponses et faire des suggestions de recherche à facettes. Par exemple, vous pouvez filtrer une réponse pour ne renvoyer qu'une version spécifique d'un document, ou vous pouvez filtrer les recherches pour ne renvoyer que 1099 types de documents fiscaux correspondant au terme recherché. Pour plus d'informations, veuillez consulter Filtrage des requêtes.

Vous pouvez également utiliser les champs du document pour ajuster manuellement la réponse à la requête. Par exemple, vous pouvez choisir d'augmenter l'importance du champ de titre afin d'augmenter le poids quiAmazon Kendra lui est attribué lors de la détermination des documents à renvoyer dans la réponse. Pour plus d'informations, consultez Réglage de la pertinence de recherche.

Si vous ajoutez un document directement à un index, vous spécifiez les champs dans le paramètre d'entrée Document de l'BatchPutDocumentAPI. Vous spécifiez les valeurs des champs personnalisés dans un tableau d'DocumentAttributeobjets. Si vous utilisez une source de données, la méthode que vous utilisez pour ajouter les champs du document dépend de la source de données. Pour plus d'informations, veuillez consulter Création de champs de document personnalisés.
