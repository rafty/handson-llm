QueryResultItem

Un seul résultat de requête.

Le résultat d'une requête contient des informations sur un document renvoyé par la requête. Cela inclut l'emplacement d'origine du document, une liste d'attributs attribués au document et le texte pertinent du document qui répond à la requête.

Table des matières

AdditionalAttributes

Un ou plusieurs attributs supplémentaires associés au résultat de la requête.

Type : tableau d’objets AdditionalResultAttribute

Obligatoire : non

Tableau d'attributs de document attribués à un document dans les résultats de recherche. Par exemple, l'auteur du document (_author) ou l'URI source (_source_uri) du document.

Type : tableau d’objets DocumentAttribute

Obligatoire : non

Extrait du texte du document. Contient des informations sur la mise en évidence des termes pertinents dans l'extrait.

Type : objet TextWithHighlights

Obligatoire : non

DocumentId

Identifiant du document.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 2048.

Obligatoire : non

DocumentTitle

Titre du document. Contient le texte du titre et les informations permettant de mettre en évidence les termes pertinents du titre.

Type : objet TextWithHighlights

Obligatoire : non

DocumentURI

URI du document document document.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 2048.

Modèle : ^(https?|ftp|file):\/\/([^\s]*)

Obligatoire : non

FeedbackToken

Un jeton qui identifie un résultat particulier à partir d'une requête particulière. Utilisez ce jeton pour fournir des commentaires sur le résultat en cliquant dessus. Pour plus d'informations, consultez la section Soumission de commentaires.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 2048.

Modèle : ^\P{C}*.\P{C}*$

Obligatoire : non

Format

Si le documentType de la réponse l'estANSWER, il s'agit soit d'uneTABLE réponse, soit d'uneTEXT réponse. S'il s'agit d'une réponse tabulaire, un extrait du tableau est renvoyéTableExcerpt. S'il s'agit d'une réponse textuelle, un extrait de texte est renvoyéDocumentExcerpt.

Type : String

Valeurs valides : TABLE | TEXT

Obligatoire : non

Id

Identifiant du résultat de la requête.

Type : String

Contraintes de longueur : Longueur minimum de 1. Longueur maximale de 73.

Obligatoire : non

ScoreAttributes

Indique la confiance d'Amazon Kendra dans le fait qu'un résultat correspond à la requête que vous avez fournie. Chaque résultat est placé dans une case qui indique le degré de confianceVERY_HIGH,HIGH,MEDIUM etLOW. Vous pouvez utiliser le score pour déterminer si une réponse répond au niveau de confiance requis pour votre application.

Le champ n'est défini queLOW lorsque leType champ est défini surDOCUMENT et Amazon Kendra n'est pas certain que le résultat correspond à la requête.

Type : objet ScoreAttributes

Obligatoire : non

TableExcerpt

Extrait d'un tableau figurant dans un document.

Type : objet TableExcerpt

Obligatoire : non

Type

Type de document inclus dans la réponse. Par exemple, une réponse peut inclure une question-réponse pertinente à la requête.

Type : String

Valeurs valides : DOCUMENT | QUESTION_ANSWER | ANSWER

Obligatoire : non
