What is AWS Sign-In?

This guide helps you understand the ways that users can sign in to your AWS account and how to determine what kind of information to provide to users for them to be able to sign in to Amazon Web Services (AWS). Each of the different AWS user types have different sign-in page URLs that they use to sign in to the AWS Management Console, a web application that comprises and refers to a broad collection of service consoles for managing AWS resources. If you are federating identity providers through AWS Identity and Access Management you can have company-specific AWS access portal. Developers that are accessing your AWS account resources sign-in through the command-line. Users can also sign in to some AWS services and tools with an AWS Builder ID. For more information about how to sign in, see one of the following tutorials.

  • Sign in to the AWS Management Console

  • Sign in to the AWS access portal

  • Sign in as a federated identity

  • Sign in through the AWS Command Line Interface

  • Sign in with AWS Builder ID
